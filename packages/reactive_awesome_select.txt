# reactive_awesome_select Package Overview
=============================================

The `reactive_awesome_select` package is a powerful and flexible dropdown select package for Flutter. It provides a customizable and reactive dropdown select widget that can be used in a variety of scenarios.

### When to Use

This package is ideal for scenarios where you need to provide a dropdown select option to users, such as:

* Selecting a country or region
* Choosing a category or tag
* Picking a date or time
* Selecting a item from a list

### Features

* Customizable dropdown select widget
* Reactive dropdown select widget that updates automatically when the selected value changes
* Supports single and multiple selection modes
* Can be used with or without a `TextFormField`
* Supports validation and error handling

# Setting Up and Using reactive_awesome_select
=============================================

### Step 1: Add the Package to Your Project

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  reactive_awesome_select: ^1.0.0
```
Then run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:reactive_awesome_select/reactive_awesome_select.dart';
```
### Step 3: Create a `ReactiveAwesomeSelect` Widget

Create a `ReactiveAwesomeSelect` widget and pass in the required properties:
```dart
ReactiveAwesomeSelect(
  items: [
    AwesomeSelectItem(
      value: 'option1',
      label: 'Option 1',
    ),
    AwesomeSelectItem(
      value: 'option2',
      label: 'Option 2',
    ),
  ],
  value: 'option1',
  onChanged: (value) {
    print(value);
  },
)
```
### Platform-Specific Details

* On Android, the dropdown select widget will use the native Android dropdown select widget.
* On iOS, the dropdown select widget will use a custom dropdown select widget that mimics the native iOS dropdown select widget.

### Configurations and Optimizations

* You can customize the appearance of the dropdown select widget by passing in a `style` property.
* You can also customize the behavior of the dropdown select widget by passing in a `configuration` property.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:reactive_awesome_select/reactive_awesome_select.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Reactive Awesome Select Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  String _selectedValue = 'option1';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Reactive Awesome Select Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            ReactiveAwesomeSelect(
              items: [
                AwesomeSelectItem(
                  value: 'option1',
                  label: 'Option 1',
                ),
                AwesomeSelectItem(
                  value: 'option2',
                  label: 'Option 2',
                ),
              ],
              value: _selectedValue,
              onChanged: (value) {
                setState(() {
                  _selectedValue = value;
                });
              },
            ),
            SizedBox(height: 20),
            Text('Selected Value: $_selectedValue'),
          ],
        ),
      ),
    );
  }
}

// The app starts by running the main function, which runs the MyApp widget.
// The MyApp widget builds a MaterialApp with a title and a theme.
// The MaterialApp has a home property that points to the MyHomePage widget.
// The MyHomePage widget is a stateful widget that builds a Scaffold with an AppBar and a body.
// The body of the Scaffold contains a Center widget that centers its child.
// The child of the Center widget is a Column widget that contains two children.
// The first child is a ReactiveAwesomeSelect widget that displays a dropdown select menu.
// The second child is a Text widget that displays the selected value.
// When the user selects a new value from the dropdown select menu, the onChanged callback is called.
// The onChanged callback updates the _selectedValue variable and calls setState to rebuild the widget tree.
// When the widget tree is rebuilt, the Text widget displays the new selected value.
```
