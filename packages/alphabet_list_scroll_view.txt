# Alphabet List Scroll View Package
=====================================

The `alphabet_list_scroll_view` package is a Flutter widget that allows users to quickly scroll through a list of items by selecting a letter from an alphabet list. This package is useful when dealing with large lists of data, such as contact lists or dictionaries.

### Features

*   Fast and efficient scrolling through lists
*   Customizable alphabet list and item list
*   Supports both Android and iOS platforms

### When to Use

*   When dealing with large lists of data that need to be quickly navigated
*   When a user needs to find a specific item in a list by its first letter

# Tutorial
==========

## Step 1: Add the Package to Your Project

Add the `alphabet_list_scroll_view` package to your `pubspec.yaml` file:

```yml
dependencies:
  alphabet_list_scroll_view: ^1.0.0
```

Then, run `flutter pub get` to install the package.

## Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:alphabet_list_scroll_view/alphabet_list_scroll_view.dart';
```

## Step 3: Create the Alphabet List Scroll View

Create the `AlphabetListScrollView` widget and pass in the required parameters:

```dart
AlphabetListScrollView(
  list: _list, // The list of items to display
  itemBuilder: (context, index) { // The builder for each item in the list
    return ListTile(
      title: Text(_list[index]),
    );
  },
  indexedHeight: (index) { // The height of each item in the list
    return 50;
  },
  headerHeight: 50, // The height of the alphabet list
  headerBuilder: (context, letter) { // The builder for each letter in the alphabet list
    return Text(
      letter,
      style: TextStyle(fontSize: 20),
    );
  },
)
```

## Platform-Specific Details

### Android

No additional configuration is required for Android.

### iOS

No additional configuration is required for iOS.

```dart
import 'package:flutter/material.dart';
import 'package:alphabet_list_scroll_view/alphabet_list_scroll_view.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Alphabet List Scroll View Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: RealFlutter(),
    );
  }
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  List<String> _list = [
    'Apple',
    'Banana',
    'Cherry',
    'Date',
    'Elderberry',
    'Fig',
    'Grape',
    'Honeydew',
    'Ice Cream',
    'Jackfruit',
    'Kiwi',
    'Lemon',
    'Mango',
    'Nectarine',
    'Orange',
    'Peach',
    'Quince',
    'Raspberry',
    'Strawberry',
    'Tangerine',
    'Ugli Fruit',
    'Victoria Plum',
    'Watermelon',
    'Xigua',
    'Yellow Passionfruit',
    'Zucchini',
  ];

  @override
  Widget build(BuildContext context) {
    // Create the Alphabet List Scroll View
    return Scaffold(
      appBar: AppBar(
        title: Text('Alphabet List Scroll View Demo'),
      ),
      body: AlphabetListScrollView(
        list: _list, // The list of items to display
        itemBuilder: (context, index) { // The builder for each item in the list
          // Return a ListTile for each item in the list
          return ListTile(
            title: Text(_list[index]),
          );
        },
        indexedHeight: (index) { // The height of each item in the list
          // Return a fixed height for each item in the list
          return 50;
        },
        headerHeight: 50, // The height of the alphabet list
        headerBuilder: (context, letter) { // The builder for each letter in the alphabet list
          // Return a Text widget for each letter in the alphabet list
          return Text(
            letter,
            style: TextStyle(fontSize: 20),
          );
        },
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user runs the app and is presented with a list of items.
// 2. The user can scroll through the list by selecting a letter from the alphabet list.
// 3. When a letter is selected, the list scrolls to the first item that starts with that letter.
// 4. The user can then scroll through the list manually to find the desired item.
```
