**esc_pos_dart Package Overview**
=====================================

The `esc_pos_dart` package is a Flutter library that allows you to print receipts and other documents using ESC/POS printers. ESC/POS is a standard protocol used by most receipt printers, making this package a versatile solution for various printing needs.

**When to Use**
---------------

Use the `esc_pos_dart` package when you need to print receipts, invoices, or other documents from your Flutter app. This package is particularly useful for:

* Point of Sale (POS) systems
* Retail and hospitality apps
* Inventory management systems
* Any app that requires printing documents

**Features**
------------

The `esc_pos_dart` package offers the following features:

* Supports ESC/POS protocol
* Compatible with most receipt printers
* Allows printing of text, images, and barcodes
* Customizable font sizes, styles, and alignments
* Supports multiple paper sizes and orientations

**Setup and Usage**
=====================

### Step 1: Add the Package to Your Pubspec

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  esc_pos_dart: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:esc_pos_dart/esc_pos_dart.dart';
```
### Step 3: Create an Instance of the Printer

Create an instance of the `EscPosPrinter` class:
```dart
final printer = EscPosPrinter(
  paperSize: PaperSize.mm80,
  printerDpi: 203,
);
```
### Step 4: Configure the Printer (Optional)

You can configure the printer settings, such as the paper size, orientation, and font size:
```dart
printer.setPaperSize(PaperSize.mm80);
printer.setOrientation(Orientation.portrait);
printer.setFontSize(24);
```
### Step 5: Print Your Document

Use the `print` method to print your document:
```dart
final document = 'Hello, World!';
printer.print(document);
```
### Platform-Specific Details

* **Android:** Make sure to add the `android.permission.BLUETOOTH` permission to your `AndroidManifest.xml` file.
* **iOS:** Make sure to add the `NSBluetoothAlwaysUsageDescription` key to your `Info.plist` file.

```dart
import 'package:flutter/material.dart';
import 'package:esc_pos_dart/esc_pos_dart.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'ESC/POS Printer Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _printer = EscPosPrinter(
    paperSize: PaperSize.mm80,
    printerDpi: 203,
  );

  void _printDocument() {
    final document = 'Hello, World!';
    _printer.print(document);
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('ESC/POS Printer Demo'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: _printDocument,
          child: Text('Print Document'),
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user clicks the "Print Document" button.
// 2. The `_printDocument` method is called.
// 3. The `EscPosPrinter` instance is created with the specified paper size and DPI.
// 4. The `print` method is called with the document to be printed.
// 5. The printer prints the document using the ESC/POS protocol.
```
