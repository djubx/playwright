# wifi_info_plugin_plus Package Overview
=====================================

The `wifi_info_plugin_plus` package is a Flutter plugin that provides an easy-to-use API for accessing WiFi information on both Android and iOS devices. This package is useful when you need to retrieve WiFi network information, such as the network's name (SSID), BSSID, and IP address.

### Key Features

*   Get the current WiFi network's SSID, BSSID, and IP address
*   Get the list of nearby WiFi networks
*   Check if the device is connected to a WiFi network

### When to Use

This package is useful in various scenarios, such as:

*   Developing a network scanner app
*   Creating a WiFi-based IoT app
*   Building an app that requires WiFi network information for analytics or logging purposes

# Setting Up and Using the wifi_info_plugin_plus Package
=====================================================

### Step 1: Add the Package to Your Flutter Project

Add the following dependency to your `pubspec.yaml` file:

```yml
dependencies:
  flutter:
    sdk: flutter
  wifi_info_plugin_plus: ^1.0.0
```

Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:wifi_info_plugin_plus/wifi_info_plugin_plus.dart';
```

### Step 3: Get WiFi Network Information

Use the `WifiInfoPluginPlus` class to get the current WiFi network's information:

```dart
WifiInfoPluginPlus wifiInfo = WifiInfoPluginPlus();
String ssid = await wifiInfo.getSSID();
String bssid = await wifiInfo.getBSSID();
String ipAddress = await wifiInfo.getIPAddress();
```

### Platform-Specific Details

*   **Android:** This package uses the `android.net.wifi.WifiManager` class to retrieve WiFi network information. Make sure to add the `ACCESS_WIFI_STATE` permission to your `AndroidManifest.xml` file:

    ```xml
<uses-permission android:name="android.permission.ACCESS_WIFI_STATE" />
```
*   **iOS:** This package uses the `SystemConfiguration` framework to retrieve WiFi network information. Make sure to add the `NSLocationWhenInUseUsageDescription` key to your `Info.plist` file:

    ```xml
<key>NSLocationWhenInUseUsageDescription</key>
<string>This app needs access to your location to retrieve WiFi network information.</string>
```

```dart
import 'package:flutter/material.dart';
import 'package:wifi_info_plugin_plus/wifi_info_plugin_plus.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'WiFi Info Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  String _ssid = '';
  String _bssid = '';
  String _ipAddress = '';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('WiFi Info Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text(
              'SSID: $_ssid',
              style: TextStyle(fontSize: 20),
            ),
            Text(
              'BSSID: $_bssid',
              style: TextStyle(fontSize: 20),
            ),
            Text(
              'IP Address: $_ipAddress',
              style: TextStyle(fontSize: 20),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _getWiFiInfo,
              child: Text('Get WiFi Info'),
            ),
          ],
        ),
      ),
    );
  }

  Future<void> _getWiFiInfo() async {
    // Create an instance of WifiInfoPluginPlus
    WifiInfoPluginPlus wifiInfo = WifiInfoPluginPlus();

    // Get the current WiFi network's SSID
    String ssid = await wifiInfo.getSSID();
    setState(() {
      _ssid = ssid;
    });

    // Get the current WiFi network's BSSID
    String bssid = await wifiInfo.getBSSID();
    setState(() {
      _bssid = bssid;
    });

    // Get the current WiFi network's IP address
    String ipAddress = await wifiInfo.getIPAddress();
    setState(() {
      _ipAddress = ipAddress;
    });
  }
}

// The app starts by running the main function, which creates an instance of MyApp.
// MyApp is a stateless widget that returns a MaterialApp with a title and a theme.
// The home property of the MaterialApp is set to MyHomePage, which is a stateful widget.
// MyHomePage has a Scaffold with an AppBar and a body that displays the WiFi network information.
// The WiFi network information is retrieved using the WifiInfoPluginPlus class.
// The _getWiFiInfo function is called when the "Get WiFi Info" button is pressed.
// This function creates an instance of WifiInfoPluginPlus and uses it to retrieve the WiFi network information.
// The retrieved information is then displayed on the screen.
```
