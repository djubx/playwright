# Whitecodel Reels Flutter Package
=====================================

The `whitecodel_reels` Flutter package is a powerful tool for creating Instagram-like reels in your Flutter applications. It provides a simple and intuitive API for creating, editing, and playing reels.

### When to Use This Package

This package is ideal for social media applications, video editing apps, or any other app that requires reel-like functionality.

### Features

*   Create reels from videos or images
*   Edit reels with text, music, and filters
*   Play reels with controls (play, pause, seek)
*   Support for both Android and iOS platforms

# Tutorial: Setting Up and Using Whitecodel Reels

## Step 1: Add the Package to Your Project

Add the following dependency to your `pubspec.yaml` file:

```yml
dependencies:
  flutter:
    sdk: flutter
  whitecodel_reels: ^1.0.0
```

Then, run `flutter pub get` to install the package.

## Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:whitecodel_reels/whitecodel_reels.dart';
```

## Step 3: Create a Reel

Create a reel from a video or image:

```dart
Reel reel = Reel(
  video: Video.file(File('path/to/video.mp4')),
  // or
  image: Image.file(File('path/to/image.jpg')),
);
```

## Step 4: Edit the Reel

Edit the reel with text, music, and filters:

```dart
reel = reel.copyWith(
  text: Text('Hello, World!'),
  music: Music.file(File('path/to/music.mp3')),
  filter: Filter.sepia(),
);
```

## Step 5: Play the Reel

Play the reel with controls:

```dart
ReelPlayer player = ReelPlayer(
  reel: reel,
  onPlay: () => print('Reel playing'),
  onPause: () => print('Reel paused'),
  onSeek: (position) => print('Reel seeked to $position'),
);
```

### Platform-Specific Details

*   Android: Make sure to add the necessary permissions to your `AndroidManifest.xml` file:

    ```xml
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
```

*   iOS: Make sure to add the necessary permissions to your `Info.plist` file:

    ```xml
<key>NSPhotoLibraryUsageDescription</key>
<string>This app needs access to your photo library to play videos.</string>
<key>NSMicrophoneUsageDescription</key>
<string>This app needs access to your microphone to record audio.</string>
```

```dart
import 'package:flutter/material.dart';
import 'package:whitecodel_reels/whitecodel_reels.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Whitecodel Reels Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  Reel _reel;

  @override
  void initState() {
    super.initState();
    _reel = Reel(
      video: Video.file(File('path/to/video.mp4')),
    );
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Whitecodel Reels Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            ReelPlayer(
              reel: _reel,
              onPlay: () => print('Reel playing'),
              onPause: () => print('Reel paused'),
              onSeek: (position) => print('Reel seeked to $position'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                setState(() {
                  _reel = _reel.copyWith(
                    text: Text('Hello, World!'),
                    music: Music.file(File('path/to/music.mp3')),
                    filter: Filter.sepia(),
                  );
                });
              },
              child: Text('Edit Reel'),
            ),
          ],
        ),
      ),
    );
  }
}

// The app starts by creating a Material app with a home page.
// The home page is a stateful widget that displays a reel player and an edit button.
// When the edit button is pressed, the reel is updated with new text, music, and filter.
// The reel player displays the updated reel and allows the user to play, pause, and seek the reel.
```
