# Side Sheet Material 3 Flutter Package
=====================================

The `side_sheet_material3` Flutter package provides a material design 3 side sheet widget. It is used to display a sheet of content on the side of the screen. This package is useful when you want to display additional information or options to the user without obstructing the main content.

### Features

*   Material design 3 side sheet widget
*   Support for both Android and iOS platforms
*   Customizable side sheet width and elevation
*   Support for displaying a header and footer in the side sheet

### When to Use

*   Use this package when you want to display additional information or options to the user without obstructing the main content.
*   Use this package when you want to provide a material design 3 side sheet widget in your Flutter app.

# Tutorial
==========

### Setup

To use the `side_sheet_material3` package, add the following dependency to your `pubspec.yaml` file:

```yml
dependencies:
  side_sheet_material3: ^1.0.0
```

Then, run `flutter pub get` to get the package.

### Basic Usage

To use the `SideSheet` widget, import the package and use the `SideSheet` widget in your widget tree:

```dart
import 'package:side_sheet_material3/side_sheet_material3.dart';

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Side Sheet Material 3 Demo'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () {
            showSideSheet(
              context: context,
              builder: (context) {
                return SideSheet(
                  child: Center(
                    child: Text('Hello, World!'),
                  ),
                );
              },
            );
          },
          child: Text('Show Side Sheet'),
        ),
      ),
    );
  }
}
```

### Customization

You can customize the `SideSheet` widget by using the following properties:

*   `width`: The width of the side sheet.
*   `elevation`: The elevation of the side sheet.
*   `header`: The header of the side sheet.
*   `footer`: The footer of the side sheet.

For example:

```dart
SideSheet(
  width: 300,
  elevation: 10,
  header: Container(
    height: 50,
    child: Center(
      child: Text('Header'),
    ),
  ),
  footer: Container(
    height: 50,
    child: Center(
      child: Text('Footer'),
    ),
  ),
  child: Center(
    child: Text('Hello, World!'),
  ),
)
```

```dart
import 'package:flutter/material.dart';
import 'package:side_sheet_material3/side_sheet_material3.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Side Sheet Material 3 Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: RealFlutter(),
    );
  }
}

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // Create a Scaffold with an AppBar and a Centered ElevatedButton
    return Scaffold(
      appBar: AppBar(
        title: Text('Side Sheet Material 3 Demo'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () {
            // Show the SideSheet when the button is pressed
            showSideSheet(
              context: context,
              builder: (context) {
                // Create a SideSheet with a width of 300 and an elevation of 10
                return SideSheet(
                  width: 300,
                  elevation: 10,
                  // Create a header for the SideSheet
                  header: Container(
                    height: 50,
                    child: Center(
                      child: Text('Header'),
                    ),
                  ),
                  // Create a footer for the SideSheet
                  footer: Container(
                    height: 50,
                    child: Center(
                      child: Text('Footer'),
                    ),
                  ),
                  // Create a centered Text widget as the child of the SideSheet
                  child: Center(
                    child: Text('Hello, World!'),
                  ),
                );
              },
            );
          },
          child: Text('Show Side Sheet'),
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user presses the ElevatedButton.
// 2. The showSideSheet function is called, which displays the SideSheet.
// 3. The SideSheet is created with a width of 300 and an elevation of 10.
// 4. The header and footer of the SideSheet are created.
// 5. The centered Text widget is created as the child of the SideSheet.
// 6. The SideSheet is displayed on the screen.

```
