**Google Directions API Flutter Package**
=====================================

The `google_directions_api` Flutter package provides a simple and efficient way to calculate routes and get directions between two points on a map. This package is ideal for apps that require turn-by-turn navigation, estimated time of arrival, and route optimization.

**When to Use This Package**
---------------------------

*   When building a ride-hailing or ride-sharing app
*   When creating a delivery or logistics app
*   When developing a navigation or mapping app
*   When you need to calculate routes and get directions between two points on a map

**Features**
------------

*   Calculate routes between two points on a map
*   Get turn-by-turn directions
*   Estimated time of arrival
*   Route optimization
*   Support for multiple transportation modes (driving, walking, bicycling)

**Setup and Usage**
-------------------

### Step 1: Add the Package to Your Pubspec.yaml File

```yml
dependencies:
  flutter:
    sdk: flutter
  google_directions_api: ^1.0.0
```

### Step 2: Import the Package in Your Dart File

```dart
import 'package:google_directions_api/google_directions_api.dart';
```

### Step 3: Set Up the Google Directions API

```dart
GoogleDirectionsApi _googleDirectionsApi = GoogleDirectionsApi(
  apiKey: 'YOUR_API_KEY',
  transport: TransportType.driving,
);
```

### Step 4: Calculate Routes and Get Directions

```dart
Future<void> _calculateRoute() async {
  final directions = await _googleDirectionsApi.directions(
    origin: 'New York, USA',
    destination: 'Los Angeles, USA',
  );
  print(directions);
}
```

### Platform-Specific Details

*   **Android:** Add the following line to your `AndroidManifest.xml` file:

    ```xml
<uses-permission android:name="android.permission.INTERNET" />
```

*   **iOS:** Add the following line to your `Info.plist` file:

    ```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:google_directions_api/google_directions_api.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Google Directions API Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _originController = TextEditingController();
  final _destinationController = TextEditingController();
  final _googleDirectionsApi = GoogleDirectionsApi(
    apiKey: 'YOUR_API_KEY',
    transport: TransportType.driving,
  );

  Future<void> _calculateRoute() async {
    // Get the origin and destination from the text fields
    final origin = _originController.text;
    final destination = _destinationController.text;

    // Calculate the route using the Google Directions API
    final directions = await _googleDirectionsApi.directions(
      origin: origin,
      destination: destination,
    );

    // Print the directions
    print(directions);
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Google Directions API Demo'),
      ),
      body: Padding(
        padding: const EdgeInsets.all(20.0),
        child: Column(
          children: [
            TextField(
              controller: _originController,
              decoration: InputDecoration(
                labelText: 'Origin',
              ),
            ),
            TextField(
              controller: _destinationController,
              decoration: InputDecoration(
                labelText: 'Destination',
              ),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _calculateRoute,
              child: Text('Calculate Route'),
            ),
          ],
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user enters the origin and destination in the text fields.
// 2. The user clicks the "Calculate Route" button.
// 3. The _calculateRoute function is called, which gets the origin and destination from the text fields.
// 4. The _calculateRoute function calls the Google Directions API to calculate the route.
// 5. The Google Directions API returns the directions, which are printed to the console.
// 6. The directions can be used to display the route on a map or to provide turn-by-turn navigation.

// Note: You should replace 'YOUR_API_KEY' with your actual Google Maps API key.

```
