# Flutter Document Reader Core Full RFID Package
=====================================================

The `flutter_document_reader_core_fullrfid` package is a powerful tool for reading and processing various types of documents, including passports, ID cards, and more. This package is particularly useful for applications that require secure and reliable document scanning, such as border control, law enforcement, and identity verification.

Key Features:

* Support for multiple document types, including passports, ID cards, and more
* Advanced image processing and enhancement for optimal document scanning
* RFID support for secure and reliable document authentication
* Cross-platform compatibility for Android and iOS devices

When to Use:

* Border control and immigration applications
* Law enforcement and identity verification applications
* Secure document scanning and authentication applications

# Setup and Usage
---------------

### Step 1: Add the Package to Your Flutter Project

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  flutter_document_reader_core_fullrfid: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:flutter_document_reader_core_fullrfid/flutter_document_reader_core_fullrfid.dart';
```
### Step 3: Initialize the Package

Initialize the package by calling the `init` method:
```dart
await FlutterDocumentReaderCoreFullrfid.init();
```
### Step 4: Configure the Package (Optional)

Configure the package by setting the document type and RFID settings:
```dart
await FlutterDocumentReaderCoreFullrfid.setDocumentType(DocumentType.passport);
await FlutterDocumentReaderCoreFullrfid.setRfidSettings(RfidSettings(true, true));
```
### Step 5: Start Scanning

Start scanning by calling the `startScanning` method:
```dart
await FlutterDocumentReaderCoreFullrfid.startScanning();
```
### Platform-Specific Details

* Android: Add the following line to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.CAMERA" />
```
* iOS: Add the following line to your `Info.plist` file:
```xml
<key>NSCameraUsageDescription</key>
<string>This app needs camera access to scan documents</string>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:flutter_document_reader_core_fullrfid/flutter_document_reader_core_fullrfid.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatefulWidget {
  @override
  _MyAppState createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Document Reader Demo'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              ElevatedButton(
                onPressed: _startScanning,
                child: Text('Start Scanning'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: _stopScanning,
                child: Text('Stop Scanning'),
              ),
            ],
          ),
        ),
      ),
    );
  }

  Future<void> _startScanning() async {
    // Initialize the package
    await FlutterDocumentReaderCoreFullrfid.init();

    // Set the document type to passport
    await FlutterDocumentReaderCoreFullrfid.setDocumentType(DocumentType.passport);

    // Set the RFID settings to enable RFID scanning
    await FlutterDocumentReaderCoreFullrfid.setRfidSettings(RfidSettings(true, true));

    // Start scanning
    await FlutterDocumentReaderCoreFullrfid.startScanning();
  }

  Future<void> _stopScanning() async {
    // Stop scanning
    await FlutterDocumentReaderCoreFullrfid.stopScanning();
  }
}

// The application flow is as follows:
// 1. The user clicks the "Start Scanning" button, which calls the _startScanning method.
// 2. The _startScanning method initializes the package, sets the document type to passport, and enables RFID scanning.
// 3. The package starts scanning for documents.
// 4. When a document is detected, the package processes the document and extracts the relevant information.
// 5. The user can then click the "Stop Scanning" button to stop the scanning process.
// 6. The _stopScanning method stops the scanning process and releases any system resources used by the package.
```
