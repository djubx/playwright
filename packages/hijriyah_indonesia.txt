# Hijriyah Indonesia Flutter Package
The `hijriyah_indonesia` Flutter package is a useful tool for converting Gregorian dates to Hijriyah dates, which are widely used in Indonesia. This package is particularly useful for developers who want to create apps that cater to the Indonesian market or for apps that require Hijriyah date conversions.

## Features
- Converts Gregorian dates to Hijriyah dates
- Supports date range from 1 January 1900 to 31 December 2099
- Easy to use and integrate into existing Flutter projects

## When to Use
This package is ideal for apps that require Hijriyah date conversions, such as:

- Islamic calendar apps
- Prayer time apps
- Indonesian cultural or religious apps
- Any app that requires Hijriyah date conversions

# Setting Up the Package
To use the `hijriyah_indonesia` package, follow these steps:

### Step 1: Add the Package to Your Project
Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  hijriyah_indonesia: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package
Import the package in your Dart file:
```dart
import 'package:hijriyah_indonesia/hijriyah_indonesia.dart';
```
### Step 3: Use the Package
Use the `Hijriyah` class to convert Gregorian dates to Hijriyah dates:
```dart
Hijriyah hijriyah = Hijriyah.fromDate(DateTime.now());
print(hijriyah.toString());
```
This will print the current date in Hijriyah format.

### Platform-Specific Details
This package works on both Android and iOS platforms without any additional configuration.

<-- START_MAIN -->
```dart
import 'package:flutter/material.dart';
import 'package:hijriyah_indonesia/hijriyah_indonesia.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  DateTime _selectedDate = DateTime.now();
  Hijriyah _hijriyahDate;

  @override
  void initState() {
    super.initState();
    _hijriyahDate = Hijriyah.fromDate(_selectedDate);
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Hijriyah Indonesia Demo',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Hijriyah Indonesia Demo'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              Text(
                'Selected Date:',
                style: TextStyle(fontSize: 20),
              ),
              Text(
                _selectedDate.toString(),
                style: TextStyle(fontSize: 20),
              ),
              SizedBox(height: 20),
              Text(
                'Hijriyah Date:',
                style: TextStyle(fontSize: 20),
              ),
              Text(
                _hijriyahDate.toString(),
                style: TextStyle(fontSize: 20),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () async {
                  final DateTime pickedDate = await showDatePicker(
                    context: context,
                    initialDate: _selectedDate,
                    firstDate: DateTime(1900),
                    lastDate: DateTime(2099),
                  );
                  if (pickedDate != null) {
                    setState(() {
                      _selectedDate = pickedDate;
                      _hijriyahDate = Hijriyah.fromDate(pickedDate);
                    });
                  }
                },
                child: Text('Select Date'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}
// This is the main entry point of the app.
// The app uses the RealFlutter class as the main widget.
// The RealFlutter class is a stateful widget that displays the current date and the corresponding Hijriyah date.
// The user can select a new date using the date picker, and the app will update the displayed dates accordingly.
// The Hijriyah date is converted from the selected date using the Hijriyah.fromDate method.
// The app uses the MaterialApp and Scaffold widgets to provide a basic material design layout.
// The app also uses the Text and ElevatedButton widgets to display the dates and provide a button for selecting a new date.
```
