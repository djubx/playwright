# Overview of dgis_map_kit Flutter Package
=============================================

The dgis_map_kit Flutter package is a powerful tool for integrating maps into your Flutter applications. It provides a range of features, including map rendering, marker placement, and geolocation services. This package is ideal for developers who want to create location-based apps, such as ride-hailing services, food delivery apps, or social media platforms with location-sharing features.

### Key Features

*   Map rendering with various tile providers (e.g., OpenStreetMap, Mapbox)
*   Marker placement with customizable icons and labels
*   Geolocation services for determining the user's current location
*   Support for both Android and iOS platforms

### When to Use This Package

Use the dgis_map_kit package when you need to integrate maps into your Flutter app, especially when you require features like marker placement, geolocation services, and customizable map tiles.

# Setting Up and Using dgis_map_kit
=====================================

### Step 1: Add the Package to Your pubspec.yaml File

To use the dgis_map_kit package, you need to add it to your project's `pubspec.yaml` file. Open the file and add the following line under the `dependencies` section:

```yml
dependencies:
  dgis_map_kit: ^1.0.0
```

Then, run `flutter pub get` in your terminal to install the package.

### Step 2: Import the Package in Your Dart File

To use the package, you need to import it in your Dart file. Add the following line at the top of your file:

```dart
import 'package:dgis_map_kit/dgis_map_kit.dart';
```

### Step 3: Create a Map Widget

To display a map, you need to create a `MapWidget` instance. Here's a basic example:

```dart
MapWidget(
  mapType: MapType.normal,
  initialCameraPosition: CameraPosition(
    target: LatLng(37.7749, -122.4194),
    zoom: 12,
  ),
)
```

This code creates a map widget with a normal map type and an initial camera position set to San Francisco.

### Platform-Specific Details

#### Android

To use the dgis_map_kit package on Android, you need to add the following permissions to your `AndroidManifest.xml` file:

```xml
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />
```

#### iOS

To use the dgis_map_kit package on iOS, you need to add the following lines to your `Info.plist` file:

```xml
<key>NSLocationWhenInUseUsageDescription</key>
<string>This app needs access to your location to work properly</string>
<key>NSLocationAlwaysUsageDescription</key>
<string>This app needs access to your location to work properly</string>
```

```dart
import 'package:flutter/material.dart';
import 'package:dgis_map_kit/dgis_map_kit.dart';

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  // Step 1: Create a map controller
  MapController _mapController = MapController();

  // Step 2: Create a map widget
  MapWidget _mapWidget;

  @override
  void initState() {
    super.initState();

    // Step 3: Initialize the map widget
    _mapWidget = MapWidget(
      mapType: MapType.normal,
      initialCameraPosition: CameraPosition(
        target: LatLng(37.7749, -122.4194),
        zoom: 12,
      ),
    );
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('DGIS Map Kit Example'),
      ),
      body: Stack(
        children: [
          // Step 4: Add the map widget to the stack
          _mapWidget,

          // Step 5: Add a button to the stack
          Positioned(
            bottom: 20,
            right: 20,
            child: FloatingActionButton(
              onPressed: () {
                // Step 6: Move the camera to a new position
                _mapController.moveCamera(
                  CameraPosition(
                    target: LatLng(37.7859, -122.4364),
                    zoom: 15,
                  ),
                );
              },
              child: Icon(Icons.location_pin),
            ),
          ),
        ],
      ),
    );
  }
}

// The application flow:
// 1. The user opens the app and sees a map of San Francisco.
// 2. The user clicks the floating action button to move the camera to a new position.
// 3. The camera moves to the new position, showing a different part of the map.
// 4. The user can continue to interact with the map, zooming in and out, and moving the camera to different positions.

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'DGIS Map Kit Example',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: RealFlutter(),
    );
  }
}
```
