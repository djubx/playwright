# Tabler Icons Flutter Package
================================

The `tabler_icons` Flutter package is a collection of icons designed by Tabler. It provides a wide range of icons that can be used in Flutter applications. The package is highly customizable and can be used in various contexts.

## When to Use
--------------

You can use the `tabler_icons` package when you need to add icons to your Flutter application. It's particularly useful when you want to use a consistent set of icons throughout your app.

## Features
------------

*   Highly customizable
*   Wide range of icons available
*   Can be used in various contexts
*   Easy to use

# Tutorial
==========

## Setup
--------

To use the `tabler_icons` package, you need to add it to your `pubspec.yaml` file:

```yml
dependencies:
  flutter:
    sdk: flutter
  tabler_icons: ^1.1.1
```

Then, run `flutter pub get` to get the package.

## Usage
-----

To use an icon from the `tabler_icons` package, you can use the `TablerIcon` widget:

```dart
import 'package:flutter/material.dart';
import 'package:tabler_icons/tabler_icons.dart';

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: TablerIcon(
          icon: TablerIcons.airplane,
          size: 50,
        ),
      ),
    );
  }
}
```

This will display an airplane icon.

## Platform-Specific Details
---------------------------

The `tabler_icons` package works on both Android and iOS platforms.

### Android

No additional configuration is required for Android.

### iOS

No additional configuration is required for iOS.

## Optimizations
--------------

You can optimize the icon rendering by using the `color` property:

```dart
TablerIcon(
  icon: TablerIcons.airplane,
  size: 50,
  color: Colors.blue,
)
```

This will render the icon in blue color.

```dart
import 'package:flutter/material.dart';
import 'package:tabler_icons/tabler_icons.dart';

void main() {
  runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Tabler Icons Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: RealFlutter(),
    );
  }
}

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // Create a scaffold with a centered icon
    return Scaffold(
      body: Center(
        // Use the TablerIcon widget to display an icon
        child: TablerIcon(
          // Choose an icon from the TablerIcons class
          icon: TablerIcons.airplane,
          // Set the icon size
          size: 50,
          // Set the icon color
          color: Colors.blue,
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user runs the app.
// 2. The MyApp widget is rendered, which contains the RealFlutter widget.
// 3. The RealFlutter widget creates a scaffold with a centered icon.
// 4. The icon is displayed using the TablerIcon widget.
// 5. The icon is chosen from the TablerIcons class.
// 6. The icon size and color are set.
// 7. The icon is rendered on the screen.

// In summary, this code creates a simple Flutter app that displays a blue airplane icon in the center of the screen.
```
