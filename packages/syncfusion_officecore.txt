# Syncfusion OfficeCore Flutter Package
=====================================

The Syncfusion OfficeCore Flutter package is a powerful tool for creating office-related applications in Flutter. It provides a range of features, including document editing, spreadsheet creation, and presentation management.

## When to Use This Package
---------------------------

This package is ideal for developers who want to create office-related applications in Flutter. It can be used to create a wide range of applications, including:

* Document editors
* Spreadsheet applications
* Presentation software
* Office suites

## Features
------------

The Syncfusion OfficeCore Flutter package includes a range of features, including:

* Document editing: Create, edit, and save documents in various formats, including Word, PDF, and Text.
* Spreadsheet creation: Create, edit, and save spreadsheets in various formats, including Excel, CSV, and PDF.
* Presentation management: Create, edit, and save presentations in various formats, including PowerPoint, PDF, and Image.

# Setting Up the Package
-------------------------

To use the Syncfusion OfficeCore Flutter package, follow these steps:

### Step 1: Add the Package to Your Project

Add the following dependency to your `pubspec.yaml` file:
```yml
dependencies:
  syncfusion_officecore: ^18.2.0.44
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:syncfusion_officecore/officecore.dart';
```
### Step 3: Initialize the Package

Initialize the package in your `main` function:
```dart
void main() {
  OfficeCore.init();
  runApp(MyApp());
}
```
### Platform-Specific Details

#### Android

Add the following code to your `AndroidManifest.xml` file:
```xml
<application>
  <meta-data
    android:name="flutter.embedded_views"
    android:resource="@array/flutter_embedded_views"/>
</application>
```
#### iOS

Add the following code to your `Info.plist` file:
```xml
<key>UIAppFonts</key>
<array>
  <string>OpenSans-Regular.ttf</string>
</array>
```
### Required Configurations

#### Document Editing

To enable document editing, add the following code to your `main` function:
```dart
void main() {
  OfficeCore.init();
  OfficeCore.documentEditingEnabled = true;
  runApp(MyApp());
}
```
#### Spreadsheet Creation

To enable spreadsheet creation, add the following code to your `main` function:
```dart
void main() {
  OfficeCore.init();
  OfficeCore.spreadsheetCreationEnabled = true;
  runApp(MyApp());
}
```
#### Presentation Management

To enable presentation management, add the following code to your `main` function:
```dart
void main() {
  OfficeCore.init();
  OfficeCore.presentationManagementEnabled = true;
  runApp(MyApp());
}
```

```dart
import 'package:flutter/material.dart';
import 'package:syncfusion_officecore/officecore.dart';

void main() {
  OfficeCore.init();
  OfficeCore.documentEditingEnabled = true;
  OfficeCore.spreadsheetCreationEnabled = true;
  OfficeCore.presentationManagementEnabled = true;
  runApp(RealFlutter());
}

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Real Flutter',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Real Flutter'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              ElevatedButton(
                onPressed: () {
                  // Create a new document
                  OfficeCore.createDocument().then((document) {
                    // Edit the document
                    document.edit();
                  });
                },
                child: Text('Create Document'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  // Create a new spreadsheet
                  OfficeCore.createSpreadsheet().then((spreadsheet) {
                    // Edit the spreadsheet
                    spreadsheet.edit();
                  });
                },
                child: Text('Create Spreadsheet'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  // Create a new presentation
                  OfficeCore.createPresentation().then((presentation) {
                    // Edit the presentation
                    presentation.edit();
                  });
                },
                child: Text('Create Presentation'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user clicks on one of the buttons to create a new document, spreadsheet, or presentation.
// 2. The OfficeCore package creates a new instance of the corresponding object (document, spreadsheet, or presentation).
// 3. The user can then edit the object using the OfficeCore package's editing features.
// 4. Once the user has finished editing, they can save the object to a file or share it with others.
```

// The above code creates a simple Flutter application with three buttons: Create Document, Create Spreadsheet, and Create Presentation. When the user clicks on one of these buttons, the OfficeCore package creates a new instance of the corresponding object (document, spreadsheet, or presentation) and allows the user to edit it using the package's editing features.
