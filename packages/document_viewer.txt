# Document Viewer Flutter Package
=====================================

The `document_viewer` Flutter package is a powerful tool for displaying various types of documents within a Flutter application. It supports a wide range of file formats, including PDF, DOCX, XLSX, PPTX, and more.

### When to Use

This package is ideal for applications that require displaying documents, such as:

* Document management systems
* E-learning platforms
* Office suites
* File explorers

### Features

* Supports multiple document formats
* Zooming and panning capabilities
* Page navigation
* Document searching
* Landscape and portrait mode support

# Setting Up the Document Viewer Package
=============================================

### Step 1: Add the Package to Your Project

Add the following dependency to your `pubspec.yaml` file:

```yml
dependencies:
  flutter:
    sdk: flutter
  document_viewer: ^[latest_version]
```

Replace `[latest_version]` with the latest version available on pub.dev.

### Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:document_viewer/document_viewer.dart';
```

### Step 3: Platform-Specific Configurations

#### Android

Add the following code to your `AndroidManifest.xml` file:

```xml
<application
  ...
  android:hardwareAccelerated="true"
  ...
>
```

#### iOS

Add the following code to your `Info.plist` file:

```xml
<key>UIAppFonts</key>
<array>
  <string>OpenSans-Regular.ttf</string>
</array>
```

### Step 4: Using the Document Viewer

Create a `DocumentViewer` widget and pass the document file path or bytes:

```dart
DocumentViewer(
  filePath: 'path/to/document.pdf',
  // or
  fileBytes: documentBytes,
)
```

```dart
import 'package:flutter/material.dart';
import 'package:document_viewer/document_viewer.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  String _filePath = 'path/to/document.pdf';

  // Step 1: Create a DocumentViewer widget
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Document Viewer Example'),
        ),
        body: Center(
          child: ElevatedButton(
            onPressed: () {
              // Step 2: Navigate to the DocumentViewer page
              Navigator.push(
                context,
                MaterialPageRoute(builder: (context) => DocumentViewerPage(_filePath)),
              );
            },
            child: Text('Open Document'),
          ),
        ),
      ),
    );
  }
}

class DocumentViewerPage extends StatefulWidget {
  final String filePath;

  DocumentViewerPage(this.filePath);

  @override
  _DocumentViewerPageState createState() => _DocumentViewerPageState();
}

class _DocumentViewerPageState extends State<DocumentViewerPage> {
  // Step 3: Create a DocumentViewer widget
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Document Viewer'),
      ),
      body: Center(
        child: DocumentViewer(
          filePath: widget.filePath,
        ),
      ),
    );
  }
}
```
