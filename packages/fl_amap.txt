In summary, the code flows as follows:

*   The user opens the app, and the AMap is initialized with the provided API key.
*   The AMapWidget is displayed, showing the map.
*   When the user presses the floating action button, a marker is added to the map at the specified location.
*   The user can interact with the map, such as zooming in and out, and moving the map around.
*   The marker is displayed on the map, and the user can tap on it to see the title.
