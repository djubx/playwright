# enx_flutter_plugin Package Overview
=====================================

The `enx_flutter_plugin` package is a Flutter plugin that enables developers to integrate Enx video conferencing capabilities into their Flutter applications. This plugin provides a simple and efficient way to add video conferencing features to your app, allowing users to connect with each other in real-time.

## Key Features

*   Video conferencing: Enable multiple users to connect and communicate with each other through video and audio.
*   Screen sharing: Allow users to share their screens with others in the conference.
*   Chat messaging: Enable users to send and receive text messages during the conference.

## When to Use

The `enx_flutter_plugin` package is ideal for applications that require video conferencing capabilities, such as:

*   Virtual meeting apps
*   Online education platforms
*   Remote collaboration tools
*   Social media apps with video conferencing features

# Setting Up the enx_flutter_plugin Package
==========================================

To use the `enx_flutter_plugin` package in your Flutter application, follow these steps:

### Step 1: Add the Package to Your pubspec.yaml File

Add the following line to the `dependencies` section of your `pubspec.yaml` file:

```yml
dependencies:
  flutter:
    sdk: flutter
  enx_flutter_plugin: ^1.0.0
```

### Step 2: Import the Package in Your Dart File

Import the package in your Dart file:

```dart
import 'package:enx_flutter_plugin/enx_flutter_plugin.dart';
```

### Step 3: Initialize the Enx Plugin

Initialize the Enx plugin before using it:

```dart
EnxFlutterPlugin enx = EnxFlutterPlugin();
enx.init();
```

### Platform-Specific Configurations

#### Android

Add the following permissions to your `AndroidManifest.xml` file:

```xml
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.RECORD_AUDIO" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
```

#### iOS

Add the following lines to your `Info.plist` file:

```xml
<key>NSCameraUsageDescription</key>
<string>Can I use the camera please?</string>
<key>NSMicrophoneUsageDescription</key>
<string>Can I use the microphone please?</string>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:enx_flutter_plugin/enx_flutter_plugin.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatefulWidget {
  @override
  _MyAppState createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  EnxFlutterPlugin enx = EnxFlutterPlugin();

  // Initialize the Enx plugin
  void _initEnx() async {
    await enx.init();
  }

  // Join a conference
  void _joinConference() async {
    // Replace with your conference ID
    String conferenceId = "your_conference_id";

    // Replace with your user ID
    String userId = "your_user_id";

    // Set the conference mode (video or audio)
    EnxConferenceMode mode = EnxConferenceMode.Video;

    // Join the conference
    await enx.joinConference(conferenceId, userId, mode);
  }

  // Leave the conference
  void _leaveConference() async {
    await enx.leaveConference();
  }

  // Send a chat message
  void _sendChatMessage() async {
    // Replace with your message
    String message = "Hello, world!";

    // Send the message
    await enx.sendChatMessage(message);
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text("Enx Flutter Plugin Example"),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              ElevatedButton(
                onPressed: _initEnx,
                child: Text("Initialize Enx"),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: _joinConference,
                child: Text("Join Conference"),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: _leaveConference,
                child: Text("Leave Conference"),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: _sendChatMessage,
                child: Text("Send Chat Message"),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user initializes the Enx plugin by clicking the "Initialize Enx" button.
// 2. The user joins a conference by clicking the "Join Conference" button.
// 3. The user can send chat messages to other users in the conference by clicking the "Send Chat Message" button.
// 4. The user can leave the conference by clicking the "Leave Conference" button.
```
