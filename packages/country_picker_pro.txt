# Country Picker Pro Package
The `country_picker_pro` package is a Flutter library that provides a simple and customizable way to display a list of countries and select one. It is useful when you need to collect country information from users, such as in registration forms or shipping address inputs.

## Features
- Displays a list of countries with their names and flags
- Allows users to search for countries by name
- Supports customization of country list and search functionality
- Works on both Android and iOS platforms

## When to Use
Use this package when you need to collect country information from users, such as:

- Registration forms
- Shipping address inputs
- Travel or tourism apps
- E-commerce apps with international shipping

# Setting Up Country Picker Pro
To use the `country_picker_pro` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File
Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  country_picker_pro: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package
Import the package in your Dart file:
```dart
import 'package:country_picker_pro/country_picker_pro.dart';
```
### Step 3: Use the CountryPickerPro Widget
Use the `CountryPickerPro` widget to display the country list:
```dart
CountryPickerPro(
  onCountrySelected: (country) {
    // Handle country selection
  },
)
```
### Platform-Specific Details
- On Android, the country list is displayed as a bottom sheet.
- On iOS, the country list is displayed as a modal view.

### Configurations and Optimizations
- You can customize the country list by providing a list of country codes to the `countryCodes` property.
- You can customize the search functionality by providing a search delegate to the `searchDelegate` property.

```dart
import 'package:flutter/material.dart';
import 'package:country_picker_pro/country_picker_pro.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  String _selectedCountry;

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Country Picker Pro Example'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Text(
                _selectedCountry != null
                    ? 'Selected Country: $_selectedCountry'
                    : 'No country selected',
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  showCountryPickerPro(
                    context: context,
                    onCountrySelected: (country) {
                      setState(() {
                        _selectedCountry = country.name;
                      });
                    },
                  );
                },
                child: Text('Select Country'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// 5. The app uses the `CountryPickerPro` widget to display the country list.
// 6. The app customizes the country list by providing a list of country codes.
// 7. The app customizes the search functionality by providing a search delegate.

// Summary:
// The app uses the `country_picker_pro` package to display a list of countries and select one.
```
