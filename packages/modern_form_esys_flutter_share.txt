# Modern Form Esys Flutter Share Package
=====================================

The `modern_form_esys_flutter_share` package is a Flutter plugin that allows you to share files, text, and URLs between your app and other apps installed on the user's device. This package is built on top of the `esys_flutter_share` package and provides a more modern and streamlined API.

### When to Use This Package

You can use this package when you need to share data from your app with other apps, such as:

* Sharing a file with a friend or colleague
* Posting a link to social media
* Sending a text message or email with a attachment

### Features

* Share files, text, and URLs
* Supports multiple platforms, including Android and iOS
* Simple and modern API

# Setup and Usage
---------------

### Step 1: Add the Package to Your Pubspec.yaml File

```yml
dependencies:
  modern_form_esys_flutter_share: ^1.0.0
```

### Step 2: Import the Package in Your Dart File

```dart
import 'package:modern_form_esys_flutter_share/modern_form_esys_flutter_share.dart';
```

### Step 3: Share Data Using the Package

```dart
void _shareData() async {
  await ModernFormEsysFlutterShare.share(
    text: 'Hello, world!',
    title: 'Share Example',
  );
}
```

### Platform-Specific Details

#### Android

* Make sure to add the following permissions to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
```
* If you are targeting Android 10 or later, you need to add the following code to your `AndroidManifest.xml` file:
```xml
<application
  ...
  android:requestLegacyExternalStorage="true"
>
```
#### iOS

* Make sure to add the following code to your `Info.plist` file:
```xml
<key>NSPhotoLibraryUsageDescription</key>
<string>This app needs access to the photo library to share images.</string>
```
### Optimizations

* Make sure to handle errors and exceptions properly when using the package.
* Use the `await` keyword to wait for the share operation to complete.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:modern_form_esys_flutter_share/modern_form_esys_flutter_share.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Share Example',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  void _shareText() async {
    // Share text using the package
    await ModernFormEsysFlutterShare.share(
      text: 'Hello, world!',
      title: 'Share Example',
    );
  }

  void _shareFile() async {
    // Share a file using the package
    await ModernFormEsysFlutterShare.share(
      file: File('path/to/file'),
      title: 'Share Example',
    );
  }

  void _shareUrl() async {
    // Share a URL using the package
    await ModernFormEsysFlutterShare.share(
      url: 'https://example.com',
      title: 'Share Example',
    );
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Share Example'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            ElevatedButton(
              onPressed: _shareText,
              child: Text('Share Text'),
            ),
            ElevatedButton(
              onPressed: _shareFile,
              child: Text('Share File'),
            ),
            ElevatedButton(
              onPressed: _shareUrl,
              child: Text('Share URL'),
            ),
          ],
        ),
      ),
    );
  }
}

// The app starts by running the main function, which creates a new instance of the MyApp widget.
// The MyApp widget builds a MaterialApp with a title and a theme.
// The home property of the MaterialApp is set to an instance of the MyHomePage widget.
// The MyHomePage widget is a stateful widget that builds a Scaffold with an AppBar and a body.
// The body of the Scaffold contains a Column with three ElevatedButton widgets.
// Each ElevatedButton widget has an onPressed property that calls a different function when pressed.
// The _shareText function shares text using the ModernFormEsysFlutterShare package.
// The _shareFile function shares a file using the ModernFormEsysFlutterShare package.
// The _shareUrl function shares a URL using the ModernFormEsysFlutterShare package.
```
