### Description
The `instagram_mention` Flutter package is a powerful tool for creating Instagram-like mentions in your Flutter applications. It allows users to mention other users in text fields, just like in Instagram comments. This package is perfect for social media apps, chat apps, or any app that requires user mentions.

The package provides a simple and easy-to-use API for creating mentions. It includes features such as:

*   Mentioning users with a specific username
*   Customizable mention styles
*   Support for multiple mention types (e.g., users, hashtags, etc.)
*   Easy integration with existing text fields

You can use this package whenever you need to create a text field that allows users to mention other users. For example, in a social media app, you can use this package to create a comment field that allows users to mention their friends.

### Tutorial
To use the `instagram_mention` package, follow these steps:

1.  Add the package to your `pubspec.yaml` file:

    ```yml
dependencies:
  flutter:
    sdk: flutter
  instagram_mention: ^1.0.0
```

2.  Run `flutter pub get` to install the package.

3.  Import the package in your Dart file:

    ```dart
import 'package:instagram_mention/instagram_mention.dart';
```

4.  Create a `MentionTextField` widget:

    ```dart
MentionTextField(
  decoration: InputDecoration(
    border: OutlineInputBorder(),
    labelText: 'Comment',
  ),
  mentionTypes: [
    MentionType(
      trigger: '@',
      style: TextStyle(color: Colors.blue),
      data: [
        MentionData(
          id: '1',
          name: 'John Doe',
          username: 'johndoe',
        ),
        MentionData(
          id: '2',
          name: 'Jane Doe',
          username: 'janedoe',
        ),
      ],
    ),
  ],
)
```

5.  Customize the mention styles and data as needed.

Note: For Android, you need to add the following code to your `AndroidManifest.xml` file:

```xml
<application
  ...
  android:hardwareAccelerated="true"
  ...
>
```

For iOS, you need to add the following code to your `Info.plist` file:

```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

```dart
import 'package:flutter/material.dart';
import 'package:instagram_mention/instagram_mention.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Instagram Mention Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _mentionController = MentionController();

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Instagram Mention Demo'),
      ),
      body: Padding(
        padding: const EdgeInsets.all(16.0),
        child: MentionTextField(
          controller: _mentionController,
          decoration: InputDecoration(
            border: OutlineInputBorder(),
            labelText: 'Comment',
          ),
          mentionTypes: [
            MentionType(
              trigger: '@',
              style: TextStyle(color: Colors.blue),
              data: [
                MentionData(
                  id: '1',
                  name: 'John Doe',
                  username: 'johndoe',
                ),
                MentionData(
                  id: '2',
                  name: 'Jane Doe',
                  username: 'janedoe',
                ),
              ],
            ),
          ],
        ),
      ),
    );
  }
}
```

// The app starts by running the `main` function, which creates a `MaterialApp` with a `MyHomePage` widget.
// The `MyHomePage` widget is a stateful widget that contains a `MentionTextField` widget.
// The `MentionTextField` widget is controlled by a `MentionController`, which is used to manage the mention data.
// The `MentionTextField` widget is decorated with an `InputDecoration` that includes a border and a label.
// The `mentionTypes` property of the `MentionTextField` widget is used to define the mention types, including the trigger, style, and data.
// The `data` property of the `MentionType` object is used to define the mention data, including the id, name, and username.
// When the user types a mention trigger (e.g., '@'), the `MentionTextField` widget displays a list of mention suggestions based on the defined data.
