# Twitter Intent Flutter Package
=====================================

The `twitter_intent` Flutter package allows you to easily integrate Twitter intent functionality into your Flutter app. With this package, you can create tweets, replies, and direct messages with just a few lines of code.

### When to Use This Package

You can use this package in various scenarios, such as:

* Sharing content from your app on Twitter
* Allowing users to reply to tweets from within your app
* Sending direct messages to other Twitter users

### Features

The `twitter_intent` package provides the following features:

* Create tweets with text, images, and URLs
* Reply to tweets with text, images, and URLs
* Send direct messages with text and images
* Support for Android and iOS platforms

# Setup and Usage
=====================

To use the `twitter_intent` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  twitter_intent: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:twitter_intent/twitter_intent.dart';
```
### Step 3: Create a Twitter Intent

Create a Twitter intent using the `TwitterIntent` class:
```dart
TwitterIntent intent = TwitterIntent(
  text: 'Hello, world!',
  hashtags: ['flutter', 'twitter'],
);
```
### Step 4: Launch the Intent

Launch the intent using the `launch` method:
```dart
intent.launch();
```
This will open the Twitter app with the pre-filled tweet.

### Platform-Specific Details

#### Android

On Android, you need to add the following intent filter to your `AndroidManifest.xml` file:
```xml
<intent-filter>
  <action android:name="android.intent.action.VIEW" />
  <category android:name="android.intent.category.DEFAULT" />
  <category android:name="android.intent.category.BROWSABLE" />
  <data android:scheme="twitter" android:host="intent" />
</intent-filter>
```
#### iOS

On iOS, you need to add the following URL scheme to your `Info.plist` file:
```xml
<key>CFBundleURLTypes</key>
<array>
  <dict>
    <key>CFBundleURLName</key>
    <string>twitter</string>
    <key>CFBundleURLSchemes</key>
    <array>
      <string>twitter</string>
    </array>
  </dict>
</array>
```

```dart
import 'package:flutter/material.dart';
import 'package:twitter_intent/twitter_intent.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Twitter Intent Example'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              ElevatedButton(
                onPressed: () {
                  // Create a tweet with text and hashtags
                  TwitterIntent intent = TwitterIntent(
                    text: 'Hello, world!',
                    hashtags: ['flutter', 'twitter'],
                  );
                  intent.launch();
                },
                child: Text('Tweet with text and hashtags'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  // Reply to a tweet with text and hashtags
                  TwitterIntent intent = TwitterIntent(
                    text: 'Reply to tweet',
                    hashtags: ['flutter', 'twitter'],
                    inReplyToStatusId: '1234567890',
                  );
                  intent.launch();
                },
                child: Text('Reply to tweet with text and hashtags'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  // Send a direct message with text
                  TwitterIntent intent = TwitterIntent(
                    text: 'Hello, world!',
                    screenName: 'twitterhandle',
                  );
                  intent.launch();
                },
                child: Text('Send direct message with text'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// The app starts by creating a Material app with a Scaffold.
// The Scaffold has an AppBar with a title and a body with a Center widget.
// The Center widget contains a Column with three ElevatedButton widgets.
// Each ElevatedButton widget has an onPressed callback that creates a TwitterIntent object and launches it.
// The TwitterIntent object is created with different parameters depending on the button pressed.
// The first button creates a tweet with text and hashtags.
// The second button replies to a tweet with text and hashtags.
// The third button sends a direct message with text.
```
