# Insta Login Flutter Package
The insta_login Flutter package is a convenient library that allows developers to easily integrate Instagram login functionality into their Flutter applications. With this package, users can authenticate with their Instagram accounts and grant access to their profile information.

## When to Use
This package is ideal for applications that require Instagram login functionality, such as social media apps, photo editing apps, or any app that needs to access Instagram user data.

## Features
The insta_login package provides the following features:

* Instagram login functionality
* Access to user profile information (e.g., username, profile picture, bio)
* Support for Android and iOS platforms

# Setup and Usage
To use the insta_login package, follow these steps:

### Step 1: Add the Package to Your Project
Add the insta_login package to your pubspec.yaml file:
```yml
dependencies:
  insta_login: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Configure Instagram App Settings
Create an Instagram app on the Facebook Developer Platform and obtain an app ID and app secret. Configure the app settings to enable Instagram login.

### Step 3: Initialize the Package
Import the package and initialize it with your app ID and app secret:
```dart
import 'package:insta_login/insta_login.dart';

void main() {
  InstaLogin.initialize(
    appId: 'YOUR_APP_ID',
    appSecret: 'YOUR_APP_SECRET',
  );
  runApp(MyApp());
}
```
### Step 4: Use the Package
Use the `InstaLogin` class to authenticate users and access their profile information:
```dart
class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Insta Login Demo'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () async {
            final user = await InstaLogin.login();
            print(user.username);
          },
          child: Text('Login with Instagram'),
        ),
      ),
    );
  }
}
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:insta_login/insta_login.dart';

void main() {
  InstaLogin.initialize(
    appId: 'YOUR_APP_ID',
    appSecret: 'YOUR_APP_SECRET',
  );
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Insta Login Demo'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () async {
            // Step 1: Login with Instagram
            final user = await InstaLogin.login();

            // Step 2: Get user profile information
            print(user.username);
            print(user.profilePicture);
            print(user.bio);

            // Step 3: Display user profile information
            Navigator.push(
              context,
              MaterialPageRoute(builder: (context) => UserProfilePage(user)),
            );
          },
          child: Text('Login with Instagram'),
        ),
      ),
    );
  }
}

class UserProfilePage extends StatelessWidget {
  final User user;

  UserProfilePage(this.user);

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('User Profile'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Image.network(user.profilePicture),
            Text(user.username),
            Text(user.bio),
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user clicks the "Login with Instagram" button.
// 2. The InstaLogin package authenticates the user and grants access to their profile information.
// 3. The user's profile information is displayed on the UserProfilePage.
// 4. The user can view their profile picture, username, and bio.
```
