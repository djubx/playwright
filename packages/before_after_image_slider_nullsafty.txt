**Before After Image Slider Null Safety Flutter Package**
===========================================================

The `before_after_image_slider_nullsafety` package is a Flutter widget that allows users to compare two images by sliding a divider between them. This package is useful for showcasing before-and-after comparisons, such as makeup tutorials, home renovations, or product demos.

**Key Features:**

*   Null safety support
*   Customizable divider color and width
*   Support for various image formats (e.g., JPEG, PNG, GIF)
*   Works on both Android and iOS platforms

**When to Use This Package:**

*   When you need to display before-and-after comparisons
*   When you want to showcase product demos or tutorials
*   When you need a customizable and easy-to-use image comparison widget

**Setup and Usage**
-------------------

### Step 1: Add the Package to Your Project

Add the following dependency to your `pubspec.yaml` file:

```yml
dependencies:
  before_after_image_slider_nullsafety: ^1.0.0
```

Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:before_after_image_slider_nullsafety/before_after_image_slider_nullsafety.dart';
```

### Step 3: Use the Widget

Use the `BeforeAfterImageSlider` widget in your Flutter app:

```dart
BeforeAfterImageSlider(
  beforeImage: Image.asset('assets/before_image.jpg'),
  afterImage: Image.asset('assets/after_image.jpg'),
  dividerColor: Colors.white,
  dividerWidth: 2.0,
)
```

### Platform-Specific Details

*   Android: No additional configuration is required.
*   iOS: No additional configuration is required.

### Optimizations

*   Use images with the same dimensions for optimal performance.
*   Avoid using large images to prevent memory issues.

import 'package:flutter/material.dart';
import 'package:before_after_image_slider_nullsafety/before_after_image_slider_nullsafety.dart';

  runApp(MyApp());

  @override
  Widget build(BuildContext context) {

      theme: ThemeData(

      ),
      home: MyHomePage(),

class MyHomePage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Before After Image Slider Demo'),

      body: Center(
        child: BeforeAfterImageSlider(
          // Before image
          beforeImage: Image.asset('assets/before_image.jpg'),
          // After image
          afterImage: Image.asset('assets/after_image.jpg'),
          // Divider color
          dividerColor: Colors.white,
          // Divider width
          dividerWidth: 2.0,
        ),
      ),
    );
  }
}
