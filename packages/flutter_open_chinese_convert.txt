# Flutter Open Chinese Convert Package
=====================================

The `flutter_open_chinese_convert` package is a Flutter library that provides a simple and efficient way to convert between Traditional Chinese and Simplified Chinese characters. This package is useful for developers who want to support both Traditional and Simplified Chinese languages in their Flutter apps.

## Features

*   Supports conversion between Traditional Chinese and Simplified Chinese characters
*   Easy to use and integrate into existing Flutter projects
*   Works on both Android and iOS platforms

## When to Use

This package is useful when you want to:

*   Display text in both Traditional and Simplified Chinese languages in your app
*   Allow users to input text in either Traditional or Simplified Chinese characters
*   Convert text from one Chinese language to another

# Tutorial: Using the Flutter Open Chinese Convert Package
=====================================================

## Step 1: Add the Package to Your Flutter Project

To use the `flutter_open_chinese_convert` package, you need to add it to your Flutter project. You can do this by adding the following line to your `pubspec.yaml` file:

```yml
dependencies:
  flutter_open_chinese_convert: ^1.0.0
```

Then, run `flutter pub get` to get the package.

## Step 2: Import the Package

To use the package, you need to import it into your Dart file. You can do this by adding the following line at the top of your file:

```dart
import 'package:flutter_open_chinese_convert/flutter_open_chinese_convert.dart';
```

## Step 3: Use the Package

To convert text from one Chinese language to another, you can use the `convert` function provided by the package. Here is an example:

```dart
String traditionalChineseText = 'Hello';
String simplifiedChineseText = ChineseConverter.convert(traditionalChineseText, ChineseType.Simplified);
print(simplifiedChineseText);
```

This code converts the text 'Hello' from Traditional Chinese to Simplified Chinese.

## Platform-Specific Details

The `flutter_open_chinese_convert` package works on both Android and iOS platforms. However, you need to configure your project settings to support Chinese languages.

### Android

To support Chinese languages on Android, you need to add the following lines to your `android/app/src/main/AndroidManifest.xml` file:

```xml
<supports-screens
    android:anyDensity="true"
    android:largeScreens="true"
    android:normalScreens="true"
    android:smallScreens="true"
    android:xlargeScreens="true" />

<uses-permission android:name="android.permission.INTERNET" />
```

### iOS

To support Chinese languages on iOS, you need to add the following lines to your `ios/Runner/Info.plist` file:

```xml
<key>CFBundleDevelopmentRegion</key>
<string>en</string>

<key>CFBundleLocalizations</key>
<array>
    <string>en</string>
    <string>zh-Hans</string>
    <string>zh-Hant</string>
</array>
```

```dart
import 'package:flutter/material.dart';
import 'package:flutter_open_chinese_convert/flutter_open_chinese_convert.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Open Chinese Convert Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _textController = TextEditingController();
  String _convertedText = '';

  void _convertText() {
    String originalText = _textController.text;
    String convertedText = ChineseConverter.convert(originalText, ChineseType.Simplified);
    setState(() {
      _convertedText = convertedText;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Flutter Open Chinese Convert Demo'),
      ),
      body: Padding(
        padding: const EdgeInsets.all(20.0),
        child: Column(
          children: <Widget>[
            TextField(
              controller: _textController,
              decoration: InputDecoration(
                labelText: 'Enter text to convert',
                border: OutlineInputBorder(),
              ),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _convertText,
              child: Text('Convert to Simplified Chinese'),
            ),
            SizedBox(height: 20),
            Text(
              _convertedText,
              style: TextStyle(fontSize: 24),
            ),
          ],
        ),
      ),
    );
  }
}

// This is the main entry point of the app.
// The app uses the Material Design theme and has a single page with a text field, a button, and a text display area.
// The user can enter text into the text field, and when the button is pressed, the text is converted to Simplified Chinese using the flutter_open_chinese_convert package.
// The converted text is then displayed in the text display area.
// The app demonstrates the use of the flutter_open_chinese_convert package to convert text from one Chinese language to another.

```
