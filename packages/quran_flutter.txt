# Quran Flutter Package
=======================

The `quran_flutter` package is a Flutter library that provides a simple and efficient way to display the Quran in your Flutter application. It includes features such as:

* Displaying Quranic text in Uthmani and Simple fonts
* Support for multiple translations and tafsir
* Ability to play audio recitations
* Customizable layout and design

This package is perfect for building Islamic apps, such as Quran readers, prayer timers, and Islamic education apps.

### When to use this package

You can use this package when you want to display the Quran in your Flutter app, and you need a simple and efficient way to do so. This package is also useful when you want to provide multiple translations and tafsir options to your users.

### Features

* Display Quranic text in Uthmani and Simple fonts
* Support for multiple translations and tafsir
* Ability to play audio recitations
* Customizable layout and design

# Tutorial
==========

In this tutorial, we will walk through the setup process and explain how to use the `quran_flutter` package.

### Step 1: Add the package to your pubspec.yaml file

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  flutter:
    sdk: flutter
  quran_flutter: ^1.0.0
```
Then, run `flutter pub get` to get the package.

### Step 2: Import the package

Import the package in your Dart file:
```dart
import 'package:quran_flutter/quran_flutter.dart';
```
### Step 3: Use the package

You can use the package by creating a `Quran` widget:
```dart
Quran(
  surah: 1,
  ayah: 1,
  font: QuranFont.uthmani,
  translation: QuranTranslation.en,
)
```
This will display the first ayah of the first surah in Uthmani font with English translation.

### Platform-specific details

* Android: You need to add the following line to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.INTERNET" />
```
* iOS: You need to add the following line to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```
### Required configurations or optimizations

* You need to add the following line to your `pubspec.yaml` file to use the audio recitation feature:
```yml
dependencies:
  flutter:
    sdk: flutter
  quran_flutter: ^1.0.0
  audioplayers: ^0.20.1
```
Then, run `flutter pub get` to get the package.

```dart
import 'package:flutter/material.dart';
import 'package:quran_flutter/quran_flutter.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Quran Flutter Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  int _surah = 1;
  int _ayah = 1;
  QuranFont _font = QuranFont.uthmani;
  QuranTranslation _translation = QuranTranslation.en;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Quran Flutter Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Quran(
              surah: _surah,
              ayah: _ayah,
              font: _font,
              translation: _translation,
            ),
            SizedBox(height: 20),
            Row(
              mainAxisAlignment: MainAxisAlignment.center,
              children: <Widget>[
                ElevatedButton(
                  onPressed: () {
                    setState(() {
                      _surah = 2;
                      _ayah = 1;
                    });
                  },
                  child: Text('Next Surah'),
                ),
                SizedBox(width: 10),
                ElevatedButton(
                  onPressed: () {
                    setState(() {
                      _font = QuranFont.simple;
                    });
                  },
                  child: Text('Change Font'),
                ),
                SizedBox(width: 10),
                ElevatedButton(
                  onPressed: () {
                    setState(() {
                      _translation = QuranTranslation.ar;
                    });
                  },
                  child: Text('Change Translation'),
                ),
              ],
            ),
          ],
        ),
      ),
    );
  }
}
```

// This is the main file of the app. It uses the Quran Flutter package to display the Quran.
// The app has a simple layout with a column of widgets in the center of the screen.
// The first widget is the Quran widget, which displays the Quranic text.
// The Quran widget takes four parameters: surah, ayah, font, and translation.
// The surah and ayah parameters determine which surah and ayah to display.
// The font parameter determines the font of the Quranic text.
// The translation parameter determines the translation of the Quranic text.
// The app also has three buttons: Next Surah, Change Font, and Change Translation.
// The Next Surah button changes the surah and ayah to the next surah.
// The Change Font button changes the font of the Quranic text.
// The Change Translation button changes the translation of the Quranic text.
