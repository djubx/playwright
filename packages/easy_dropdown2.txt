# Easy Dropdown2 Package Overview
=====================================

The `easy_dropdown2` package is a Flutter widget that provides a simple and customizable dropdown menu. It is designed to be easy to use and flexible, making it a great option for a wide range of applications.

### When to Use

The `easy_dropdown2` package is perfect for situations where you need to provide a list of options for the user to select from. This can include:

* Selecting a country or region
* Choosing a language
* Picking a color or theme
* And many more!

### Features

The `easy_dropdown2` package comes with a range of features, including:

* Customizable dropdown menu
* Support for multiple selection
* Ability to add icons to options
* Option to disable certain options
* And many more!

# Setting Up and Using Easy Dropdown2
=====================================

### Step 1: Add the Package to Your Pubspec.yaml File

To use the `easy_dropdown2` package, you need to add it to your `pubspec.yaml` file. You can do this by adding the following line:
```yml
dependencies:
  easy_dropdown2: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Next, you need to import the package in your Dart file. You can do this by adding the following line:
```dart
import 'package:easy_dropdown2/easy_dropdown2.dart';
```
### Step 3: Create a Dropdown Menu

To create a dropdown menu, you need to create a `EasyDropdown2` widget. Here is an example:
```dart
EasyDropdown2(
  title: Text('Select an option'),
  options: [
    DropdownOption(
      value: 'option1',
      child: Text('Option 1'),
    ),
    DropdownOption(
      value: 'option2',
      child: Text('Option 2'),
    ),
  ],
  onChanged: (value) {
    print(value);
  },
)
```
This will create a dropdown menu with two options. When an option is selected, the `onChanged` callback will be called with the selected value.

### Platform-Specific Details

The `easy_dropdown2` package works on both Android and iOS. However, there are some platform-specific details to keep in mind:

* On Android, the dropdown menu will be displayed as a popup window.
* On iOS, the dropdown menu will be displayed as a bottom sheet.

```dart
import 'package:flutter/material.dart';
import 'package:easy_dropdown2/easy_dropdown2.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Easy Dropdown2 Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  String _selectedValue;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Easy Dropdown2 Demo'),
      ),
      body: Center(
        child: EasyDropdown2(
          title: Text('Select an option'),
          options: [
            DropdownOption(
              value: 'option1',
              child: Text('Option 1'),
            ),
            DropdownOption(
              value: 'option2',
              child: Text('Option 2'),
            ),
          ],
          onChanged: (value) {
            setState(() {
              _selectedValue = value;
            });
          },
        ),
      ),
      bottomNavigationBar: BottomAppBar(
        child: Row(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text('Selected value: $_selectedValue'),
          ],
        ),
      ),
    );
  }
}

// The app starts by running the main function, which creates a new instance of the MyApp widget.
// The MyApp widget is a stateless widget that wraps the entire app.
// It uses the MaterialApp widget to provide a material design theme to the app.
// The home property of the MaterialApp widget is set to the MyHomePage widget, which is the main page of the app.
// The MyHomePage widget is a stateful widget that contains a scaffold with an app bar and a body.
// The body of the scaffold contains a center widget with an EasyDropdown2 widget.
// The EasyDropdown2 widget is used to create a dropdown menu with two options.
// When an option is selected, the onChanged callback is called with the selected value.
// The selected value is then stored in the _selectedValue variable and displayed in the bottom navigation bar.
```
