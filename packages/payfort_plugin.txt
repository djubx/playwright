
The code flow is as follows:

1. The user clicks the "Make Payment" button.
2. The PayFort plugin is initialized with the merchant ID and API key.
3. A payment request is created with the required parameters (amount, currency, payment method).
4. The payment request is processed using the PayFort plugin.
5. The payment response is received and checked for success or failure.
6. If the payment is successful, a success message is printed.
7. If the payment fails, a failure message is printed.
8. If an error occurs during payment processing, an error message is printed.
