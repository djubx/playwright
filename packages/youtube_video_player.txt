# youtube_video_player Flutter Package
=====================================

The `youtube_video_player` Flutter package allows developers to embed YouTube videos in their Flutter applications. This package provides a simple and efficient way to play YouTube videos without having to worry about the underlying complexities.

### When to Use

This package is ideal for applications that require playing YouTube videos, such as:

* Video tutorials
* Product demos
* Entertainment apps
* Educational apps

### Features

* Plays YouTube videos in a Flutter app
* Supports both Android and iOS platforms
* Provides a simple and intuitive API
* Allows for customization of the video player

# Setup and Usage
================

### Step 1: Add the Package to Your Project

Add the following dependency to your `pubspec.yaml` file:
```yml
dependencies:
  flutter:
    sdk: flutter
  youtube_video_player: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:youtube_video_player/youtube_video_player.dart';
```
### Step 3: Create a YouTube Video Player

Create a `YoutubeVideoPlayer` widget and pass the video ID:
```dart
YoutubeVideoPlayer(
  videoId: 'VIDEO_ID_HERE',
  width: 400,
  height: 300,
)
```
### Platform-Specific Details

* Android: Make sure to add the following permission to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.INTERNET" />
```
* iOS: No additional setup is required.

### Optimizations

* To improve performance, you can set the `cache` property to `true`:
```dart
YoutubeVideoPlayer(
  videoId: 'VIDEO_ID_HERE',
  width: 400,
  height: 300,
  cache: true,
)
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:youtube_video_player/youtube_video_player.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'YouTube Video Player Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  String _videoId = 'VIDEO_ID_HERE';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('YouTube Video Player Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            // Create a YouTube video player
            YoutubeVideoPlayer(
              videoId: _videoId,
              width: 400,
              height: 300,
              // Set the cache property to true for better performance
              cache: true,
            ),
            SizedBox(height: 20),
            // Button to change the video ID
            ElevatedButton(
              onPressed: () {
                setState(() {
                  _videoId = 'NEW_VIDEO_ID_HERE';
                });
              },
              child: Text('Change Video ID'),
            ),
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. The app starts and displays a YouTube video player with the initial video ID.
// 2. The user can change the video ID by pressing the "Change Video ID" button.
// 3. The app updates the video ID and reloads the video player with the new video.
// 4. The user can watch the new video and interact with the video player controls.
// 5. The app continues to run and display the video player until the user closes it.

```
