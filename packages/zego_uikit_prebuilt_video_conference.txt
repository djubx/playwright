# Zego UIKit Prebuilt Video Conference Flutter Package
=====================================================

The `zego_uikit_prebuilt_video_conference` Flutter package is a pre-built UI kit for video conferencing applications. It provides a simple and efficient way to integrate video conferencing functionality into your Flutter app.

### When to Use This Package

This package is ideal for developers who want to quickly add video conferencing features to their Flutter app without having to build everything from scratch. It's perfect for applications such as:

* Virtual meetings
* Online classes
* Remote training sessions
* Video consultations

### Features

The `zego_uikit_prebuilt_video_conference` package comes with the following features:

* Pre-built UI components for video conferencing
* Support for multiple video streams
* Audio and video controls (e.g., mute, unmute, switch camera)
* Screen sharing
* Chat functionality

# Setting Up the Package
-------------------------

To use the `zego_uikit_prebuilt_video_conference` package, follow these steps:

### Step 1: Add the Package to Your Flutter Project

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  flutter:
    sdk: flutter
  zego_uikit_prebuilt_video_conference: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:zego_uikit_prebuilt_video_conference/zego_uikit_prebuilt_video_conference.dart';
```
### Step 3: Initialize the Package

Initialize the package by calling the `init` method:
```dart
void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Zego UIKit Prebuilt Video Conference Demo',
      home: RealFlutter(),
    );
  }
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  @override
  void initState() {
    super.initState();
    ZegoUIKitPrebuiltVideoConference.init();
  }

  @override
  Widget build(BuildContext context) {
    // Build your UI here
  }
}
```
### Platform-Specific Configurations

For Android, add the following lines to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.RECORD_AUDIO" />
```
For iOS, add the following lines to your `Info.plist` file:
```xml
<key>NSCameraUsageDescription</key>
<string>This app needs camera access to work</string>
<key>NSMicrophoneUsageDescription</key>
<string>This app needs microphone access to work</string>
```

```dart
import 'package:flutter/material.dart';
import 'package:zego_uikit_prebuilt_video_conference/zego_uikit_prebuilt_video_conference.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Zego UIKit Prebuilt Video Conference Demo',
      home: RealFlutter(),
    );
  }
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  // Initialize the package
  @override
  void initState() {
    super.initState();
    ZegoUIKitPrebuiltVideoConference.init();
  }

  // Build the UI
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Zego UIKit Prebuilt Video Conference Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            // Join a room
            ElevatedButton(
              onPressed: () async {
                // Join a room with the room ID and user ID
                await ZegoUIKitPrebuiltVideoConference.joinRoom(
                  roomID: 'your_room_id',
                  userID: 'your_user_id',
                );
              },
              child: Text('Join Room'),
            ),
            // Leave a room
            ElevatedButton(
              onPressed: () async {
                // Leave the current room
                await ZegoUIKitPrebuiltVideoConference.leaveRoom();
              },
              child: Text('Leave Room'),
            ),
            // Switch camera
            ElevatedButton(
              onPressed: () async {
                // Switch the camera
                await ZegoUIKitPrebuiltVideoConference.switchCamera();
              },
              child: Text('Switch Camera'),
            ),
            // Mute or unmute the microphone
            ElevatedButton(
              onPressed: () async {
                // Mute or unmute the microphone
                await ZegoUIKitPrebuiltVideoConference.muteMicrophone(
                  mute: true,
                );
              },
              child: Text('Mute/Unmute Microphone'),
            ),
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. Initialize the package by calling `ZegoUIKitPrebuiltVideoConference.init()`.
// 2. Join a room by calling `ZegoUIKitPrebuiltVideoConference.joinRoom()` with the room ID and user ID.
// 3. Leave a room by calling `ZegoUIKitPrebuiltVideoConference.leaveRoom()`.
// 4. Switch the camera by calling `ZegoUIKitPrebuiltVideoConference.switchCamera()`.
// 5. Mute or unmute the microphone by calling `ZegoUIKitPrebuiltVideoConference.muteMicrophone()`.

// Summary:
// This code demonstrates how to use the `zego_uikit_prebuilt_video_conference` package to join a room, leave a room, switch the camera, and mute or unmute the microphone.
```
