# Audience Network Flutter Package
=====================================

The Audience Network Flutter package is a powerful tool for integrating Facebook's Audience Network into your Flutter applications. This package allows you to display ads from Facebook's Audience Network, providing a seamless and engaging user experience.

## When to Use This Package
---------------------------

You should use this package when you want to monetize your Flutter application with ads from Facebook's Audience Network. This package is ideal for applications with a large user base, as it provides a reliable and efficient way to display ads.

## Features
------------

*   Display ads from Facebook's Audience Network
*   Support for banner, interstitial, and rewarded video ads
*   Customizable ad formats and targeting options
*   Integration with Facebook's SDK for accurate ad tracking and reporting

# Setting Up the Audience Network Flutter Package
=====================================================

## Step 1: Add the Package to Your Project
-----------------------------------------

To add the Audience Network Flutter package to your project, simply add the following line to your `pubspec.yaml` file:

```yml
dependencies:
  flutter:
    sdk: flutter
  audience_network: ^0.1.0
```

Then, run `flutter pub get` to install the package.

## Step 2: Import the Package
---------------------------

To import the package, add the following line to the top of your Dart file:

```dart
import 'package:audience_network/audience_network.dart';
```

## Step 3: Initialize the Audience Network SDK
---------------------------------------------

To initialize the Audience Network SDK, call the `AudienceNetwork.init` method in your `main` function:

```dart
void main() {
  AudienceNetwork.init(
    appId: 'YOUR_APP_ID',
    adFormat: AdFormat.banner,
  );
  runApp(MyApp());
}
```

Replace `YOUR_APP_ID` with your actual Facebook app ID.

## Step 4: Display Ads
-------------------

To display ads, use the `AudienceNetworkAd` widget:

```dart
AudienceNetworkAd(
  adFormat: AdFormat.banner,
  adSize: AdSize.banner,
  targeting: Targeting(
    age: 25,
    interests: ['sports', 'music'],
  ),
)
```

This will display a banner ad with the specified targeting options.

## Platform-Specific Details
---------------------------

### Android

To use the Audience Network Flutter package on Android, you need to add the following lines to your `AndroidManifest.xml` file:

```xml
<application>
  ...
  <meta-data
    android:name="com.facebook.sdk.ApplicationId"
    android:value="YOUR_APP_ID" />
  <activity
    android:name="com.facebook.ads.AudienceNetworkActivity"
    android:configChanges="keyboardHidden|orientation|screenSize" />
</application>
```

Replace `YOUR_APP_ID` with your actual Facebook app ID.

### iOS

To use the Audience Network Flutter package on iOS, you need to add the following lines to your `Info.plist` file:

```xml
<key>FacebookAppID</key>
<string>YOUR_APP_ID</string>
<key>FacebookAppSecret</key>
<string>YOUR_APP_SECRET</string>
```

Replace `YOUR_APP_ID` and `YOUR_APP_SECRET` with your actual Facebook app ID and secret.

```dart
import 'package:flutter/material.dart';
import 'package:audience_network/audience_network.dart';

void main() {
  AudienceNetwork.init(
    appId: 'YOUR_APP_ID',
    adFormat: AdFormat.banner,
  );
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Audience Network Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Audience Network Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            AudienceNetworkAd(
              adFormat: AdFormat.banner,
              adSize: AdSize.banner,
              targeting: Targeting(
                age: 25,
                interests: ['sports', 'music'],
              ),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                Navigator.push(
                  context,
                  MaterialPageRoute(builder: (context) => InterstitialAdPage()),
                );
              },
              child: Text('Show Interstitial Ad'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                Navigator.push(
                  context,
                  MaterialPageRoute(builder: (context) => RewardedVideoAdPage()),
                );
              },
              child: Text('Show Rewarded Video Ad'),
            ),
          ],
        ),
      ),
    );
  }
}

class InterstitialAdPage extends StatefulWidget {
  @override
  _InterstitialAdPageState createState() => _InterstitialAdPageState();
}

class _InterstitialAdPageState extends State<InterstitialAdPage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Interstitial Ad'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () {
            AudienceNetwork.showInterstitialAd();
          },
          child: Text('Show Interstitial Ad'),
        ),
      ),
    );
  }
}

class RewardedVideoAdPage extends StatefulWidget {
  @override
  _RewardedVideoAdPageState createState() => _RewardedVideoAdPageState();
}

class _RewardedVideoAdPageState extends State<RewardedVideoAdPage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Rewarded Video Ad'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () {
            AudienceNetwork.showRewardedVideoAd();
          },
          child: Text('Show Rewarded Video Ad'),
        ),
      ),
    );
  }
}
```

// This is the main entry point of the app.
// The AudienceNetwork.init method is called to initialize the Audience Network SDK.
// The appId and adFormat are passed as parameters to the init method.
// The runApp method is called to start the app.

// The MyApp widget is the root widget of the app.
// It returns a MaterialApp widget with a title and a home page.

// The MyHomePage widget is the home page of the app.
// It returns a Scaffold widget with an AppBar and a body.
// The body contains a Center widget with a Column widget.
// The Column widget contains an AudienceNetworkAd widget, a SizedBox widget, and two ElevatedButton widgets.
// The AudienceNetworkAd widget displays a banner ad.
// The SizedBox widget adds a gap between the ad and the buttons.
// The ElevatedButton widgets navigate to the InterstitialAdPage and RewardedVideoAdPage when pressed.

// The InterstitialAdPage widget displays an interstitial ad when the button is pressed.
// The RewardedVideoAdPage widget displays a rewarded video ad when the button is pressed.

// The AudienceNetwork.showInterstitialAd and AudienceNetwork.showRewardedVideoAd methods are called to display the interstitial and rewarded video ads.
