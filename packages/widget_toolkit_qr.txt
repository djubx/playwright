In summary, the code flows as follows:

1. The user opens the app and sees a QR code generated with the specified data.
2. The user clicks the "Scan QR Code" button to open the device's camera.
3. The user scans a QR code using the device's camera.
4. The scanned data is displayed on the screen.
5. The user can repeat the process by clicking the "Scan QR Code" button again.
