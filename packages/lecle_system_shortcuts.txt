# lecle_system_shortcuts Flutter Package
The `lecle_system_shortcuts` package is a Flutter plugin that allows developers to create custom system shortcuts for their apps. With this package, you can create shortcuts that perform specific actions within your app, making it easier for users to access frequently used features.

## When to Use
This package is useful when you want to provide users with quick access to specific features or actions within your app. For example, you can create a shortcut to open a specific screen, perform a search, or trigger a specific action.

## Features
The `lecle_system_shortcuts` package provides the following features:

* Create custom system shortcuts for your app
* Define custom icons and labels for your shortcuts
* Handle shortcut clicks and perform specific actions within your app
* Support for both Android and iOS platforms

# Setup and Usage
To use the `lecle_system_shortcuts` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File
Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  lecle_system_shortcuts: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package
Import the package in your Dart file:
```dart
import 'package:lecle_system_shortcuts/lecle_system_shortcuts.dart';
```
### Step 3: Create a Shortcut
Create a shortcut using the `LecleSystemShortcuts` class:
```dart
LecleSystemShortcuts _shortcuts = LecleSystemShortcuts();

_shortcuts.createShortcut(
  id: 'my_shortcut',
  label: 'My Shortcut',
  icon: Icons.search,
  onTap: () {
    // Handle shortcut click
  },
);
```
### Platform-Specific Details

#### Android
On Android, you need to add the following code to your `AndroidManifest.xml` file:
```xml
<application>
  ...
  <activity
    android:name=".MainActivity"
    android:exported="true">
    <intent-filter>
      <action android:name="android.intent.action.MAIN" />
      <category android:name="android.intent.category.LAUNCHER" />
    </intent-filter>
  </activity>
  ...
</application>
```
#### iOS
On iOS, you need to add the following code to your `Info.plist` file:
```xml
<key>UIApplicationShortcutItems</key>
<array>
  <dict>
    <key>UIApplicationShortcutItemType</key>
    <string>com.example.myapp.myshortcut</string>
    <key>UIApplicationShortcutItemTitle</key>
    <string>My Shortcut</string>
    <key>UIApplicationShortcutItemIconFile</key>
    <string>my_shortcut_icon</string>
  </dict>
</array>
```
### Step 4: Handle Shortcut Clicks
Handle shortcut clicks using the `onTap` callback:
```dart
_shortcuts.createShortcut(
  id: 'my_shortcut',
  label: 'My Shortcut',
  icon: Icons.search,
  onTap: () {
    // Handle shortcut click
    Navigator.push(
      context,
      MaterialPageRoute(builder: (context) => MyScreen()),
    );
  },
);
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:lecle_system_shortcuts/lecle_system_shortcuts.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Lecle System Shortcuts Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  LecleSystemShortcuts _shortcuts = LecleSystemShortcuts();

  @override
  void initState() {
    super.initState();
    _shortcuts.createShortcut(
      id: 'my_shortcut',
      label: 'My Shortcut',
      icon: Icons.search,
      onTap: () {
        // Handle shortcut click
        Navigator.push(
          context,
          MaterialPageRoute(builder: (context) => MyScreen()),
        );
      },
    );
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Lecle System Shortcuts Demo'),
      ),
      body: Center(
        child: Text('Create a shortcut to open MyScreen'),
      ),
    );
  }
}

class MyScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('MyScreen'),
      ),
      body: Center(
        child: Text('This screen was opened from a shortcut'),
      ),
    );
  }
}
```

// The app starts by creating a material app with a home page.
// The home page creates a LecleSystemShortcuts instance and defines a shortcut with an id, label, and icon.
// When the shortcut is clicked, it navigates to a new screen called MyScreen.
// The MyScreen widget is a simple scaffold with a title and a center text.
// The app uses the LecleSystemShortcuts package to create and handle shortcuts.
