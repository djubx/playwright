# languagetool_textfield Flutter Package
=====================================

The `languagetool_textfield` package is a powerful tool for Flutter developers to integrate language grammar and spell checking into their applications. It provides a simple and efficient way to check for grammar and spelling errors in text fields, making it an essential package for any app that involves text input.

### When to Use This Package

This package is ideal for apps that require users to input text, such as:

* Text editors
* Chat applications
* Email clients
* Blogging platforms

### Features

* Grammar and spell checking
* Auto-correction suggestions
* Customizable dictionary
* Support for multiple languages

# Setting Up the Package
-------------------------

To use the `languagetool_textfield` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  languagetool_textfield: ^1.0.0
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:languagetool_textfield/languagetool_textfield.dart';
```
### Step 3: Create a Text Field

Create a text field and wrap it with the `LanguageToolTextField` widget:
```dart
LanguageToolTextField(
  decoration: InputDecoration(
    labelText: 'Text Field',
  ),
  onChanged: (text) {
    // Handle text changes
  },
)
```
### Platform-Specific Details

* Android: No additional setup required.
* iOS: Add the following line to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```
### Configurations and Optimizations

* You can customize the dictionary by providing a list of words to ignore:
```dart
LanguageToolTextField(
  ignoreWords: ['ignore', 'this', 'word'],
)
```
* You can also set the language code to support multiple languages:
```dart
LanguageToolTextField(
  languageCode: 'fr',
)
```

```dart
import 'package:flutter/material.dart';
import 'package:languagetool_textfield/languagetool_textfield.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Language Tool Text Field Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _textController = TextEditingController();

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Language Tool Text Field Demo'),
      ),
      body: Padding(
        padding: const EdgeInsets.all(16.0),
        child: Column(
          children: [
            LanguageToolTextField(
              decoration: InputDecoration(
                labelText: 'Text Field',
              ),
              onChanged: (text) {
                // Handle text changes
              },
              controller: _textController,
            ),
            SizedBox(height: 16),
            ElevatedButton(
              onPressed: () {
                // Get the text from the text field
                final text = _textController.text;
                // Check for grammar and spelling errors
                final errors = LanguageTool.checkText(text);
                // Display the errors
                if (errors.isNotEmpty) {
                  showDialog(
                    context: context,
                    builder: (context) {
                      return AlertDialog(
                        title: Text('Errors'),
                        content: Text(errors.join('\n')),
                      );
                    },
                  );
                } else {
                  // No errors found
                  ScaffoldMessenger.of(context).showSnackBar(
                    SnackBar(content: Text('No errors found')),
                  );
                }
              },
              child: Text('Check Text'),
            ),
          ],
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user types some text into the text field.
// 2. The text field is wrapped with the LanguageToolTextField widget, which checks for grammar and spelling errors.
// 3. When the user clicks the "Check Text" button, the text from the text field is retrieved and checked for errors.
// 4. If errors are found, they are displayed in a dialog box.
// 5. If no errors are found, a snackbar is displayed indicating that no errors were found.

// This code demonstrates the key features of the languagetool_textfield package, including grammar and spell checking, auto-correction suggestions, and customizable dictionary.

// The code flow is straightforward, with the LanguageToolTextField widget handling the grammar and spell checking, and the ElevatedButton handling the text checking and error display.

// Overall, this code provides a simple and effective way to integrate language grammar and spell checking into a Flutter application.
```
