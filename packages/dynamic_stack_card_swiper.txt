# Dynamic Stack Card Swiper Package
=====================================

The `dynamic_stack_card_swiper` package is a Flutter library that allows you to create a dynamic stack of cards that can be swiped left or right. This package is perfect for creating apps that require a tinder-like swipe feature.

### Features

*   Dynamic stack of cards
*   Swipe left or right to dismiss cards
*   Customizable card layout and design
*   Supports multiple card types

### When to Use

This package is ideal for creating apps that require a swipe feature, such as:

*   Dating apps
*   Social media apps
*   Product showcase apps
*   Quiz or game apps

# Setting Up the Package
-------------------------

To use the `dynamic_stack_card_swiper` package, follow these steps:

### Step 1: Add the Package to Your Pubspec.yaml File

```yml
dependencies:
  flutter:
    sdk: flutter
  dynamic_stack_card_swiper: ^1.0.0
```

### Step 2: Import the Package in Your Dart File

```dart
import 'package:dynamic_stack_card_swiper/dynamic_stack_card_swiper.dart';
```

### Step 3: Create a Card Layout

Create a custom card layout using the `Card` widget:

```dart
Card(
  child: Container(
    height: 200,
    width: 200,
    child: Center(
      child: Text('Card 1'),
    ),
  ),
)
```

### Step 4: Create a Dynamic Stack Card Swiper

Create a `DynamicStackCardSwiper` widget and pass in your card layout:

```dart
DynamicStackCardSwiper(
  cards: [
    Card(
      child: Container(
        height: 200,
        width: 200,
        child: Center(
          child: Text('Card 1'),
        ),
      ),
    ),
    Card(
      child: Container(
        height: 200,
        width: 200,
        child: Center(
          child: Text('Card 2'),
        ),
      ),
    ),
  ],
)
```

### Platform-Specific Details

This package works on both Android and iOS platforms. However, you may need to adjust the card layout and design to fit your specific use case.

### Required Configurations or Optimizations

You can customize the card layout and design by using the `Card` widget properties. You can also adjust the swipe animation by using the `DynamicStackCardSwiper` widget properties.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:dynamic_stack_card_swiper/dynamic_stack_card_swiper.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Dynamic Stack Card Swiper Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Dynamic Stack Card Swiper Demo'),
      ),
      body: Center(
        child: DynamicStackCardSwiper(
          cards: [
            // Card 1
            Card(
              child: Container(
                height: 200,
                width: 200,
                child: Center(
                  child: Text('Card 1'),
                ),
              ),
            ),
            // Card 2
            Card(
              child: Container(
                height: 200,
                width: 200,
                child: Center(
                  child: Text('Card 2'),
                ),
              ),
            ),
            // Card 3
            Card(
              child: Container(
                height: 200,
                width: 200,
                child: Center(
                  child: Text('Card 3'),
                ),
              ),
            ),
          ],
        ),
      ),
    );
  }
}

// The app starts by running the main function, which calls the runApp function with the MyApp widget.
// The MyApp widget is a stateless widget that returns a MaterialApp widget with a title and a theme.
// The home property of the MaterialApp widget is set to the MyHomePage widget.
// The MyHomePage widget is a stateful widget that returns a Scaffold widget with an AppBar and a body.
// The body of the Scaffold widget is a Center widget that contains a DynamicStackCardSwiper widget.
// The DynamicStackCardSwiper widget is passed a list of Card widgets, each with a different text label.
// When the app is run, the DynamicStackCardSwiper widget displays the first card in the list, and the user can swipe left or right to display the next card.
```
