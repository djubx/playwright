# Neubrutalism UI Flutter Package
=====================================

The Neubrutalism UI Flutter package is a design system that provides a set of pre-built UI components and a style guide to help developers create visually appealing and consistent user interfaces. Neubrutalism is a design movement that emerged in the 1950s and is characterized by the use of raw concrete, exposed ductwork, and a focus on functionality over aesthetics.

This package is ideal for developers who want to create apps with a unique and distinctive design that stands out from the crowd. It's particularly well-suited for apps that require a bold and futuristic look, such as gaming, technology, and innovation-focused apps.

Some of the key features of the Neubrutalism UI Flutter package include:

* A set of pre-built UI components, including buttons, text fields, and navigation bars
* A style guide that provides guidance on typography, color, and layout
* Support for both light and dark themes
* Customizable components to fit your app's specific needs

# Setting Up the Neubrutalism UI Flutter Package
---------------------------------------------

To use the Neubrutalism UI Flutter package, follow these steps:

### Step 1: Add the Package to Your Pubspec.yaml File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  neubrutalism_ui: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:neubrutalism_ui/neubrutalism_ui.dart';
```
### Step 3: Use the Components

Use the pre-built UI components, such as the `NeubrutalismButton`:
```dart
NeubrutalismButton(
  onPressed: () {
    // Handle button press
  },
  child: Text('Click me'),
)
```
### Platform-Specific Details

* For Android, make sure to add the following line to your `AndroidManifest.xml` file:
```xml
<application
  ...
  android:theme="@style/NeubrutalismTheme"
  ...
>
```
* For iOS, add the following line to your `Info.plist` file:
```xml
<key>UIAppFonts</key>
<array>
  <string>Neubrutalism-Font</string>
</array>
```
### Required Configurations and Optimizations

* Make sure to use the `NeubrutalismTheme` as the app's theme:
```dart
MaterialApp(
  title: 'Neubrutalism UI Demo',
  theme: NeubrutalismTheme(),
  home: MyHomePage(),
)
```
* Use the `NeubrutalismTextStyle` for text:
```dart
Text(
  'Hello World',
  style: NeubrutalismTextStyle(),
)
```

```dart
import 'package:flutter/material.dart';
import 'package:neubrutalism_ui/neubrutalism_ui.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Neubrutalism UI Demo',
      theme: NeubrutalismTheme(),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: NeubrutalismAppBar(
        title: Text('Neubrutalism UI Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            NeubrutalismButton(
              onPressed: () {
                // Handle button press
              },
              child: Text('Click me'),
            ),
            SizedBox(height: 20),
            Text(
              'Hello World',
              style: NeubrutalismTextStyle(),
            ),
          ],
        ),
      ),
    );
  }
}
```
