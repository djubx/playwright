# Water Bottle Flutter Package
The water_bottle Flutter package is a customizable water intake tracking library for Flutter applications. It provides a simple and easy-to-use API for tracking water intake, setting reminders, and viewing progress.

## Features
- Customizable water bottle sizes
- Water intake tracking
- Reminders for drinking water
- Progress tracking

## When to Use
This package is ideal for health and fitness applications that want to encourage users to stay hydrated.

<-- END_DESCRIPTION -->

<-- START_TUTORIAL -->
# Setting Up the Water Bottle Package
To use the water_bottle package, add the following dependency to your pubspec.yaml file:

```yml
dependencies:
  water_bottle: ^1.0.0
```

Then, run `flutter pub get` to install the package.

## Platform-Specific Configurations
### Android
No additional configurations are required for Android.

### iOS
Add the following lines to your Info.plist file to enable notifications:

```xml
<key>UIBackgroundModes</key>
<array>
  <string>processing</string>
</array>
```

## Using the Package
Import the package in your Dart file:

```dart
import 'package:water_bottle/water_bottle.dart';
```

Create a new instance of the WaterBottle class:

```dart
WaterBottle waterBottle = WaterBottle(
  size: 1000, // in milliliters
  reminderInterval: Duration(hours: 2),
);
```

Track water intake:

```dart
waterBottle.drink(500); // in milliliters
```

<-- START_MAIN -->
```dart
import 'package:flutter/material.dart';
import 'package:water_bottle/water_bottle.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  WaterBottle waterBottle;

  @override
  void initState() {
    super.initState();
    // Create a new instance of the WaterBottle class
    waterBottle = WaterBottle(
      size: 1000, // in milliliters
      reminderInterval: Duration(hours: 2),
    );
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Water Bottle Demo'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              // Display the current water level
              Text('Water Level: ${waterBottle.currentLevel}ml'),
              SizedBox(height: 20),
              // Button to drink water
              ElevatedButton(
                onPressed: () {
                  // Track water intake
                  waterBottle.drink(500); // in milliliters
                  setState(() {}); // Update the UI
                },
                child: Text('Drink Water'),
              ),
              SizedBox(height: 20),
              // Button to view progress
              ElevatedButton(
                onPressed: () {
                  // View progress
                  Navigator.push(
                    context,
                    MaterialPageRoute(builder: (context) => ProgressPage(waterBottle)),
                  );
                },
                child: Text('View Progress'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// Page to view progress
class ProgressPage extends StatelessWidget {
  final WaterBottle waterBottle;

  ProgressPage(this.waterBottle);

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Progress'),
      ),
      body: Center(
        child: Text('You have drunk ${waterBottle.currentLevel}ml of water today!'),
      ),
    );
  }
}
```

// The application starts by creating a new instance of the WaterBottle class.
// The user can then track water intake by pressing the "Drink Water" button.
// The current water level is displayed on the screen.
// The user can view their progress by pressing the "View Progress" button.
// The progress page displays the amount of water the user has drunk.

// The code flows by first initializing the WaterBottle instance in the initState method.
// The build method then creates the UI, including the buttons and text displays.
// When the user presses a button, the corresponding method is called (e.g. drink or view progress).
// The setState method is used to update the UI after the user drinks water.
