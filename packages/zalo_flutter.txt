# zalo_flutter Package Overview
=====================================

The `zalo_flutter` package is a Flutter plugin that allows developers to integrate Zalo, a popular Vietnamese social media platform, into their Flutter applications. With this package, you can easily authenticate users with Zalo, share content, and more.

### When to Use This Package

You should use this package when you want to:

* Authenticate users with Zalo in your Flutter app
* Share content from your app to Zalo
* Use Zalo's social features in your app

### Features

* Authenticate users with Zalo
* Share content (text, images, videos) to Zalo
* Get user profile information
* Handle Zalo login and logout

# Setting Up and Using the zalo_flutter Package
=============================================

### Step 1: Add the Package to Your pubspec.yaml File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  zalo_flutter: ^1.0.0
```
Then, run `flutter pub get` to get the package.

### Step 2: Configure the Package

Create a new file called `zalo_config.dart` with the following content:
```dart
import 'package:zalo_flutter/zalo_flutter.dart';

class ZaloConfig {
  static const String appId = 'YOUR_APP_ID';
  static const String appSecret = 'YOUR_APP_SECRET';
}
```
Replace `YOUR_APP_ID` and `YOUR_APP_SECRET` with your actual Zalo app ID and secret.

### Step 3: Initialize the Package

In your `main.dart` file, add the following code:
```dart
import 'package:flutter/material.dart';
import 'package:zalo_flutter/zalo_flutter.dart';
import 'zalo_config.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Zalo Flutter Demo',
      home: ZaloLoginScreen(),
    );
  }
}

class ZaloLoginScreen extends StatefulWidget {
  @override
  _ZaloLoginScreenState createState() => _ZaloLoginScreenState();
}

class _ZaloLoginScreenState extends State<ZaloLoginScreen> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Zalo Login'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () async {
            await ZaloFlutter.init(ZaloConfig.appId, ZaloConfig.appSecret);
            await ZaloFlutter.login();
          },
          child: Text('Login with Zalo'),
        ),
      ),
    );
  }
}
```
This code initializes the package and logs in the user with Zalo.

### Platform-Specific Details

* For Android, add the following code to your `AndroidManifest.xml` file:
```xml
<activity
    android:name="com.zalo.sdk.ZaloLoginActivity"
    android:exported="true" />
```
* For iOS, add the following code to your `Info.plist` file:
```xml
<key>CFBundleURLTypes</key>
<array>
    <dict>
        <key>CFBundleURLName</key>
        <string>zalo</string>
        <key>CFBundleURLSchemes</key>
        <array>
            <string>zalo</string>
        </array>
    </dict>
</array>
```

```dart
import 'package:flutter/material.dart';
import 'package:zalo_flutter/zalo_flutter.dart';

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  // Initialize the package
  Future<void> _initPackage() async {
    await ZaloFlutter.init('YOUR_APP_ID', 'YOUR_APP_SECRET');
  }

  // Login with Zalo
  Future<void> _loginWithZalo() async {
    await ZaloFlutter.login();
  }

  // Share content to Zalo
  Future<void> _shareToZalo() async {
    await ZaloFlutter.share('Hello, world!');
  }

  // Get user profile information
  Future<void> _getUserProfile() async {
    final userProfile = await ZaloFlutter.getUserProfile();
    print(userProfile);
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Zalo Flutter Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            ElevatedButton(
              onPressed: _initPackage,
              child: Text('Initialize Package'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _loginWithZalo,
              child: Text('Login with Zalo'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _shareToZalo,
              child: Text('Share to Zalo'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _getUserProfile,
              child: Text('Get User Profile'),
            ),
          ],
        ),
      ),
    );
  }
}

// 4. The user gets their profile information by clicking the "Get User Profile" button.

// In summary, this code demonstrates how to use the zalo_flutter package to initialize the package, login with Zalo, share content to Zalo, and get user profile information.
```
