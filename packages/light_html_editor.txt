# Light HTML Editor Flutter Package
=====================================

The `light_html_editor` Flutter package provides a lightweight HTML editor for mobile applications. It allows users to create and edit HTML content with a simple and intuitive interface.

## When to Use
--------------

This package is ideal for applications that require users to create or edit HTML content, such as:

* Blogging apps
* Email clients
* Content management systems
* Text editors

## Features
------------

* Simple and intuitive interface
* Supports basic HTML tags (e.g., bold, italic, underline, links, images)
* Allows users to create and edit HTML content
* Customizable toolbar with various options (e.g., font size, color, alignment)

# Setup and Usage
------------------

## Step 1: Add the Package to Your Project
-----------------------------------------

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  light_html_editor: ^1.0.0
```
Then, run `flutter pub get` to install the package.

## Step 2: Import the Package
-----------------------------

Import the package in your Dart file:
```dart
import 'package:light_html_editor/light_html_editor.dart';
```
## Step 3: Create an Instance of the Editor
-----------------------------------------

Create an instance of the `LightHtmlEditor` widget:
```dart
LightHtmlEditor(
  controller: _controller,
  toolbarOptions: [
    ToolbarOption.bold,
    ToolbarOption.italic,
    ToolbarOption.underline,
  ],
)
```
## Platform-Specific Details
---------------------------

### Android

No additional configuration is required for Android.

### iOS

Add the following line to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```
This allows the editor to load images from external sources.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:light_html_editor/light_html_editor.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Light HTML Editor Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _controller = TextEditingController();

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Light HTML Editor Demo'),
      ),
      body: Padding(
        padding: const EdgeInsets.all(16.0),
        child: LightHtmlEditor(
          controller: _controller,
          toolbarOptions: [
            ToolbarOption.bold,
            ToolbarOption.italic,
            ToolbarOption.underline,
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user opens the app and is presented with a simple HTML editor.
// 2. The user can type in the editor and use the toolbar options to format the text.
// 3. The user can create and edit HTML content using the editor.
// 4. The app displays the formatted HTML content in real-time.
// Summary: The app provides a simple and intuitive HTML editor for users to create and edit HTML content.
```
