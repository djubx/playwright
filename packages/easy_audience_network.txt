# Easy Audience Network Flutter Package

The **easy_audience_network** Flutter package is a powerful tool designed to simplify the integration of Facebook Audience Network ads into Flutter applications. This package allows developers to monetize their apps effectively by displaying ads from Facebook's extensive ad network, which includes banner ads, interstitial ads, and rewarded video ads.

## When to Use This Package

You should consider using the **easy_audience_network** package when:
- You want to monetize your Flutter application through ads.
- You are looking for a straightforward way to integrate Facebook Audience Network ads without dealing with complex native code.
- You need support for multiple ad formats, including banners, interstitials, and rewarded videos.

## Features

- **Easy Integration**: Simplifies the process of integrating Facebook Audience Network ads into Flutter apps.
- **Multiple Ad Formats**: Supports various ad formats, including banner ads, interstitial ads, and rewarded video ads.
- **Cross-Platform Support**: Works seamlessly on both Android and iOS platforms.
- **Event Handling**: Provides callbacks for ad events, allowing developers to respond to ad loading, display, and errors.

# Tutorial: Setting Up Easy Audience Network

In this tutorial, we will walk through the setup process for the **easy_audience_network** package and demonstrate how to use it in your Flutter application.

## Step 1: Add Dependency

To get started, add the **easy_audience_network** package to your `pubspec.yaml` file:

```yaml
dependencies:
  flutter:
    sdk: flutter
  easy_audience_network: ^latest_version
```

Make sure to replace `^latest_version` with the latest version of the package available on [pub.dev](https://pub.dev/packages/easy_audience_network).

## Step 2: Platform-Specific Configuration

### Android Configuration

1. **Update AndroidManifest.xml**: Open your `AndroidManifest.xml` file located in `android/app/src/main/AndroidManifest.xml` and add the following permissions and metadata:

```xml
<manifest>
    <uses-permission android:name="android.permission.INTERNET"/>
    <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE"/>

    <application>
        <meta-data
            android:name="com.facebook.sdk.ApplicationId"
            android:value="@string/facebook_app_id"/>
        <meta-data
            android:name="com.facebook.adnetwork.sdk.AdNetworkId"
            android:value="@string/facebook_ad_network_id"/>
    </application>
</manifest>
```

2. **Add Facebook App ID**: In your `res/values/strings.xml`, add your Facebook App ID and Ad Network ID:

```xml
<resources>
    <string name="facebook_app_id">YOUR_FACEBOOK_APP_ID</string>
    <string name="facebook_ad_network_id">YOUR_AD_NETWORK_ID</string>
</resources>
```

### iOS Configuration

1. **Update Info.plist**: Open your `Info.plist` file located in `ios/Runner/Info.plist` and add the following entries:

```xml
<key>FacebookAppID</key>
<string>YOUR_FACEBOOK_APP_ID</string>
<key>FacebookDisplayName</key>
<string>YOUR_APP_NAME</string>
<key>NSAppTransportSecurity</key>
<dict>
    <key>NSAllowsArbitraryLoads</key>
    <true/>
</dict>
```

2. **Add Required Frameworks**: Ensure you have the necessary frameworks in your `ios/Podfile`:

```ruby
pod 'FBSDKCoreKit'
pod 'FBAudienceNetwork'
```

## Step 3: Initialize the Package

In your main Dart file, initialize the package before running the app:

```dart
import 'package:flutter/material.dart';
import 'package:easy_audience_network/easy_audience_network.dart';

void main() {
  WidgetsFlutterBinding.ensureInitialized();
  EasyAudienceNetwork.init(); // Initialize the package
  runApp(MyApp());
}
```

# Complete Example: Flutter Main File

```dart
import 'package:flutter/material.dart';
import 'package:easy_audience_network/easy_audience_network.dart';

void main() {
  WidgetsFlutterBinding.ensureInitialized();
  EasyAudienceNetwork.init(); // Initialize the Easy Audience Network package
  runApp(RealFlutter()); // Run the main application
}

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Easy Audience Network Example')),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Text('Welcome to Easy Audience Network!'),
              SizedBox(height: 20),
              // Display a banner ad
              EasyAudienceNetworkBanner(
                adUnitId: 'YOUR_BANNER_AD_UNIT_ID', // Replace with your ad unit ID
                listener: (event) {
                  // Handle ad events
                  print('Banner Ad Event: $event'); // Log ad events to console
                },
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  // Load and show an interstitial ad
                  EasyAudienceNetworkInterstitial.load(
                    adUnitId: 'YOUR_INTERSTITIAL_AD_UNIT_ID', // Replace with your ad unit ID
                    listener: (event) {
                      // Handle interstitial ad events
                      print('Interstitial Ad Event: $event');
                    },
                  );
                },
                child: Text('Show Interstitial Ad'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// Application Flow Explanation:
// 1. The app starts by initializing the Easy Audience Network package.
// 2. The main widget, RealFlutter, is built, which contains a simple UI with a welcome message.
// 3. A banner ad is displayed using EasyAudienceNetworkBanner, and its events are logged to the console.
// 4. An ElevatedButton is provided to load and show an interstitial ad when pressed.
// 5. The interstitial ad's events are also logged to the console for debugging purposes.
```
