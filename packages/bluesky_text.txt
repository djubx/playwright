# Bluesky Text Flutter Package
================================

The bluesky_text Flutter package is a powerful library for creating and editing rich text documents in Flutter applications. It provides a wide range of features, including text formatting, image insertion, and document management.

## When to Use
--------------

The bluesky_text package is ideal for applications that require rich text editing capabilities, such as:

* Text editors
* Document management systems
* Note-taking apps
* Blogging platforms

## Features
------------

* Rich text editing capabilities
* Text formatting (bold, italic, underline, etc.)
* Image insertion
* Document management (create, edit, delete)
* Platform-specific optimizations for Android and iOS

# Setting Up the Bluesky Text Package
--------------------------------------

To use the bluesky_text package in your Flutter application, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  bluesky_text: ^1.0.0
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:bluesky_text/bluesky_text.dart';
```
### Step 3: Create a Bluesky Text Editor

Create a `BlueskyTextEditor` widget and pass in a `TextEditingController`:
```dart
BlueskyTextEditor(
  controller: _textController,
)
```
### Platform-Specific Configurations

#### Android

Add the following line to your `AndroidManifest.xml` file:
```xml
<application
  ...
  android:hardwareAccelerated="true"
  ...
>
```
#### iOS

Add the following line to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:bluesky_text/bluesky_text.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  final _textController = TextEditingController();

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Bluesky Text Example'),
        ),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: Column(
            children: [
              // Create a Bluesky Text Editor
              BlueskyTextEditor(
                controller: _textController,
              ),
              SizedBox(height: 16),
              // Create a button to insert an image
              ElevatedButton(
                onPressed: () {
                  // Insert an image into the text editor
                  _textController.insertImage(
                    'https://example.com/image.jpg',
                  );
                },
                child: Text('Insert Image'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user opens the app and is presented with a text editor.
// 2. The user types some text into the editor.
// 3. The user clicks the "Insert Image" button.
// 4. An image is inserted into the text editor at the current cursor position.
// 5. The user can continue typing and editing the text.
// 6. The user can also delete the image by selecting it and pressing the delete key.

// Summary:
// This code creates a simple text editor using the Bluesky Text package.
// It allows the user to type and edit text, as well as insert images into the text.
// The application flow is straightforward, with the user interacting with the text editor and image insertion button.
```
