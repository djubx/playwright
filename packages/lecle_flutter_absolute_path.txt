# lecle_flutter_absolute_path Package Overview
=====================================================

The `lecle_flutter_absolute_path` package is a Flutter plugin that provides a simple way to get the absolute path of a file or directory on both Android and iOS platforms. This package is useful when you need to access files or directories outside of the app's sandboxed environment.

### When to Use This Package

* When you need to access files or directories on the device's external storage.
* When you need to share files or directories between apps.
* When you need to access files or directories on the device's internal storage.

### Features

* Get the absolute path of a file or directory on both Android and iOS platforms.
* Supports accessing files and directories on the device's external storage.
* Supports accessing files and directories on the device's internal storage.

# Setup and Usage
=====================

### Add the Package to Your Flutter Project

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  lecle_flutter_absolute_path: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Import the Package

Import the package in your Dart file:
```dart
import 'package:lecle_flutter_absolute_path/lecle_flutter_absolute_path.dart';
```
### Get the Absolute Path

Use the `getAbsolutePath` method to get the absolute path of a file or directory:
```dart
String absolutePath = await LecleFlutterAbsolutePath.getAbsolutePath('/path/to/file');
```
### Platform-Specific Details

#### Android

On Android, the package uses the `Context.getExternalFilesDir()` method to get the absolute path of a file or directory on the device's external storage.

#### iOS

On iOS, the package uses the `NSSearchPathForDirectoriesInDomains()` function to get the absolute path of a file or directory on the device's internal storage.

### Required Configurations or Optimizations

No additional configurations or optimizations are required to use this package.

```dart
import 'package:flutter/material.dart';
import 'package:lecle_flutter_absolute_path/lecle_flutter_absolute_path.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Lecle Flutter Absolute Path Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  String _absolutePath = '';

  Future<void> _getAbsolutePath() async {
    // Get the absolute path of a file or directory
    String absolutePath = await LecleFlutterAbsolutePath.getAbsolutePath('/path/to/file');
    setState(() {
      _absolutePath = absolutePath;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Lecle Flutter Absolute Path Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text(
              'Absolute Path:',
              style: TextStyle(fontSize: 24),
            ),
            Text(
              _absolutePath,
              style: TextStyle(fontSize: 18),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _getAbsolutePath,
              child: Text('Get Absolute Path'),
            ),
          ],
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user clicks the "Get Absolute Path" button.
// 2. The `_getAbsolutePath` method is called, which uses the `LecleFlutterAbsolutePath.getAbsolutePath` method to get the absolute path of a file or directory.
// 3. The absolute path is stored in the `_absolutePath` variable and the UI is updated to display the absolute path.
// 4. The user can view the absolute path on the screen.

// Summary: This code demonstrates how to use the `lecle_flutter_absolute_path` package to get the absolute path of a file or directory in a Flutter app.
```
