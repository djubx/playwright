Phonecodes Flutter Package
==========================

The phonecodes Flutter package is a utility library that provides a simple way to work with phone numbers and country codes in Flutter applications. It allows you to easily validate, format, and extract information from phone numbers, making it a useful tool for any app that requires phone number input.

When to Use This Package
------------------------

You should use this package in the following scenarios:

* When you need to validate phone numbers in your app
* When you need to format phone numbers in a specific way (e.g., international format)
* When you need to extract information from phone numbers (e.g., country code, national number)

Features
--------

The phonecodes package provides the following features:

* Phone number validation
* Phone number formatting
* Country code extraction
* National number extraction

Setup and Usage
---------------

To use the phonecodes package, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  phonecodes: ^1.0.0
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:phonecodes/phonecodes.dart';
```
### Step 3: Use the Package

Use the package to validate, format, or extract information from phone numbers:
```dart
void main() {
  // Validate a phone number
  String phoneNumber = '+1234567890';
  bool isValid = Phonecodes.validate(phoneNumber);
  print(isValid); // Output: true

  // Format a phone number
  String formattedPhoneNumber = Phonecodes.format(phoneNumber);
  print(formattedPhoneNumber); // Output: +1 234-567-890

  // Extract country code
  String countryCode = Phonecodes.getCountryCode(phoneNumber);
  print(countryCode); // Output: +1

  // Extract national number
  String nationalNumber = Phonecodes.getNationalNumber(phoneNumber);
  print(nationalNumber); // Output: 234567890
}
```
Platform-Specific Details
-------------------------

The phonecodes package works on both Android and iOS platforms. However, there are some platform-specific details to note:

* On Android, the package uses the `libphonenumber` library to validate and format phone numbers.
* On iOS, the package uses the `NBPhoneNumberUtil` library to validate and format phone numbers.

```dart
import 'package:flutter/material.dart';
import 'package:phonecodes/phonecodes.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  final _phoneNumberController = TextEditingController();
  String _formattedPhoneNumber = '';
  String _countryCode = '';
  String _nationalNumber = '';

  void _validatePhoneNumber() {
    // Get the phone number from the text field
    String phoneNumber = _phoneNumberController.text;

    // Validate the phone number
    bool isValid = Phonecodes.validate(phoneNumber);

    // If the phone number is valid, format it and extract country code and national number
    if (isValid) {
      _formattedPhoneNumber = Phonecodes.format(phoneNumber);
      _countryCode = Phonecodes.getCountryCode(phoneNumber);
      _nationalNumber = Phonecodes.getNationalNumber(phoneNumber);
    } else {
      _formattedPhoneNumber = '';
      _countryCode = '';
      _nationalNumber = '';
    }

    // Update the UI
    setState(() {});
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Phonecodes Example'),
        ),
        body: Padding(
          padding: const EdgeInsets.all(20.0),
          child: Column(
            children: [
              TextField(
                controller: _phoneNumberController,
                decoration: InputDecoration(
                  labelText: 'Phone Number',
                  border: OutlineInputBorder(),
                ),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: _validatePhoneNumber,
                child: Text('Validate Phone Number'),
              ),
              SizedBox(height: 20),
              Text('Formatted Phone Number: $_formattedPhoneNumber'),
              Text('Country Code: $_countryCode'),
              Text('National Number: $_nationalNumber'),
            ],
          ),
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user enters a phone number in the text field.
// 2. The user clicks the "Validate Phone Number" button.
// 3. The _validatePhoneNumber function is called, which gets the phone number from the text field.
// 4. The phone number is validated using the Phonecodes.validate function.
// 5. If the phone number is valid, it is formatted using the Phonecodes.format function, and the country code and national number are extracted using the Phonecodes.getCountryCode and Phonecodes.getNationalNumber functions, respectively.
// 6. The UI is updated with the formatted phone number, country code, and national number.
// 7. If the phone number is not valid, the UI is updated with empty strings for the formatted phone number, country code, and national number.

// This example demonstrates how to use the phonecodes package to validate, format, and extract information from phone numbers in a Flutter application.

```
