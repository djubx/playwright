# iglu_color_picker_flutter Package Overview
The `iglu_color_picker_flutter` package is a Flutter library that provides a customizable color picker for mobile applications. It allows users to select colors from a variety of palettes, including material design colors, RGB colors, and hex code colors.

## When to Use This Package
This package is ideal for applications that require users to select colors, such as:

* Graphic design apps
* Painting apps
* Color palette generators
* Interior design apps

## Features
The `iglu_color_picker_flutter` package offers the following features:

* Customizable color palettes
* Material design color palette
* RGB color palette
* Hex code color palette
* Color wheel for selecting colors
* Opacity slider for adjusting color transparency

# Setting Up the Package
To use the `iglu_color_picker_flutter` package, follow these steps:

1. Add the package to your `pubspec.yaml` file:
```yml
dependencies:
  iglu_color_picker_flutter: ^1.0.0
```
2. Run `flutter pub get` to install the package.
3. Import the package in your Dart file:
```dart
import 'package:iglu_color_picker_flutter/iglu_color_picker_flutter.dart';
```

## Platform-Specific Details
The package works on both Android and iOS platforms. However, on Android, you need to add the following line to your `AndroidManifest.xml` file:
```xml
<application
    ...
    android:hardwareAccelerated="true"
    ...>
```
This enables hardware acceleration for the color picker.

## Using the Package
To use the package, create an instance of the `IgluColorPicker` widget:
```dart
IgluColorPicker(
  onColorChanged: (color) {
    // Handle color change
  },
)
```
You can customize the color palettes and other features using the various properties provided by the `IgluColorPicker` widget.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:iglu_color_picker_flutter/iglu_color_picker_flutter.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  Color _selectedColor = Colors.white;

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Iglu Color Picker Demo'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              Text('Select a color:'),
              IgluColorPicker(
                onColorChanged: (color) {
                  // Handle color change
                  setState(() {
                    _selectedColor = color;
                  });
                },
              ),
              Text('Selected color:'),
              Container(
                width: 50,
                height: 50,
                color: _selectedColor,
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user runs the app and is presented with a color picker.
// 2. The user selects a color from the color picker.
// 3. The `onColorChanged` callback is called with the selected color.
// 4. The app updates the selected color state.
// 5. The app displays the selected color in a container.
// 6. The user can repeat the process to select a different color.

// Summary:
// The app uses the `IgluColorPicker` widget to allow the user to select a color.
// The app updates the selected color state when the user selects a new color.
// The app displays the selected color in a container.
```
