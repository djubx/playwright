iban_utilities_lib Flutter Package
=====================================

The iban_utilities_lib package is a utility library for working with International Bank Account Numbers (IBANs) in Flutter applications. It provides a set of tools for validating, formatting, and manipulating IBANs.

When to Use This Package
------------------------

This package is useful when developing financial or banking applications that require IBAN validation and formatting. It can also be used in e-commerce applications that accept bank transfers as a payment method.

Features
--------

*   IBAN validation
*   IBAN formatting
*   Country code extraction
*   Bank code extraction
*   Account number extraction

Setup and Usage
----------------

### Step 1: Add the Package to Your Project

Add the following dependency to your `pubspec.yaml` file:

```yml
dependencies:
  iban_utilities_lib: ^1.0.0
```

Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:iban_utilities_lib/iban_utilities_lib.dart';
```

### Step 3: Validate an IBAN

Use the `IbanUtilities` class to validate an IBAN:

```dart
void main() {
  String iban = 'DE89370400404432013000';
  bool isValid = IbanUtilities.validateIban(iban);
  print('Is valid: $isValid');
}
```

### Step 4: Format an IBAN

Use the `IbanUtilities` class to format an IBAN:

```dart
void main() {
  String iban = 'DE89370400404432013000';
  String formattedIban = IbanUtilities.formatIban(iban);
  print('Formatted IBAN: $formattedIban');
}
```

### Platform-Specific Details

This package works on both Android and iOS platforms without any additional configuration.

```dart
import 'package:flutter/material.dart';
import 'package:iban_utilities_lib/iban_utilities_lib.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'IBAN Utilities Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _ibanController = TextEditingController();
  bool _isValid = false;
  String _formattedIban = '';

  void _validateIban() {
    String iban = _ibanController.text;
    bool isValid = IbanUtilities.validateIban(iban);
    setState(() {
      _isValid = isValid;
    });
  }

  void _formatIban() {
    String iban = _ibanController.text;
    String formattedIban = IbanUtilities.formatIban(iban);
    setState(() {
      _formattedIban = formattedIban;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('IBAN Utilities Demo'),
      ),
      body: Padding(
        padding: const EdgeInsets.all(20.0),
        child: Column(
          children: [
            TextField(
              controller: _ibanController,
              decoration: InputDecoration(
                labelText: 'Enter IBAN',
                border: OutlineInputBorder(),
              ),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _validateIban,
              child: Text('Validate IBAN'),
            ),
            SizedBox(height: 20),
            Text('Is valid: $_isValid'),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _formatIban,
              child: Text('Format IBAN'),
            ),
            SizedBox(height: 20),
            Text('Formatted IBAN: $_formattedIban'),
          ],
        ),
      ),
    );
  }
}

// This is the main entry point of the application.
// The MyApp widget is the root of the application.
// The MyHomePage widget is the home page of the application.
// The _MyHomePageState class is the state of the MyHomePage widget.
// The _validateIban function is called when the "Validate IBAN" button is pressed.
// The _formatIban function is called when the "Format IBAN" button is pressed.
// The IbanUtilities class is used to validate and format the IBAN.

// The application flow is as follows:
// 1. The user enters an IBAN in the text field.
// 2. The user presses the "Validate IBAN" button.
// 3. The _validateIban function is called, which validates the IBAN using the IbanUtilities class.
// 4. The result of the validation is displayed on the screen.
// 5. The user presses the "Format IBAN" button.
// 6. The _formatIban function is called, which formats the IBAN using the IbanUtilities class.
// 7. The formatted IBAN is displayed on the screen.
```
