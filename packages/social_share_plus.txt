# Social Share Plus Flutter Package
=====================================

The `social_share_plus` Flutter package is a powerful tool for sharing content on various social media platforms. It provides a simple and easy-to-use API for sharing text, images, and other media on platforms like Facebook, Twitter, Instagram, and more.

### When to Use This Package

This package is ideal for apps that require social sharing features, such as:

* News and media apps that want to allow users to share articles and videos on social media.
* Photo and video editing apps that want to allow users to share their creations on social media.
* Social media apps that want to allow users to share content with their friends and followers.

### Features

The `social_share_plus` package provides the following features:

* Share text, images, and other media on various social media platforms.
* Support for multiple platforms, including Facebook, Twitter, Instagram, and more.
* Simple and easy-to-use API.
* Customizable share dialog.

# Setting Up the Package
-------------------------

To use the `social_share_plus` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  social_share_plus: ^1.0.0
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:social_share_plus/social_share_plus.dart';
```
### Step 3: Configure the Package (Optional)

You can configure the package to use a custom share dialog by providing a `ShareDialog` instance:
```dart
SocialSharePlus.shareDialog = ShareDialog(
  title: 'Share',
  cancelButtonText: 'Cancel',
);
```
### Platform-Specific Details

#### Android

On Android, you need to add the following permissions to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.INTERNET" />
```
#### iOS

On iOS, you need to add the following lines to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

```dart
import 'package:flutter/material.dart';
import 'package:social_share_plus/social_share_plus.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Social Share Plus Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Social Share Plus Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            ElevatedButton(
              onPressed: () {
                // Share text on Facebook
                SocialSharePlus.share(
                  ShareType.text,
                  'https://www.facebook.com/sharer/sharer.php?u=https://www.example.com',
                  'Example text',
                );
              },
              child: Text('Share on Facebook'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                // Share image on Twitter
                SocialSharePlus.share(
                  ShareType.image,
                  'https://twitter.com/intent/tweet?url=https://www.example.com&text=Example%20image',
                  'https://www.example.com/image.jpg',
                );
              },
              child: Text('Share on Twitter'),
            ),
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user clicks on the "Share on Facebook" button.
// 2. The SocialSharePlus.share method is called with the ShareType.text type and the Facebook sharer URL.
// 3. The user is redirected to the Facebook sharer page with the example text.
// 4. The user clicks on the "Share on Twitter" button.
// 5. The SocialSharePlus.share method is called with the ShareType.image type and the Twitter intent URL.
// 6. The user is redirected to the Twitter intent page with the example image.

// Summary:
// The Social Share Plus package provides a simple and easy-to-use API for sharing content on various social media platforms.
// The package supports multiple platforms, including Facebook, Twitter, Instagram, and more.
// The example code demonstrates how to use the package to share text and images on Facebook and Twitter.
```
