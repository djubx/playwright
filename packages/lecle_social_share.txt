# lecle_social_share Flutter Package
=====================================

The `lecle_social_share` Flutter package is a powerful tool for sharing content on various social media platforms. It provides a simple and easy-to-use API for sharing text, images, and other media on platforms like Facebook, Twitter, Instagram, and more.

### When to Use This Package

This package is ideal for apps that require social sharing features, such as:

* News and media apps that want to allow users to share articles and videos on social media.
* Social media apps that want to allow users to share content with their friends and followers.
* E-commerce apps that want to allow users to share products and promotions on social media.

### Features

The `lecle_social_share` package provides the following features:

* Share text, images, and other media on various social media platforms.
* Support for multiple platforms, including Facebook, Twitter, Instagram, and more.
* Simple and easy-to-use API for sharing content.

# Setup and Usage
==================

To use the `lecle_social_share` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  lecle_social_share: ^1.0.0
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:lecle_social_share/lecle_social_share.dart';
```
### Step 3: Initialize the Package

Initialize the package by calling the `init` method:
```dart
LecleSocialShare.init();
```
### Step 4: Share Content

Share content using the `share` method:
```dart
LecleSocialShare.share(
  text: 'Hello, world!',
  image: 'https://example.com/image.jpg',
  platform: LecleSocialSharePlatform.facebook,
);
```
### Platform-Specific Details

* Android: Make sure to add the following lines to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
```
* iOS: Make sure to add the following lines to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:lecle_social_share/lecle_social_share.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Lecle Social Share Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  void initState() {
    super.initState();
    LecleSocialShare.init();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Lecle Social Share Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text(
              'Share content on social media:',
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                LecleSocialShare.share(
                  text: 'Hello, world!',
                  image: 'https://example.com/image.jpg',
                  platform: LecleSocialSharePlatform.facebook,
                );
              },
              child: Text('Share on Facebook'),
            ),
            SizedBox(height: 10),
            ElevatedButton(
              onPressed: () {
                LecleSocialShare.share(
                  text: 'Hello, world!',
                  image: 'https://example.com/image.jpg',
                  platform: LecleSocialSharePlatform.twitter,
                );
              },
              child: Text('Share on Twitter'),
            ),
          ],
        ),
      ),
    );
  }
}

// The app starts by initializing the LecleSocialShare package in the initState method.
// The user is then presented with a screen that has two buttons: Share on Facebook and Share on Twitter.
// When the user clicks on either button, the LecleSocialShare package is used to share the content on the respective social media platform.
// The content to be shared is defined in the onPressed method of each button.
// The platform on which to share the content is also defined in the onPressed method.
// The LecleSocialShare package takes care of the rest, including opening the social media app and populating the share dialog with the content.
```
