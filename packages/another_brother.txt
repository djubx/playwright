In summary, the code flows as follows:

1.  The user opens the app and the Bluetooth manager is initialized.
2.  The app starts scanning for nearby devices and listens for device discovery events.
3.  When a device is discovered, it is added to the list of devices.
4.  The user selects a device from the list and the app establishes a connection with the device.
5.  The user sends a test message to the connected device using the send data button.
6.  The app receives data from the connected device and prints it to the console.
