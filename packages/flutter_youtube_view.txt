# Flutter YouTube View Package
================================

The `flutter_youtube_view` package is a Flutter library that allows you to embed YouTube videos in your Flutter app. It provides a simple and easy-to-use API for playing YouTube videos.

## When to Use This Package
---------------------------

You can use this package when you want to:

* Play YouTube videos in your Flutter app
* Embed YouTube videos in your app's UI
* Customize the video player's appearance and behavior

## Features
------------

* Play YouTube videos by video ID or URL
* Customize the video player's appearance, such as the player's size, aspect ratio, and controls
* Handle video playback events, such as when the video starts playing, pauses, or ends
* Support for both Android and iOS platforms

# Setup and Usage
==================

## Step 1: Add the Package to Your Pubspec File
---------------------------------------------

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  flutter_youtube_view: ^1.1.0
```
Then, run `flutter pub get` to install the package.

## Step 2: Import the Package
-----------------------------

Import the package in your Dart file:
```dart
import 'package:flutter_youtube_view/flutter_youtube_view.dart';
```
## Step 3: Create a YouTube Video Player
-----------------------------------------

Create a `YouTubeVideoPlayer` widget and pass the video ID or URL to the `videoId` property:
```dart
YouTubeVideoPlayer(
  videoId: 'VIDEO_ID_HERE',
  aspectRatio: 16 / 9,
  autoPlay: true,
)
```
## Platform-Specific Details
---------------------------

### Android

* Make sure to add the following line to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.INTERNET" />
```
### iOS

* Make sure to add the following line to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

```dart
import 'package:flutter/material.dart';
import 'package:flutter_youtube_view/flutter_youtube_view.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter YouTube View Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  String _videoId = 'VIDEO_ID_HERE';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Flutter YouTube View Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            YouTubeVideoPlayer(
              videoId: _videoId,
              aspectRatio: 16 / 9,
              autoPlay: true,
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                setState(() {
                  _videoId = 'ANOTHER_VIDEO_ID_HERE';
                });
              },
              child: Text('Play Another Video'),
            ),
          ],
        ),
      ),
    );
  }
}

// The app starts by running the main function, which creates a MaterialApp with a title and theme.
// The home property is set to MyHomePage, which is a stateful widget.
// MyHomePage has a Scaffold with an AppBar and a body that contains a Column with a YouTubeVideoPlayer and an ElevatedButton.
// The YouTubeVideoPlayer is created with a videoId, aspectRatio, and autoPlay properties.
// When the ElevatedButton is pressed, the _videoId is updated, and the setState function is called to rebuild the widget tree.
// This causes the YouTubeVideoPlayer to update with the new videoId.
```
