# Flutter Nanonets OCR Package
================================

The `flutter_nanonets_ocr` package is a powerful tool for Optical Character Recognition (OCR) in Flutter applications. It uses the Nanonets API to recognize text in images and provides a simple, easy-to-use interface for developers.

## When to Use This Package
---------------------------

This package is ideal for applications that require text recognition from images, such as:

* Document scanning and processing
* Image-based data entry
* Text recognition from photographs

## Features
------------

* Recognize text in images using the Nanonets API
* Support for multiple image formats (JPEG, PNG, etc.)
* Simple, easy-to-use interface for developers

# Setting Up the Package
-------------------------

To use the `flutter_nanonets_ocr` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  flutter_nanonets_ocr: ^1.0.0
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:flutter_nanonets_ocr/flutter_nanonets_ocr.dart';
```
### Step 3: Set Up the Nanonets API

Create a Nanonets API account and obtain an API key. Set up the API key in your Dart file:
```dart
String apiKey = 'YOUR_API_KEY_HERE';
```
### Step 4: Recognize Text in an Image

Use the `recognizeText` method to recognize text in an image:
```dart
File imageFile = File('path/to/image.jpg');
String recognizedText = await FlutterNanonetsOcr.recognizeText(imageFile, apiKey);
```
### Platform-Specific Details

* Android: Add the following line to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.INTERNET" />
```
* iOS: Add the following line to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
    <key>NSAllowsArbitraryLoads</key>
    <true/>
</dict>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:flutter_nanonets_ocr/flutter_nanonets_ocr.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Nanonets OCR Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  String _recognizedText = '';

  // Step 1: Set up the Nanonets API key
  String _apiKey = 'YOUR_API_KEY_HERE';

  // Step 2: Select an image file
  File _imageFile;

  // Step 3: Recognize text in the image
  void _recognizeText() async {
    // Use the recognizeText method to recognize text in the image
    String recognizedText = await FlutterNanonetsOcr.recognizeText(_imageFile, _apiKey);

    // Update the UI with the recognized text
    setState(() {
      _recognizedText = recognizedText;
    });
  }

  // Step 4: Display the recognized text
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Flutter Nanonets OCR Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            // Display the selected image
            _imageFile != null
                ? Image.file(_imageFile)
                : Text('No image selected'),

            // Display the recognized text
            Text(_recognizedText),

            // Button to select an image
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () async {
                // Select an image file
                final pickedFile = await ImagePicker().getImage(source: ImageSource.gallery);
                setState(() {
                  if (pickedFile != null) {
                    _imageFile = File(pickedFile.path);
                  } else {
                    _imageFile = null;
                  }
                });
              },
              child: Text('Select Image'),
            ),

            // Button to recognize text
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _recognizeText,
              child: Text('Recognize Text'),
            ),
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user selects an image file using the ImagePicker.
// 2. The selected image is displayed on the screen.
// 3. The user clicks the "Recognize Text" button.
// 4. The recognizeText method is called to recognize text in the image.
// 5. The recognized text is displayed on the screen.

// Summary:
// This code demonstrates how to use the Flutter Nanonets OCR package to recognize text in an image.
// The user selects an image file, and then clicks the "Recognize Text" button to recognize text in the image.
// The recognized text is then displayed on the screen.
```
