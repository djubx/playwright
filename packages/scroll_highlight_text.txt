**Scroll Highlight Text Package**
================================

The `scroll_highlight_text` package is a Flutter library that allows you to highlight text as you scroll through a list of items. This package is useful when you want to draw attention to specific text or items in a list.

**When to Use This Package**
---------------------------

You can use this package in various scenarios, such as:

* Highlighting search results in a list
* Drawing attention to specific items in a list
* Creating a tutorial or guided tour of your app

**Features**
------------

* Highlights text as you scroll through a list
* Customizable highlight color and style
* Supports multiple highlight colors and styles

<-- END_DESCRIPTION -->

<-- START_TUTORIAL -->

**Setup and Usage**
-------------------

To use the `scroll_highlight_text` package, follow these steps:

1. Add the package to your `pubspec.yaml` file:
```yml
dependencies:
  scroll_highlight_text: ^1.0.0
```
2. Import the package in your Dart file:
```dart
import 'package:scroll_highlight_text/scroll_highlight_text.dart';
```
3. Create a `ScrollHighlightText` widget and pass in the text to highlight:
```dart
ScrollHighlightText(
  text: 'Hello World!',
  highlightColor: Colors.yellow,
)
```
4. You can customize the highlight color and style by passing in additional properties:
```dart
ScrollHighlightText(
  text: 'Hello World!',
  highlightColor: Colors.yellow,
  highlightStyle: TextStyle(fontSize: 24),
)
```
**Platform-Specific Details**
---------------------------

* Android: No additional configuration is required.
* iOS: You need to add the following line to your `Info.plist` file:
```xml
<key>NSAllowsArbitraryLoads</key>
<true/>
```
This allows the app to load arbitrary URLs, which is required for the highlight text feature to work.

**Optimizations**
----------------

To improve performance, you can use the `cache` property to cache the highlighted text:
```dart
ScrollHighlightText(
  text: 'Hello World!',
  highlightColor: Colors.yellow,
  cache: true,
)
```
This will cache the highlighted text and reuse it when the widget is rebuilt.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:scroll_highlight_text/scroll_highlight_text.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Scroll Highlight Text Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _scrollController = ScrollController();

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Scroll Highlight Text Demo'),
      ),
      body: ListView.builder(
        controller: _scrollController,
        itemCount: 100,
        itemBuilder: (context, index) {
          return ListTile(
            title: ScrollHighlightText(
              text: 'Item $index',
              highlightColor: Colors.yellow,
            ),
          );
        },
      ),
    );
  }
}

// Application flow:
// 1. The user scrolls through the list of items.
// 2. As the user scrolls, the `ScrollHighlightText` widget highlights the text.
// 3. The highlighted text is displayed in yellow.
// 4. The user can continue scrolling through the list, and the highlighted text will update accordingly.

// Summary:
// The `ScrollHighlightText` package allows you to highlight text as you scroll through a list of items.
// The package is easy to use and customize, and it provides a smooth and seamless user experience.
```
