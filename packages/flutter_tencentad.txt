# Flutter TencentAd Package Overview
=====================================

The `flutter_tencentad` package is a Flutter plugin that allows developers to integrate Tencent ads into their Flutter applications. Tencent is a popular advertising platform in China, and this package provides a convenient way to monetize Flutter apps with Tencent ads.

## When to Use This Package
---------------------------

You should use this package if you want to display Tencent ads in your Flutter app. This package supports various ad formats, including banner ads, interstitial ads, and rewarded video ads.

## Features
------------

*   Supports banner ads, interstitial ads, and rewarded video ads
*   Allows customization of ad layout and behavior
*   Provides callbacks for ad events, such as clicks and impressions

# Setting Up the Flutter TencentAd Package
=============================================

To use the `flutter_tencentad` package, follow these steps:

### Step 1: Add the Package to Your Flutter Project

Add the following dependency to your `pubspec.yaml` file:

```yml
dependencies:
  flutter_tencentad: ^1.0.0
```

Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:flutter_tencentad/flutter_tencentad.dart';
```

### Step 3: Initialize the TencentAd SDK

Initialize the TencentAd SDK in your app's `main` function:

```dart
void main() {
  runApp(MyApp());
  TencentAd.init(appId: 'your_app_id');
}
```

Replace `'your_app_id'` with your actual TencentAd app ID.

### Step 4: Create an Ad Widget

Create an ad widget using one of the following methods:

*   `TencentAdBanner`: creates a banner ad widget
*   `TencentAdInterstitial`: creates an interstitial ad widget
*   `TencentAdRewardedVideo`: creates a rewarded video ad widget

For example, to create a banner ad widget:

```dart
TencentAdBanner(
  adUnitId: 'your_ad_unit_id',
  width: 320,
  height: 50,
)
```

Replace `'your_ad_unit_id'` with your actual TencentAd ad unit ID.

### Platform-Specific Details

*   **Android:** Make sure to add the following permissions to your `AndroidManifest.xml` file:

    ```xml
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />
```

*   **iOS:** Make sure to add the following frameworks to your `Info.plist` file:

    ```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

```dart
import 'package:flutter/material.dart';
import 'package:flutter_tencentad/flutter_tencentad.dart';

void main() {
  runApp(MyApp());
  TencentAd.init(appId: 'your_app_id');
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('TencentAd Demo'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              TencentAdBanner(
                adUnitId: 'your_ad_unit_id',
                width: 320,
                height: 50,
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  TencentAdInterstitial(
                    adUnitId: 'your_ad_unit_id',
                  ).show();
                },
                child: Text('Show Interstitial Ad'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  TencentAdRewardedVideo(
                    adUnitId: 'your_ad_unit_id',
                  ).show();
                },
                child: Text('Show Rewarded Video Ad'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// The app starts by initializing the TencentAd SDK with the app ID.
// Then, it creates a MaterialApp with a Scaffold containing an AppBar and a Column.
// The Column contains a TencentAdBanner widget, which displays a banner ad.
// Below the banner ad, there are two ElevatedButton widgets.
// The first button shows an interstitial ad using the TencentAdInterstitial widget.
// The second button shows a rewarded video ad using the TencentAdRewardedVideo widget.
// When the user clicks on an ad, the ad's callback functions are called.
// For example, when the user clicks on the banner ad, the ad's onClick callback is called.
// The ad's callback functions can be used to track ad events, such as clicks and impressions.
```
