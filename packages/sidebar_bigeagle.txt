# Sidebar BigEagle Flutter Package
The `sidebar_bigeagle` Flutter package is a customizable sidebar navigation menu that can be used to enhance the user experience of your Flutter applications. It provides a modern and sleek design that can be easily integrated into your existing Flutter projects.

## Features
- Customizable sidebar navigation menu
- Support for multiple menu items
- Optional header and footer sections
- Smooth animation effects
- Compatible with both Android and iOS platforms

## When to Use
The `sidebar_bigeagle` package is ideal for applications that require a navigation menu with multiple options. It can be used in various scenarios, such as:

- Navigation menus for main application screens
- Settings menus for application configuration
- Context menus for specific actions

# Setup and Usage
To use the `sidebar_bigeagle` package in your Flutter project, follow these steps:

1. Add the package to your `pubspec.yaml` file:
```yml
dependencies:
  sidebar_bigeagle: ^1.0.0
```
2. Run `flutter pub get` to install the package.
3. Import the package in your Dart file:
```dart
import 'package:sidebar_bigeagle/sidebar_bigeagle.dart';
```
4. Create a `SidebarBigEagle` widget and customize its properties:
```dart
SidebarBigEagle(
  header: Text('Header'),
  footer: Text('Footer'),
  items: [
    SidebarItem(
      icon: Icons.home,
      label: 'Home',
      onTap: () {
        // Handle tap event
      },
    ),
    SidebarItem(
      icon: Icons.settings,
      label: 'Settings',
      onTap: () {
        // Handle tap event
      },
    ),
  ],
)
```
5. Add the `SidebarBigEagle` widget to your application's layout.

### Platform-Specific Details
- Android: No additional configuration required.
- iOS: Add the following code to your `Info.plist` file to enable the sidebar menu:
```xml
<key>UIViewControllerBasedStatusBarAppearance</key>
<false/>
```
