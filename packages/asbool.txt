# asbool Flutter Package

The `asbool` Flutter package is a lightweight utility designed to simplify the conversion of various data types into boolean values. This package is particularly useful in scenarios where you need to handle dynamic data sources, such as APIs or user inputs, which may not always return boolean values directly. 

## When to Use `asbool`

You might find `asbool` beneficial in the following situations:
- **API Responses**: When dealing with APIs that return strings like "true" or "false" instead of actual boolean values.
- **User Inputs**: When capturing user inputs that may be in different formats (e.g., checkboxes, switches) and need to be converted to boolean.
- **Configuration Settings**: When reading configuration files or environment variables that may return strings or integers instead of booleans.

## Features
- **Type Safety**: Ensures that the conversion is handled safely, reducing the risk of runtime errors.
- **Flexible Input Handling**: Can handle various input types, including strings, integers, and null values.
- **Simple API**: Easy to integrate and use within your Flutter applications.

## Example Usage
```dart
import 'package:asbool/asbool.dart';

void main() {
  // Example of using asbool
  bool isActive = asbool('true'); // Converts string 'true' to boolean true
  bool isEnabled = asbool(1);      // Converts integer 1 to boolean true
  bool isVisible = asbool(null);   // Converts null to boolean false

  print(isActive);   // Output: true
  print(isEnabled);  // Output: true
  print(isVisible);  // Output: false
}
```

# Tutorial: Setting Up and Using the asbool Package

## Step 1: Adding the Dependency
To use the `asbool` package in your Flutter project, you need to add it to your `pubspec.yaml` file. Open the file and add the following line under dependencies:

```yaml
dependencies:
  flutter:
    sdk: flutter
  asbool: ^1.0.0  # Check for the latest version on pub.dev
```

## Step 2: Installing the Package
After adding the dependency, run the following command in your terminal to install the package:

```bash
flutter pub get
```

## Step 3: Importing the Package
In your Dart files where you want to use the `asbool` functionality, import the package:

```dart
import 'package:asbool/asbool.dart';
```

## Platform-Specific Details
### Android
No specific configurations are required for Android. Just ensure that your `minSdkVersion` is set to at least 21 in your `android/app/build.gradle` file.
