# gma_mediation_liftoffmonetize Flutter Package
=============================================

The `gma_mediation_liftoffmonetize` package is a Flutter plugin that provides a simple way to integrate Google Mobile Ads (GMA) mediation with Liftoff Monetize. This package allows you to easily display ads from multiple ad networks, including Liftoff Monetize, in your Flutter app.

## Features

* Supports banner, interstitial, and rewarded video ads
* Easy integration with Google Mobile Ads mediation
* Supports Liftoff Monetize ad network

## When to Use

Use this package when you want to display ads from multiple ad networks, including Liftoff Monetize, in your Flutter app. This package is particularly useful when you want to use Google Mobile Ads mediation to manage your ad inventory.

# Setup and Usage
================

## Step 1: Add the Package to Your Pubspec.yaml File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  gma_mediation_liftoffmonetize: ^1.0.0
```
## Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:gma_mediation_liftoffmonetize/gma_mediation_liftoffmonetize.dart';
```
## Step 3: Initialize the Package

Initialize the package in your app's `main` function:
```dart
void main() {
  WidgetsFlutterBinding.ensureInitialized();
  GmaMediationLiftoffmonetize.initialize();
  runApp(MyApp());
}
```
## Step 4: Create an Ad Unit

Create an ad unit in your app's UI:
```dart
GmaMediationLiftoffmonetizeAdUnit adUnit = GmaMediationLiftoffmonetizeAdUnit(
  adUnitId: 'YOUR_AD_UNIT_ID',
  adFormat: GmaMediationLiftoffmonetizeAdFormat.banner,
);
```
## Step 5: Load and Display the Ad

Load and display the ad:
```dart
adUnit.loadAd().then((ad) {
  adUnit.showAd();
});
```
## Platform-Specific Details

### Android

Add the following lines to your `AndroidManifest.xml` file:
```xml
<application>
  <meta-data
    android:name="com.google.android.gms.ads.APPLICATION_ID"
    android:value="YOUR_ADMOB_APP_ID" />
</application>
```
### iOS

Add the following lines to your `Info.plist` file:
```xml
<key>GADApplicationIdentifier</key>
<string>YOUR_ADMOB_APP_ID</string>
```

```dart
import 'package:flutter/material.dart';
import 'package:gma_mediation_liftoffmonetize/gma_mediation_liftoffmonetize.dart';

void main() {
  // Initialize the package
  WidgetsFlutterBinding.ensureInitialized();
  GmaMediationLiftoffmonetize.initialize();

  // Run the app
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  // Create an ad unit
  GmaMediationLiftoffmonetizeAdUnit adUnit = GmaMediationLiftoffmonetizeAdUnit(
    adUnitId: 'YOUR_AD_UNIT_ID',
    adFormat: GmaMediationLiftoffmonetizeAdFormat.banner,
  );

  @override
  void initState() {
    super.initState();

    // Load the ad
    adUnit.loadAd().then((ad) {
      // Display the ad
      adUnit.showAd();
    });
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Real Flutter'),
        ),
        body: Center(
          child: Text('Ad is loading...'),
        ),
      ),
    );
  }
}

// The app flow is as follows:
// 1. The package is initialized in the main function.
// 2. An ad unit is created with the ad unit ID and ad format.
// 3. The ad is loaded using the loadAd method.
// 4. Once the ad is loaded, it is displayed using the showAd method.
// 5. The app UI is built using the MaterialApp and Scaffold widgets.
// 6. The ad is displayed at the bottom of the screen.

// Note: Replace 'YOUR_AD_UNIT_ID' with your actual ad unit ID.
```
