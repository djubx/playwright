# Text Link Flutter Package
The `text_link` Flutter package is a powerful tool for creating clickable links within text widgets. It allows developers to embed links, emails, and phone numbers within text, making it easier for users to interact with the app.

## When to Use
This package is useful when you need to display text that contains links, such as:

* Terms and conditions
* Privacy policies
* Instructional content
* News articles

## Features
The `text_link` package offers the following features:

* Clickable links
* Email and phone number detection
* Customizable link styles
* Support for multiple link types (e.g., HTTP, HTTPS, FTP)

# Setting Up the Package
To use the `text_link` package, follow these steps:

1. Add the package to your `pubspec.yaml` file:
```yml
dependencies:
  text_link: ^1.0.0
```
2. Run `flutter pub get` to install the package.
3. Import the package in your Dart file:
```dart
import 'package:text_link/text_link.dart';
```

## Platform-Specific Details
For Android, you need to add the following configuration to your `AndroidManifest.xml` file:
```xml
<application>
  ...
  <uses-permission android:name="android.permission.INTERNET" />
</application>
```
For iOS, you need to add the following configuration to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

## Using the Package
To use the package, create a `TextLink` widget and pass in the text and link styles:
```dart
TextLink(
  text: 'Visit our website: https://example.com',
  linkStyle: TextStyle(color: Colors.blue),
)
```
You can also customize the link detection by passing in a `LinkDetector` object:
```dart
TextLink(
  text: 'Call us: 555-555-5555',
  linkDetector: LinkDetector(
    schemes: ['tel'],
  ),
)
```

```dart
import 'package:flutter/material.dart';
import 'package:text_link/text_link.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Text Link Example',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Text Link Example'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              // Create a TextLink widget with a clickable link
              TextLink(
                text: 'Visit our website: https://example.com',
                linkStyle: TextStyle(color: Colors.blue),
              ),
              SizedBox(height: 20),
              // Create a TextLink widget with a clickable email
              TextLink(
                text: 'Email us: example@example.com',
                linkDetector: LinkDetector(
                  schemes: ['mailto'],
                ),
              ),
              SizedBox(height: 20),
              // Create a TextLink widget with a clickable phone number
              TextLink(
                text: 'Call us: 555-555-5555',
                linkDetector: LinkDetector(
                  schemes: ['tel'],
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user runs the app and is presented with a screen containing three TextLink widgets.
// 2. The user can click on each TextLink widget to open the corresponding link, email, or phone number.
// 3. The link detection is handled by the LinkDetector object, which can be customized to detect specific schemes.
// 4. The TextLink widget uses the linkStyle property to customize the appearance of the link.
// 5. The app uses the Material Design theme to provide a consistent look and feel.

```
