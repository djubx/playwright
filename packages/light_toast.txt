Light Toast Flutter Package
==========================

The `light_toast` Flutter package is a lightweight and customizable toast notification library for Flutter applications. It provides a simple and easy-to-use API for displaying toast notifications on both Android and iOS platforms.

When to Use
------------

The `light_toast` package is ideal for displaying temporary notifications or alerts to the user, such as:

* Success or failure messages after a network request
* Validation errors or warnings
* Temporary reminders or notifications

Features
--------

* Customizable toast notification styles and layouts
* Support for displaying toast notifications on both Android and iOS platforms
* Simple and easy-to-use API

Setup and Usage
---------------

To use the `light_toast` package in your Flutter application, follow these steps:

1. Add the `light_toast` package to your `pubspec.yaml` file:
```yml
dependencies:
  light_toast: ^1.0.0
```
2. Run `flutter pub get` to install the package.
3. Import the `light_toast` package in your Dart file:
```dart
import 'package:light_toast/light_toast.dart';
```
4. Create a `LightToast` instance and call the `show` method to display a toast notification:
```dart
LightToast().show(
  context: context,
  message: 'Hello, World!',
  duration: Duration(seconds: 2),
);
```
Platform-Specific Details
-------------------------

### Android

On Android, the `light_toast` package uses the `Toast` class to display toast notifications. You can customize the toast notification style by using the `ToastStyle` class:
```dart
LightToast().show(
  context: context,
  message: 'Hello, World!',
  duration: Duration(seconds: 2),
  style: ToastStyle(
    backgroundColor: Colors.blue,
    textColor: Colors.white,
  ),
);
```
### iOS

On iOS, the `light_toast` package uses the `UIAlertController` class to display toast notifications. You can customize the toast notification style by using the `UIAlertControllerStyle` class:
```dart
LightToast().show(
  context: context,
  message: 'Hello, World!',
  duration: Duration(seconds: 2),
  style: UIAlertControllerStyle(
    preferredStyle: UIAlertControllerStyle.alert,
  ),
);
```

```dart
import 'package:flutter/material.dart';
import 'package:light_toast/light_toast.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Light Toast Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Light Toast Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            ElevatedButton(
              onPressed: () {
                // Show a success toast notification
                LightToast().show(
                  context: context,
                  message: 'Success!',
                  duration: Duration(seconds: 2),
                  style: ToastStyle(
                    backgroundColor: Colors.green,
                    textColor: Colors.white,
                  ),
                );
              },
              child: Text('Show Success Toast'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                // Show an error toast notification
                LightToast().show(
                  context: context,
                  message: 'Error!',
                  duration: Duration(seconds: 2),
                  style: ToastStyle(
                    backgroundColor: Colors.red,
                    textColor: Colors.white,
                  ),
                );
              },
              child: Text('Show Error Toast'),
            ),
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user clicks on the "Show Success Toast" button.
// 2. The `LightToast` instance is created and the `show` method is called with the success message and style.
// 3. The toast notification is displayed on the screen for 2 seconds.
// 4. The user clicks on the "Show Error Toast" button.
// 5. The `LightToast` instance is created and the `show` method is called with the error message and style.
// 6. The toast notification is displayed on the screen for 2 seconds.

// Summary:
// The `light_toast` package is used to display toast notifications on the screen.
// The `LightToast` instance is created and the `show` method is called with the message and style.
// The toast notification is displayed on the screen for a specified duration.
```
