

// The application flow is as follows:

// 2. The `MyApp` widget is built, which returns a `MaterialApp` widget.

// 4. The `MyHomePage` widget uses the `ResponsiveBuilder` widget to detect the screen size and device type.
// 5. Based on the screen size and device type, the `MyHomePage` widget builds the corresponding layout (mobile, tablet, or desktop).
// 6. The layout is then displayed to the user.
