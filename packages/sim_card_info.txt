# sim_card_info Flutter Package
================================

The `sim_card_info` Flutter package provides a simple way to access SIM card information on Android and iOS devices. This package can be used to retrieve details such as the SIM card's ICCID, IMSI, and phone number.

## When to Use This Package
---------------------------

This package is useful when you need to:

* Retrieve SIM card information for authentication or verification purposes
* Display SIM card details to the user
* Use SIM card information for analytics or logging purposes

## Features
------------

The `sim_card_info` package provides the following features:

* Retrieve SIM card ICCID, IMSI, and phone number
* Supports Android and iOS platforms
* Easy-to-use API for accessing SIM card information

# Setup and Usage
-----------------

## Step 1: Add the Package to Your Flutter Project
------------------------------------------------

Add the following dependency to your `pubspec.yaml` file:
```yml
dependencies:
  sim_card_info: ^1.0.0
```
Then, run `flutter pub get` to install the package.

## Step 2: Import the Package
---------------------------

Import the package in your Dart file:
```dart
import 'package:sim_card_info/sim_card_info.dart';
```
## Step 3: Request Permissions (Android Only)
------------------------------------------

On Android, you need to request the `READ_PHONE_STATE` permission to access SIM card information. Add the following code to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.READ_PHONE_STATE" />
```
## Step 4: Use the Package
-------------------------

Create an instance of the `SimCardInfo` class and call the `getSimCardInfo` method to retrieve SIM card information:
```dart
SimCardInfo simCardInfo = SimCardInfo();

simCardInfo.getSimCardInfo().then((info) {
  print('ICCID: ${info.iccid}');
  print('IMSI: ${info.imsi}');
  print('Phone Number: ${info.phoneNumber}');
});
```

```dart
import 'package:flutter/material.dart';
import 'package:sim_card_info/sim_card_info.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Sim Card Info Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  String _iccid = '';
  String _imsi = '';
  String _phoneNumber = '';

  @override
  void initState() {
    super.initState();
    _getSimCardInfo();
  }

  Future<void> _getSimCardInfo() async {
    // Create an instance of the SimCardInfo class
    SimCardInfo simCardInfo = SimCardInfo();

    // Call the getSimCardInfo method to retrieve SIM card information
    SimCardInfoData info = await simCardInfo.getSimCardInfo();

    // Update the UI with the retrieved information
    setState(() {
      _iccid = info.iccid;
      _imsi = info.imsi;
      _phoneNumber = info.phoneNumber;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Sim Card Info Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text('ICCID: $_iccid'),
            Text('IMSI: $_imsi'),
            Text('Phone Number: $_phoneNumber'),
          ],
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user opens the app, and the _getSimCardInfo method is called in the initState method.
// 2. The _getSimCardInfo method creates an instance of the SimCardInfo class and calls the getSimCardInfo method to retrieve SIM card information.
// 3. The getSimCardInfo method returns a Future that resolves with the SIM card information.
// 4. The _getSimCardInfo method updates the UI with the retrieved information by calling the setState method.
// 5. The UI is updated with the ICCID, IMSI, and phone number of the SIM card.

// In summary, the code flows as follows:
// initState -> _getSimCardInfo -> getSimCardInfo -> setState -> UI update
```
