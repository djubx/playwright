**Introduction to dart_periphery**
=====================================

The `dart_periphery` package is a Flutter library that provides a set of APIs to interact with the peripheral devices of a mobile device, such as the camera, microphone, and GPS. This package is useful when you need to access these devices in your Flutter application.

**When to use dart_periphery**
-----------------------------

You can use `dart_periphery` in the following scenarios:

*   When you need to capture images or videos using the device's camera.
*   When you need to record audio using the device's microphone.
*   When you need to access the device's location using the GPS.

**Features of dart_periphery**
------------------------------

The `dart_periphery` package provides the following features:

*   Camera API: allows you to capture images and videos using the device's camera.
*   Microphone API: allows you to record audio using the device's microphone.
*   GPS API: allows you to access the device's location using the GPS.

**Setting up dart_periphery**
=============================

To use the `dart_periphery` package in your Flutter application, follow these steps:

### Step 1: Add the package to your pubspec.yaml file

```yml
dependencies:
  flutter:
    sdk: flutter
  dart_periphery: ^1.0.0
```

### Step 2: Import the package in your Dart file

```dart
import 'package:dart_periphery/dart_periphery.dart';
```

### Step 3: Request permissions

Before you can use the `dart_periphery` package, you need to request the necessary permissions. You can do this using the `permission_handler` package.

```dart
import 'package:permission_handler/permission_handler.dart';

Future<void> requestPermissions() async {
  await Permission.camera.request();
  await Permission.microphone.request();
  await Permission.location.request();
}
```

### Step 4: Initialize the package

```dart
Future<void> initPeriphery() async {
  await Periphery.initialize();
}
```

### Platform-specific details

*   Android: You need to add the following lines to your `AndroidManifest.xml` file:

    ```xml
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.RECORD_AUDIO" />
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
```

*   iOS: You need to add the following lines to your `Info.plist` file:

    ```xml
<key>NSCameraUsageDescription</key>
<string>This app needs camera access to work</string>
<key>NSMicrophoneUsageDescription</key>
<string>This app needs microphone access to work</string>
<key>NSLocationWhenInUseUsageDescription</key>
<string>This app needs location access to work</string>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:dart_periphery/dart_periphery.dart';
import 'package:permission_handler/permission_handler.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Dart Periphery Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  // Request permissions when the app starts
  @override
  void initState() {
    super.initState();
    requestPermissions();
  }

  // Request permissions
  Future<void> requestPermissions() async {
    // Request camera permission
    await Permission.camera.request();
    // Request microphone permission
    await Permission.microphone.request();
    // Request location permission
    await Permission.location.request();
  }

  // Initialize the package when the button is pressed
  Future<void> initPeriphery() async {
    // Initialize the package
    await Periphery.initialize();
  }

  // Capture an image when the button is pressed
  Future<void> captureImage() async {
    // Get the camera instance
    Camera camera = await Periphery.getCamera();
    // Capture an image
    await camera.captureImage();
  }

  // Record audio when the button is pressed
  Future<void> recordAudio() async {
    // Get the microphone instance
    Microphone microphone = await Periphery.getMicrophone();
    // Record audio
    await microphone.recordAudio();
  }

  // Get the location when the button is pressed
  Future<void> getLocation() async {
    // Get the GPS instance
    GPS gps = await Periphery.getGPS();
    // Get the location
    await gps.getLocation();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Dart Periphery Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            ElevatedButton(
              onPressed: initPeriphery,
              child: Text('Initialize Periphery'),
            ),
            ElevatedButton(
              onPressed: captureImage,
              child: Text('Capture Image'),
            ),
            ElevatedButton(
              onPressed: recordAudio,
              child: Text('Record Audio'),
            ),
            ElevatedButton(
              onPressed: getLocation,
              child: Text('Get Location'),
            ),
          ],
        ),
      ),
    );
  }
}

// The app starts by requesting permissions.
// When the permissions are granted, the app initializes the package.
// After the package is initialized, the app can capture images, record audio, and get the location.
// Each of these actions is performed when the corresponding button is pressed.
```
