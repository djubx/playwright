Apphud Flutter Package: A Comprehensive Overview
=====================================================

The Apphud Flutter package is a powerful tool for managing in-app purchases and subscriptions on both Android and iOS platforms. It provides a simple and unified API for handling various tasks, such as fetching products, making purchases, and checking subscription status.

When to Use Apphud
------------------

Apphud is ideal for applications that offer in-app purchases or subscriptions, such as:

* Games with in-game purchases or premium features
* Music or video streaming services with subscription-based models
* Productivity apps with premium features or subscription-based services

Features of Apphud
------------------

* Unified API for Android and iOS platforms
* Support for various purchase types, including consumable, non-consumable, and subscription-based products
* Automatic handling of purchase validation and receipt verification
* Real-time updates for subscription status and purchase history

Setting Up Apphud
-----------------

To use Apphud in your Flutter application, follow these steps:

### Step 1: Add the Apphud Package to Your Project

Add the following dependency to your `pubspec.yaml` file:
```yml
dependencies:
  apphud: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Configure Apphud for Android

In your `AndroidManifest.xml` file, add the following code:
```xml
<application>
  ...
  <meta-data
    android:name="com.apphud.api_key"
    android:value="YOUR_API_KEY" />
</application>
```
Replace `YOUR_API_KEY` with your actual Apphud API key.

### Step 3: Configure Apphud for iOS

In your `Info.plist` file, add the following code:
```xml
<key>ApphudApiKey</key>
<string>YOUR_API_KEY</string>
```
Replace `YOUR_API_KEY` with your actual Apphud API key.

### Step 4: Initialize Apphud

In your Dart code, import the Apphud package and initialize it:
```dart
import 'package:apphud/apphud.dart';

void main() {
  Apphud.init('YOUR_API_KEY');
  runApp(MyApp());
}
```
Replace `YOUR_API_KEY` with your actual Apphud API key.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:apphud/apphud.dart';

void main() {
  Apphud.init('YOUR_API_KEY');
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  // Step 1: Fetch products from Apphud
  Future<void> _fetchProducts() async {
    final products = await Apphud.fetchProducts();
    print('Fetched products: $products');
  }

  // Step 2: Make a purchase using Apphud
  Future<void> _makePurchase() async {
    final product = await Apphud.fetchProduct('product_id');
    if (product != null) {
      final purchase = await Apphud.makePurchase(product);
      print('Made purchase: $purchase');
    } else {
      print('Product not found');
    }
  }

  // Step 3: Check subscription status using Apphud
  Future<void> _checkSubscriptionStatus() async {
    final status = await Apphud.checkSubscriptionStatus();
    print('Subscription status: $status');
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Apphud Example'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              ElevatedButton(
                onPressed: _fetchProducts,
                child: Text('Fetch Products'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: _makePurchase,
                child: Text('Make Purchase'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: _checkSubscriptionStatus,
                child: Text('Check Subscription Status'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user opens the app and the Apphud package is initialized.
// 2. The user clicks the "Fetch Products" button, which calls the _fetchProducts function.
// 3. The _fetchProducts function fetches the products from Apphud and prints the result.
// 4. The user clicks the "Make Purchase" button, which calls the _makePurchase function.
// 5. The _makePurchase function makes a purchase using Apphud and prints the result.
// 6. The user clicks the "Check Subscription Status" button, which calls the _checkSubscriptionStatus function.
// 7. The _checkSubscriptionStatus function checks the subscription status using Apphud and prints the result.

// In summary, this example demonstrates how to use the Apphud package to fetch products, make purchases, and check subscription status.
```
