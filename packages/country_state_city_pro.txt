# Country State City Pro Flutter Package
=====================================

The `country_state_city_pro` Flutter package is a powerful tool for handling country, state, and city data in your Flutter applications. It provides a simple and efficient way to manage this data, making it ideal for applications that require location-based information.

### When to Use This Package

This package is perfect for applications that require users to select their country, state, or city. Some examples of such applications include:

* E-commerce platforms that require users to select their location for shipping purposes
* Travel applications that need to provide location-based information
* Social media platforms that require users to select their location

### Features

The `country_state_city_pro` package comes with the following features:

* A comprehensive list of countries, states, and cities
* Easy-to-use API for accessing country, state, and city data
* Support for filtering and sorting data

# Setting Up the Package
-------------------------

To use the `country_state_city_pro` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  country_state_city_pro: ^1.0.0
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:country_state_city_pro/country_state_city_pro.dart';
```
### Step 3: Initialize the Package

Initialize the package by calling the `init` method:
```dart
await CountryStateCityPro.init();
```
### Platform-Specific Details

#### Android

No additional configuration is required for Android.

#### iOS

No additional configuration is required for iOS.

### Required Configurations or Optimizations

No additional configurations or optimizations are required.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:country_state_city_pro/country_state_city_pro.dart';

void main() async {
  // Initialize the package
  await CountryStateCityPro.init();

  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Country State City Pro Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  // List of countries
  List<Country> _countries = [];

  // Selected country
  Country _selectedCountry;

  // List of states
  List<State> _states = [];

  // Selected state
  State _selectedState;

  // List of cities
  List<City> _cities = [];

  // Selected city
  City _selectedCity;

  @override
  void initState() {
    super.initState();

    // Get the list of countries
    CountryStateCityPro.getCountries().then((countries) {
      setState(() {
        _countries = countries;
      });
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Country State City Pro Demo'),
      ),
      body: Padding(
        padding: const EdgeInsets.all(20.0),
        child: Column(
          children: <Widget>[
            // Country dropdown
            DropdownButton<Country>(
              value: _selectedCountry,
              onChanged: (country) {
                setState(() {
                  _selectedCountry = country;

                  // Get the list of states for the selected country
                  CountryStateCityPro.getStates(country.id).then((states) {
                    setState(() {
                      _states = states;
                    });
                  });
                });
              },
              items: _countries.map((country) {
                return DropdownMenuItem<Country>(
                  value: country,
                  child: Text(country.name),
                );
              }).toList(),
            ),

            // State dropdown
            DropdownButton<State>(
              value: _selectedState,
              onChanged: (state) {
                setState(() {
                  _selectedState = state;

                  // Get the list of cities for the selected state
                  CountryStateCityPro.getCities(state.id).then((cities) {
                    setState(() {
                      _cities = cities;
                    });
                  });
                });
              },
              items: _states.map((state) {
                return DropdownMenuItem<State>(
                  value: state,
                  child: Text(state.name),
                );
              }).toList(),
            ),

            // City dropdown
            DropdownButton<City>(
              value: _selectedCity,
              onChanged: (city) {
                setState(() {
                  _selectedCity = city;
                });
              },
              items: _cities.map((city) {
                return DropdownMenuItem<City>(
                  value: city,
                  child: Text(city.name),
                );
              }).toList(),
            ),
          ],
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user selects a country from the country dropdown.
// 2. The list of states for the selected country is retrieved and displayed in the state dropdown.
// 3. The user selects a state from the state dropdown.
// 4. The list of cities for the selected state is retrieved and displayed in the city dropdown.
// 5. The user selects a city from the city dropdown.
```
