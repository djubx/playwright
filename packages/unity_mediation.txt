# Unity Mediation Flutter Package
=====================================

The `unity_mediation` Flutter package is a powerful tool for integrating Unity Ads into your Flutter applications. It provides a simple and efficient way to display ads, handle ad events, and reward users.

## When to Use This Package
---------------------------

You should use this package when you want to monetize your Flutter app with Unity Ads. This package is particularly useful when you want to:

* Display video ads, such as rewarded videos or interstitial ads
* Handle ad events, such as ad completion or ad clicks
* Reward users for watching ads

## Features
------------

The `unity_mediation` package offers the following features:

* Support for multiple ad formats, including video, display, and rewarded ads
* Ad event handling, including ad completion, ad clicks, and ad errors
* User rewarding system
* Support for both Android and iOS platforms

# Setting Up the Package
-------------------------

To use the `unity_mediation` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  unity_mediation: ^1.0.0
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:unity_mediation/unity_mediation.dart';
```
### Step 3: Initialize the Package

Initialize the package in your app's `main` function:
```dart
void main() {
  UnityMediation.initialize(
    gameId: 'YOUR_GAME_ID',
    adFormat: AdFormat.video,
  );
  runApp(MyApp());
}
```
### Step 4: Display Ads

Display ads using the `showAd` method:
```dart
UnityMediation.showAd(
  adId: 'YOUR_AD_ID',
  adFormat: AdFormat.video,
);
```
### Platform-Specific Details

#### Android

Add the following lines to your `AndroidManifest.xml` file:
```xml
<application>
  <meta-data
    android:name="unityads.gameid"
    android:value="YOUR_GAME_ID" />
</application>
```
#### iOS

Add the following lines to your `Info.plist` file:
```xml
<key>UnityAdsGameId</key>
<string>YOUR_GAME_ID</string>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:unity_mediation/unity_mediation.dart';

void main() {
  // Initialize the package
  UnityMediation.initialize(
    gameId: 'YOUR_GAME_ID',
    adFormat: AdFormat.video,
  );
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Unity Mediation Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  // Display ad
  void _showAd() {
    UnityMediation.showAd(
      adId: 'YOUR_AD_ID',
      adFormat: AdFormat.video,
    );
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Unity Mediation Demo'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: _showAd,
          child: Text('Show Ad'),
        ),
      ),
    );
  }
}

// Application flow:
// 1. The app initializes the Unity Mediation package with the game ID and ad format.
// 2. The app displays a button to show an ad.
// 3. When the button is pressed, the app calls the showAd method to display the ad.
// 4. The ad is displayed, and the user can interact with it.
// 5. When the ad is completed, the app can reward the user.
```
