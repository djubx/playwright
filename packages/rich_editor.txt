Rich Editor Flutter Package
==========================

The `rich_editor` Flutter package is a powerful text editor that allows users to create and edit rich text content. It provides a wide range of features, including text formatting, image insertion, and link creation.

When to Use
------------

The `rich_editor` package is ideal for applications that require a robust text editor, such as:

* Blogging apps
* Note-taking apps
* Content management systems
* Social media platforms

Features
--------

* Text formatting (bold, italic, underline, etc.)
* Image insertion
* Link creation
* Undo and redo functionality
* Customizable toolbar

Setup and Usage
---------------

To use the `rich_editor` package, follow these steps:

1. Add the package to your `pubspec.yaml` file:
```yml
dependencies:
  rich_editor: ^1.0.0
```
2. Import the package in your Dart file:
```dart
import 'package:rich_editor/rich_editor.dart';
```
3. Create a `RichEditor` widget:
```dart
RichEditor(
  controller: _controller,
  toolbar: RichToolbar(
    actions: [
      RichToolbarAction.bold,
      RichToolbarAction.italic,
      RichToolbarAction.undo,
      RichToolbarAction.redo,
    ],
  ),
)
```
4. Configure the toolbar:
```dart
RichToolbar(
  actions: [
    RichToolbarAction.bold,
    RichToolbarAction.italic,
    RichToolbarAction.undo,
    RichToolbarAction.redo,
  ],
  customActions: [
    IconButton(
      icon: Icon(Icons.image),
      onPressed: () {
        // Insert image logic here
      },
    ),
  ],
)
```
Platform-Specific Details
-------------------------

### Android

* Add the following line to your `AndroidManifest.xml` file:
```xml
<application
  ...
  android:hardwareAccelerated="true"
  ...
>
```
### iOS

* Add the following line to your `Info.plist` file:
```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

```dart
import 'package:flutter/material.dart';
import 'package:rich_editor/rich_editor.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  final _controller = RichTextController();

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Rich Editor Demo',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Rich Editor Demo'),
        ),
        body: Column(
          children: [
            RichEditor(
              controller: _controller,
              toolbar: RichToolbar(
                actions: [
                  RichToolbarAction.bold,
                  RichToolbarAction.italic,
                  RichToolbarAction.undo,
                  RichToolbarAction.redo,
                ],
                customActions: [
                  IconButton(
                    icon: Icon(Icons.image),
                    onPressed: () {
                      // Insert image logic here
                    },
                  ),
                ],
              ),
            ),
            Expanded(
              child: Container(
                padding: EdgeInsets.all(16),
                child: RichText(
                  text: _controller.text,
                ),
              ),
            ),
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user types in the RichEditor widget.
// 2. The user can format the text using the toolbar actions (bold, italic, etc.).
// 3. The user can insert an image using the custom image button.
// 4. The user can undo and redo changes using the toolbar actions.
// 5. The formatted text is displayed in the RichText widget below the editor.
```
