# gma_mediation_mintegral Flutter Package
=====================================

The `gma_mediation_mintegral` package is a Flutter plugin that provides a mediation solution for Mintegral ads using the Google Mobile Ads (GMA) SDK. This package allows developers to easily integrate Mintegral ads into their Flutter apps and manage ad mediation through the GMA platform.

## When to Use This Package
---------------------------

Use this package when you want to:

* Display Mintegral ads in your Flutter app
* Manage ad mediation through the Google Mobile Ads platform
* Increase ad revenue by using multiple ad networks

## Features
------------

* Supports banner, interstitial, and rewarded video ads
* Automatic ad mediation through the GMA platform
* Easy integration with Mintegral ad network

# Setup and Usage
================

## Step 1: Add the Package to Your Flutter Project
------------------------------------------------

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  gma_mediation_mintegral: ^1.0.0
```
Then, run `flutter pub get` to install the package.

## Step 2: Configure the GMA SDK
------------------------------

Create a new file called `gma_config.xml` in the `android/app/src/main/assets` directory with the following content:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<GoogleMobileAds>
    <Mediation>
        <Mintegral
            appId="YOUR_Mintegral_APP_ID"
            apiKey="YOUR_Mintegral_API_KEY"
        />
    </Mediation>
</GoogleMobileAds>
```
Replace `YOUR_Mintegral_APP_ID` and `YOUR_Mintegral_API_KEY` with your actual Mintegral app ID and API key.

## Step 3: Initialize the GMA SDK
-------------------------------

In your `main.dart` file, add the following code to initialize the GMA SDK:
```dart
import 'package:flutter/material.dart';
import 'package:gma_mediation_mintegral/gma_mediation_mintegral.dart';

void main() {
  WidgetsFlutterBinding.ensureInitialized();
  GmaMediationMintegral.initialize();
  runApp(MyApp());
}
```
## Step 4: Display Ads
------------------

To display ads, use the following code:
```dart
import 'package:flutter/material.dart';
import 'package:gma_mediation_mintegral/gma_mediation_mintegral.dart';

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('GMA Mediation Mintegral Demo'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              GmaMediationMintegralBannerAd(
                adUnitId: 'YOUR_AD_UNIT_ID',
                size: GmaMediationMintegralBannerSize.banner,
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () async {
                  await GmaMediationMintegralInterstitialAd.show(
                    adUnitId: 'YOUR_AD_UNIT_ID',
                  );
                },
                child: Text('Show Interstitial Ad'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () async {
                  await GmaMediationMintegralRewardedVideoAd.show(
                    adUnitId: 'YOUR_AD_UNIT_ID',
                  );
                },
                child: Text('Show Rewarded Video Ad'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}
```
Replace `YOUR_AD_UNIT_ID` with your actual ad unit ID.

```dart
import 'package:flutter/material.dart';
import 'package:gma_mediation_mintegral/gma_mediation_mintegral.dart';

void main() {
  // Initialize the GMA SDK
  WidgetsFlutterBinding.ensureInitialized();
  GmaMediationMintegral.initialize();

  // Run the app
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('GMA Mediation Mintegral Demo'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              // Display a banner ad
              GmaMediationMintegralBannerAd(
                adUnitId: 'YOUR_AD_UNIT_ID',
                size: GmaMediationMintegralBannerSize.banner,
              ),
              SizedBox(height: 20),

              // Display an interstitial ad
              ElevatedButton(
                onPressed: () async {
                  // Show the interstitial ad
                  await GmaMediationMintegralInterstitialAd.show(
                    adUnitId: 'YOUR_AD_UNIT_ID',
                  );
                },
                child: Text('Show Interstitial Ad'),
              ),
              SizedBox(height: 20),

              // Display a rewarded video ad
              ElevatedButton(
                onPressed: () async {
                  // Show the rewarded video ad
                  await GmaMediationMintegralRewardedVideoAd.show(
                    adUnitId: 'YOUR_AD_UNIT_ID',
                  );
                },
                child: Text('Show Rewarded Video Ad'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// The app flow is as follows:
// 1. The user opens the app and the GMA SDK is initialized.
// 2. The app displays a banner ad at the top of the screen.
// 3. The user can click on the "Show Interstitial Ad" button to display an interstitial ad.
// 4. The user can click on the "Show Rewarded Video Ad" button to display a rewarded video ad.
// 5. The app continues to run and display ads until the user closes it.
```
