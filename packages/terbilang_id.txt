# terbilang_id Flutter Package

The `terbilang_id` package is a Flutter library designed to convert numbers into their Indonesian word representation. This package is particularly useful in applications that require displaying numbers in a more human-readable format, such as financial applications, educational tools, or any app that needs to present numerical data in a user-friendly way.

## When to Use This Package
- **Financial Applications**: Displaying amounts in words for invoices or receipts.
- **Educational Apps**: Teaching children how to read numbers in Indonesian.
- **User Interfaces**: Enhancing the readability of numerical data in forms or reports.

## Features
- Converts integers and decimals into their Indonesian word equivalents.
- Supports large numbers and can handle various formats.
- Easy to integrate and use within Flutter applications.

## Tutorial: Setting Up and Using terbilang_id

### Installation
To use the `terbilang_id` package in your Flutter project, you need to add it to your `pubspec.yaml` file. Open the file and add the following line under dependencies:

```yaml
dependencies:
  terbilang_id: ^1.0.0  # Check for the latest version on pub.dev
```

After adding the dependency, run the following command in your terminal to install the package:

```bash
flutter pub get
```

Once the package is installed, you can start using it in your Flutter application. Below are the steps to implement it:

1. **Import the Package**: Import `terbilang_id` in your Dart file.
2. **Convert Numbers**: Use the `Terbilang` class to convert numbers to words.

The `terbilang_id` package is designed to work seamlessly on both Android and iOS platforms. There are no additional configurations required for either platform. Just ensure that your Flutter environment is set up correctly.

## Complete Example of terbilang_id in a Flutter App

```dart
import 'package:flutter/material.dart';
import 'package:terbilang_id/terbilang_id.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Terbilang ID Example',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Terbilang ID Example'),
        ),
        body: Center(
          child: NumberToWordsWidget(),
        ),
      ),
    );
  }
}

class NumberToWordsWidget extends StatefulWidget {
  @override
  _NumberToWordsWidgetState createState() => _NumberToWordsWidgetState();
}

class _NumberToWordsWidgetState extends State<NumberToWordsWidget> {
  final TextEditingController _controller = TextEditingController();
  String _result = '';

  void _convertNumber() {
    // Create an instance of the Terbilang class
    final terbilang = Terbilang();
    
    // Get the number from the text field and convert it
    int number = int.tryParse(_controller.text) ?? 0;
    _result = terbilang.convert(number);
    
    // Update the UI with the converted result
    setState(() {});
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      mainAxisAlignment: MainAxisAlignment.center,
      children: [
        TextField(
          controller: _controller,
          decoration: InputDecoration(
            labelText: 'Enter a number',
          ),
          keyboardType: TextInputType.number,
        ),
        SizedBox(height: 20),
        ElevatedButton(
          onPressed: _convertNumber,
          child: Text('Convert to Words'),
        ),
        SizedBox(height: 20),
        Text(
          _result,
          style: TextStyle(fontSize: 24),
        ),
      ],
    );
  }
}
