# Media Extension Flutter Package
=====================================

The `media_extension` Flutter package provides a simple way to handle media files in your Flutter application. It allows you to play, pause, and stop media files, as well as get the current playback position and duration.

## When to Use This Package
---------------------------

This package is useful when you need to play media files in your Flutter application, such as audio or video files. It provides a simple and easy-to-use API for handling media playback.

## Features
------------

* Play, pause, and stop media files
* Get the current playback position and duration
* Support for both audio and video files

# Setting Up the Package
-------------------------

To use the `media_extension` package, you need to add it to your `pubspec.yaml` file:

```yml
dependencies:
  media_extension: ^1.0.0
```

Then, run `flutter pub get` to install the package.

## Platform-Specific Details
---------------------------

### Android

On Android, you need to add the following permissions to your `AndroidManifest.xml` file:

```xml
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
```

### iOS

On iOS, you need to add the following lines to your `Info.plist` file:

```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

## Using the Package
---------------------

To use the package, you need to import it in your Dart file:

```dart
import 'package:media_extension/media_extension.dart';
```

Then, you can use the `MediaExtension` class to play, pause, and stop media files:

```dart
MediaExtension mediaExtension = MediaExtension();

mediaExtension.play('path/to/media/file.mp3');
mediaExtension.pause();
mediaExtension.stop();
```

```dart
import 'package:flutter/material.dart';
import 'package:media_extension/media_extension.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Media Extension Example',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  MediaExtension _mediaExtension;
  String _mediaPath;
  bool _isPlaying = false;

  @override
  void initState() {
    super.initState();
    _mediaExtension = MediaExtension();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Media Extension Example'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text(
              _isPlaying ? 'Playing' : 'Not playing',
              style: TextStyle(fontSize: 24),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () async {
                // Get the media path from the user
                _mediaPath = await _getMediaPath();

                // Play the media file
                _mediaExtension.play(_mediaPath);

                // Set the playing state to true
                setState(() {
                  _isPlaying = true;
                });
              },
              child: Text('Play'),
            ),
            SizedBox(height: 10),
            ElevatedButton(
              onPressed: () {
                // Pause the media file
                _mediaExtension.pause();

                // Set the playing state to false
                setState(() {
                  _isPlaying = false;
                });
              },
              child: Text('Pause'),
            ),
            SizedBox(height: 10),
            ElevatedButton(
              onPressed: () {
                // Stop the media file
                _mediaExtension.stop();

                // Set the playing state to false
                setState(() {
                  _isPlaying = false;
                });
              },
              child: Text('Stop'),
            ),
          ],
        ),
      ),
    );
  }

  Future<String> _getMediaPath() async {
    // Get the media path from the user
    // For example, using a file picker
    return 'path/to/media/file.mp3';
  }
}

// The application flow is as follows:
// 1. The user opens the app and is presented with a play button.
// 2. When the user clicks the play button, the app gets the media path from the user.
// 3. The app then plays the media file using the MediaExtension class.
// 4. The app sets the playing state to true and updates the UI to show that the media file is playing.
// 5. The user can then pause or stop the media file by clicking the corresponding buttons.
// 6. When the user pauses or stops the media file, the app sets the playing state to false and updates the UI to show that the media file is not playing.

// In summary, the app uses the MediaExtension class to play, pause, and stop media files, and updates the UI to show the current playing state.
```
