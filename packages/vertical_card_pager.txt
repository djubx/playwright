Vertical Card Pager
====================
The `vertical_card_pager` package is a Flutter package that allows you to create a vertical card pager, where cards are displayed in a vertical list and can be paged through. This package is useful when you want to display a list of items in a card format and allow users to navigate through them.

Features
--------

*   Vertical card pager with page indicator
*   Customizable card size and spacing
*   Support for infinite scrolling

When to Use
------------

*   When you want to display a list of items in a card format
*   When you want to allow users to navigate through a list of items
*   When you want to create a visually appealing and interactive list

<-- END_DESCRIPTION -->

Setup and Usage
----------------

### Step 1: Add the Package to Your Project

Add the `vertical_card_pager` package to your `pubspec.yaml` file:

```yml
dependencies:
  flutter:
    sdk: flutter
  vertical_card_pager: ^1.0.0
```

Then, run `flutter pub get` to get the package.

### Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:vertical_card_pager/vertical_card_pager.dart';
```

### Step 3: Create a Vertical Card Pager

Create a `VerticalCardPager` widget and pass in a list of widgets to display:

```dart
VerticalCardPager(
  children: [
    // Your widgets here
  ],
)
```

### Step 4: Customize the Card Size and Spacing

You can customize the card size and spacing by passing in a `CardSize` and `CardSpacing` object:

```dart
VerticalCardPager(
  children: [
    // Your widgets here
  ],
  cardSize: CardSize(
    height: 200,
    width: 300,
  ),
  cardSpacing: CardSpacing(
    top: 10,
    bottom: 10,
  ),
)
```

### Step 5: Add a Page Indicator

You can add a page indicator by passing in a `PageIndicator` object:

```dart
VerticalCardPager(
  children: [
    // Your widgets here
  ],
  pageIndicator: PageIndicator(
    activeColor: Colors.blue,
    inactiveColor: Colors.grey,
  ),
)
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:vertical_card_pager/vertical_card_pager.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Vertical Card Pager Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Vertical Card Pager Demo'),
      ),
      body: Center(
        child: VerticalCardPager(
          children: [
            // Card 1
            Container(
              decoration: BoxDecoration(
                color: Colors.red,
                borderRadius: BorderRadius.circular(10),
              ),
              child: Center(
                child: Text(
                  'Card 1',
                  style: TextStyle(fontSize: 24),
                ),
              ),
            ),
            // Card 2
            Container(
              decoration: BoxDecoration(
                color: Colors.green,
                borderRadius: BorderRadius.circular(10),
              ),
              child: Center(
                child: Text(
                  'Card 2',
                  style: TextStyle(fontSize: 24),
                ),
              ),
            ),
            // Card 3
            Container(
              decoration: BoxDecoration(
                color: Colors.blue,
                borderRadius: BorderRadius.circular(10),
              ),
              child: Center(
                child: Text(
                  'Card 3',
                  style: TextStyle(fontSize: 24),
                ),
              ),
            ),
          ],
          cardSize: CardSize(
            height: 200,
            width: 300,
          ),
          cardSpacing: CardSpacing(
            top: 10,
            bottom: 10,
          ),
          pageIndicator: PageIndicator(
            activeColor: Colors.blue,
            inactiveColor: Colors.grey,
          ),
        ),
      ),
    );
  }
}

// The app starts by running the main function, which runs the MyApp widget.
// The MyApp widget builds a MaterialApp with a title and theme.
// The home of the MaterialApp is the MyHomePage widget.
// The MyHomePage widget builds a Scaffold with an AppBar and a body.
// The body of the Scaffold is a Center widget with a VerticalCardPager widget.
// The VerticalCardPager widget displays a list of cards, which are containers with a decoration and a center child.
// The cards are spaced evenly apart and have a page indicator at the bottom.
// The user can navigate through the cards by swiping up or down.
```
