# Mindful Minutes Flutter Package

The **mindful_minutes** Flutter package is designed to help developers integrate mindfulness and meditation features into their applications. This package provides a simple and effective way to encourage users to take a moment for themselves, promoting mental well-being through guided sessions and timers.

## When to Use This Package
- **Wellness Apps**: If you're building an app focused on mental health, meditation, or wellness, this package can enhance user experience by providing mindfulness features.
- **Productivity Tools**: Integrate mindful breaks into productivity applications to help users manage stress and improve focus.
- **Educational Apps**: Use it in educational settings to promote mindfulness among students.

## Features
- **Guided Meditations**: Access a library of guided meditation sessions.
- **Custom Timers**: Set timers for meditation sessions with customizable durations.
- **User Progress Tracking**: Track user engagement and progress over time.
- **Notifications**: Remind users to take mindful breaks with scheduled notifications.

The **mindful_minutes** package is a valuable tool for developers looking to promote mindfulness and well-being in their applications.

# Tutorial: Setting Up and Using Mindful Minutes

## Installation
To add the **mindful_minutes** package to your Flutter project, include it in your `pubspec.yaml` file:

```yaml
dependencies:
  mindful_minutes: ^1.0.0
```

Run `flutter pub get` to install the package.

## Platform-Specific Configuration

### Android
1. Open `android/app/build.gradle`.
2. Ensure that the `minSdkVersion` is set to at least 21:

```groovy
android {
    ...
    defaultConfig {
        ...
        minSdkVersion 21
    }
}
```

### iOS
1. Open `ios/Runner/Info.plist`.
2. Add the following permissions to allow notifications:

```xml
<key>UIBackgroundModes</key>
<array>
    <string>fetch</string>
    <string>remote-notification</string>
</array>
```

## Using the Package
To use the **mindful_minutes** package, you can create a simple interface that allows users to select a meditation session or set a timer. Below is a basic example of how to implement this in your Flutter app.

```dart
import 'package:flutter/material.dart';
import 'package:mindful_minutes/mindful_minutes.dart';

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Mindful Minutes',
      home: MindfulHome(),
    );
  }
}

class MindfulHome extends StatefulWidget {
  @override
  _MindfulHomeState createState() => _MindfulHomeState();
}

class _MindfulHomeState extends State<MindfulHome> {
  // Initialize the MindfulMinutes instance
  final MindfulMinutes _mindfulMinutes = MindfulMinutes();

  @override
  void initState() {
    super.initState();
    // Request permissions for notifications
    _mindfulMinutes.requestPermissions();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Mindful Minutes'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            ElevatedButton(
              onPressed: () {
                // Start a guided meditation session
                _mindfulMinutes.startGuidedMeditation();
              },
              child: Text('Start Guided Meditation'),
            ),
            ElevatedButton(
              onPressed: () {
                // Set a timer for 5 minutes
                _mindfulMinutes.setTimer(Duration(minutes: 5));
              },
              child: Text('Set 5 Minute Timer'),
            ),
          ],
        ),
      ),
    );
  }
}
```
