In summary, the code flows as follows:

1.  The user selects a country from the `CountrySelector` widget.
2.  The `onCountrySelected` event is triggered, and the selected country is stored in the `_selectedCountry` variable.
3.  The selected country is displayed in the bottom navigation bar.
4.  The user can select another country, and the process repeats.
