# Share It Flutter Package
==========================

The `share_it` Flutter package is a powerful tool for sharing content from your Flutter app to other apps on the user's device. It provides a simple and easy-to-use API for sharing text, images, and other types of content.

## When to Use This Package
---------------------------

You can use this package in a variety of scenarios, such as:

* Sharing a link to a webpage or a social media post
* Sharing an image or a video from your app's gallery
* Sharing a piece of text, such as a quote or a message

## Features
------------

The `share_it` package provides the following features:

* Support for sharing text, images, and other types of content
* Support for sharing to multiple apps at once
* Support for customizing the sharing dialog

# Setting Up the Package
-------------------------

To use the `share_it` package, you need to add it to your Flutter project's `pubspec.yaml` file. Here's how to do it:

1. Open your `pubspec.yaml` file and add the following line to the `dependencies` section:
```yml
dependencies:
  flutter:
    sdk: flutter
  share_it: ^1.0.0
```
2. Run `flutter pub get` in your terminal to get the package.

## Platform-Specific Details
---------------------------

### Android

On Android, you need to add the following lines to your `AndroidManifest.xml` file:
```xml
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
```
### iOS

On iOS, you need to add the following lines to your `Info.plist` file:
```xml
<key>NSPhotoLibraryUsageDescription</key>
<string>This app needs access to your photo library to share images.</string>
```
## Using the Package
---------------------

To use the package, you need to import it in your Dart file:
```dart
import 'package:share_it/share_it.dart';
```
Then, you can use the `ShareIt` class to share content:
```dart
ShareIt.shareText('Hello, world!');
```

```dart
import 'package:flutter/material.dart';
import 'package:share_it/share_it.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Share It Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  // Step 1: Create a text field to input the text to share
  final _textController = TextEditingController();

  // Step 2: Create a button to share the text
  void _shareText() {
    // Step 3: Get the text from the text field
    final text = _textController.text;

    // Step 4: Share the text using the ShareIt class
    ShareIt.shareText(text);
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Share It Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            // Step 5: Display the text field
            TextField(
              controller: _textController,
              decoration: InputDecoration(
                border: OutlineInputBorder(),
                labelText: 'Enter text to share',
              ),
            ),
            // Step 6: Display the share button
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _shareText,
              child: Text('Share'),
            ),
          ],
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user inputs the text to share in the text field.
// 2. The user clicks the share button.
// 3. The _shareText function is called, which gets the text from the text field.
// 4. The ShareIt class is used to share the text.
// 5. The sharing dialog is displayed, allowing the user to choose an app to share to.
// 6. The text is shared to the chosen app.

// Final summary: This code demonstrates how to use the ShareIt package to share text from a Flutter app.
// The user inputs the text to share, clicks the share button, and the text is shared to a chosen app.
```
