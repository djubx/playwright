# Calendar Slider Package for Flutter
=====================================

The `calendar_slider` package is a powerful and customizable Flutter widget that allows users to select dates from a calendar. It provides a simple and intuitive way to navigate through dates, making it an ideal solution for applications that require date selection.

### When to Use

The `calendar_slider` package is perfect for applications that require users to select dates, such as:

* Scheduling appointments
* Booking flights or hotels
* Selecting dates for events or meetings
* Filtering data by date

### Features

* Customizable calendar layout
* Support for multiple date selection
* Ability to disable specific dates
* Customizable date formatting
* Support for internationalization

# Setting Up the Calendar Slider Package
=====================================

To use the `calendar_slider` package, follow these steps:

### Step 1: Add the Package to Your Pubspec File

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  calendar_slider: ^1.0.0
```
### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:calendar_slider/calendar_slider.dart';
```
### Step 3: Create a Calendar Slider Widget

Create a `CalendarSlider` widget and customize it as needed:
```dart
CalendarSlider(
  // Customize the calendar layout
  layout: CalendarLayout.monthly,
  // Support for multiple date selection
  multipleSelection: true,
  // Disable specific dates
  disabledDates: [DateTime(2022, 1, 1), DateTime(2022, 1, 15)],
  // Customize date formatting
  dateFormat: 'yyyy-MM-dd',
  // Support for internationalization
  locale: 'en_US',
)
```
### Platform-Specific Details

* For Android, make sure to add the following line to your `AndroidManifest.xml` file:
```xml
<uses-feature android:name="android.hardware.touchscreen" android:required="false" />
```
* For iOS, make sure to add the following line to your `Info.plist` file:
```xml
<key>NSCalendarsUsageDescription</key>
<string>This app needs access to your calendar to function properly</string>
```
