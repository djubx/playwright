# Fluent UI Emoji Icon Package for Flutter
=============================================

The `fluentui_emoji_icon` package is a Flutter library that provides a set of emoji icons from Microsoft's Fluent UI design system. It allows developers to easily integrate these icons into their Flutter applications, enhancing the user experience with visually appealing and consistent iconography.

## When to Use This Package
---------------------------

You can use this package in various scenarios, such as:

* Creating a mobile app that requires a modern and consistent icon set
* Developing a desktop application that needs to align with Microsoft's Fluent UI design system
* Building a web application that wants to incorporate Fluent UI icons

## Features
------------

The `fluentui_emoji_icon` package offers the following features:

* A wide range of emoji icons from Microsoft's Fluent UI design system
* Easy integration into Flutter applications
* Consistent icon design across different platforms

# Setting Up the Package
-------------------------

To use the `fluentui_emoji_icon` package in your Flutter project, follow these steps:

### Step 1: Add the Package to Your `pubspec.yaml` File

```yml
dependencies:
  flutter:
    sdk: flutter
  fluentui_emoji_icon: ^1.0.0
```

### Step 2: Run `flutter pub get` to Get the Package

Open your terminal and run the following command:

```bash
flutter pub get
```

### Step 3: Import the Package in Your Dart File

```dart
import 'package:flutter/material.dart';
import 'package:fluentui_emoji_icon/fluentui_emoji_icon.dart';
```

### Platform-Specific Details

* For Android, make sure to add the following line to your `android/app/build.gradle` file:

```groovy
apply plugin: 'com.android.application'
```

* For iOS, add the following line to your `ios/Podfile` file:

```ruby
pod 'FluentUIEmojiIcon'
```

### Required Configurations or Optimizations

* Make sure to use the correct version of the package that matches your Flutter version.
* If you encounter any issues, try running `flutter clean` and then `flutter pub get` again.

```dart
import 'package:flutter/material.dart';
import 'package:fluentui_emoji_icon/fluentui_emoji_icon.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Fluent UI Emoji Icon Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Fluent UI Emoji Icon Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            // Use the FluentUIEmojiIcon widget to display an emoji icon
            FluentUIEmojiIcon(
              emoji: 'üòä',
              size: 48,
            ),
            SizedBox(height: 20),
            // Use the FluentUIEmojiIcon widget with a different emoji and size
            FluentUIEmojiIcon(
              emoji: 'üëç',
              size: 64,
            ),
          ],
        ),
      ),
    );
  }
}
```

// The UI consists of a Scaffold with an AppBar and a Center widget.
