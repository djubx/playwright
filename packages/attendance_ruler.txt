# Attendance Ruler Flutter Package
=====================================

The `attendance_ruler` Flutter package is a powerful tool for tracking and managing attendance in various settings, such as schools, universities, and workplaces. It provides a simple and intuitive interface for users to mark their attendance, view attendance records, and receive notifications.

## Key Features

*   Mark attendance with a single tap
*   View attendance records by date, month, or year
*   Receive notifications for attendance reminders
*   Customizable attendance rules and settings
*   Support for multiple user roles (e.g., student, teacher, admin)

## When to Use This Package

This package is ideal for any Flutter application that requires attendance tracking and management. It can be used in various scenarios, such as:

*   Educational institutions: schools, universities, colleges
*   Workplaces: offices, factories, warehouses
*   Events: conferences, meetups, workshops

# Tutorial: Setting Up and Using the Attendance Ruler Package
===========================================================

## Step 1: Add the Package to Your Flutter Project

Add the following dependency to your `pubspec.yaml` file:

```yml
dependencies:
  attendance_ruler: ^1.0.0
```

Then, run `flutter pub get` to install the package.

## Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:attendance_ruler/attendance_ruler.dart';
```

## Step 3: Initialize the Package

Initialize the package with your app's settings:

```dart
void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Attendance Ruler Demo',
      home: AttendanceRuler(
        // Your app's settings
        appName: 'My App',
        attendanceRules: [
          AttendanceRule(
            id: 1,
            name: 'Daily Attendance',
            frequency: AttendanceFrequency.DAILY,
          ),
        ],
      ),
    );
  }
}
```

## Platform-Specific Details

### Android

*   Add the following permissions to your `AndroidManifest.xml` file:

    ```xml
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />
```

### iOS

*   Add the following permissions to your `Info.plist` file:

    ```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

```dart
import 'package:flutter/material.dart';
import 'package:attendance_ruler/attendance_ruler.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // Create a Material app with a title and a home page
    return MaterialApp(
      title: 'Attendance Ruler Demo',
      home: AttendanceRuler(
        // Set the app name
        appName: 'RealFlutter',
        
        // Define attendance rules
        attendanceRules: [
          AttendanceRule(
            id: 1,
            name: 'Daily Attendance',
            frequency: AttendanceFrequency.DAILY,
          ),
          AttendanceRule(
            id: 2,
            name: 'Weekly Attendance',
            frequency: AttendanceFrequency.WEEKLY,
          ),
        ],
        
        // Set the initial route
        initialRoute: '/',
        
        // Define routes
        routes: {
          '/': (context) => MarkAttendancePage(),
          '/records': (context) => AttendanceRecordsPage(),
          '/settings': (context) => AttendanceSettingsPage(),
        },
      ),
    );
  }
}

// Page to mark attendance
class MarkAttendancePage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // Create a Scaffold with a body
    return Scaffold(
      appBar: AppBar(
        title: Text('Mark Attendance'),
      ),
      body: Center(
        child: ElevatedButton(
          onPressed: () {
            // Mark attendance when the button is pressed
            AttendanceRuler.of(context).markAttendance();
          },
          child: Text('Mark Attendance'),
        ),
      ),
    );
  }
}

// Page to view attendance records
class AttendanceRecordsPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // Create a Scaffold with a body
    return Scaffold(
      appBar: AppBar(
        title: Text('Attendance Records'),
      ),
      body: Center(
        child: Text('View attendance records here'),
      ),
    );
  }
}

// Page to view attendance settings
class AttendanceSettingsPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // Create a Scaffold with a body
    return Scaffold(
      appBar: AppBar(
        title: Text('Attendance Settings'),
      ),
      body: Center(
        child: Text('View attendance settings here'),
      ),
    );
  }
}

// The app starts by running the main function, which creates a RealFlutter app.
// The RealFlutter app creates a Material app with a title and a home page.
// The home page is an AttendanceRuler, which is a package that provides attendance tracking and management.
// The AttendanceRuler has an initial route and defines several routes for marking attendance, viewing attendance records, and viewing attendance settings.
// When the user navigates to a route, the corresponding page is displayed.
// The pages use Scaffolds with bodies to display content.
// The MarkAttendancePage has a button that marks attendance when pressed.
```
