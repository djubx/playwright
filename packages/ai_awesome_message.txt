# ai_awesome_message Flutter Package
=====================================

The `ai_awesome_message` Flutter package is a powerful tool for displaying customizable, animated messages in your Flutter applications. With its easy-to-use API and flexible configuration options, this package is perfect for adding a touch of personality to your app's user interface.

### When to Use This Package

* Displaying welcome messages or tutorials to new users
* Showing alerts or notifications with a custom design
* Creating interactive, animated messages for user engagement
* Adding a touch of personality to your app's UI with customizable messages

### Key Features

* Customizable message text, color, and font
* Animated message display with various animation options
* Support for multiple message types (e.g., success, error, warning)
* Platform-specific configurations for Android and iOS

# Setting Up and Using the ai_awesome_message Package
---------------------------------------------------

### Step 1: Add the Package to Your pubspec.yaml File

```yml
dependencies:
  ai_awesome_message: ^1.0.0
```

### Step 2: Import the Package in Your Dart File

```dart
import 'package:ai_awesome_message/ai_awesome_message.dart';
```

### Step 3: Display a Message

```dart
AiAwesomeMessage(
  message: 'Hello, World!',
  messageType: MessageType.success,
  animationType: AnimationType.fadeIn,
);
```

### Platform-Specific Configurations

#### Android

* Add the following code to your `AndroidManifest.xml` file:
```xml
<application
  ...
  android:hardwareAccelerated="true"
  ...
>
```

#### iOS

* Add the following code to your `Info.plist` file:
```xml
<key>UIAppFonts</key>
<array>
  <string>OpenSans-Regular.ttf</string>
</array>
```

### Optimizations

* Use the `AiAwesomeMessage` widget as a child of a `Stack` widget to display the message on top of other content.
* Use the `animationType` property to customize the animation of the message.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:ai_awesome_message/ai_awesome_message.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Ai Awesome Message Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Ai Awesome Message Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            AiAwesomeMessage(
              message: 'Hello, World!',
              messageType: MessageType.success,
              animationType: AnimationType.fadeIn,
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                AiAwesomeMessage(
                  message: 'Button clicked!',
                  messageType: MessageType.info,
                  animationType: AnimationType.slideIn,
                );
              },
              child: Text('Click me!'),
            ),
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. The app starts and displays the home page with a button.
// 2. When the button is clicked, a new message is displayed with a slide-in animation.
// 3. The message is displayed on top of the other content using a Stack widget.
// 4. The message is customizable, with options for text, color, font, and animation.
// 5. The app uses platform-specific configurations for Android and iOS.
// 6. The app uses optimizations such as hardware acceleration and custom fonts.

```
