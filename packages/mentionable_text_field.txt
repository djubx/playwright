# Introduction to mentionable_text_field Flutter Package
=====================================================

The `mentionable_text_field` package is a powerful Flutter library that allows you to create text fields with mentionable functionality, similar to those found in social media platforms. With this package, you can easily create text fields that support mentioning users, hashtags, or any other type of entity.

### When to Use This Package

This package is perfect for social media apps, messaging apps, or any other type of app that requires mentionable text fields. It's also a great solution for apps that need to support hashtags or other types of entities.

### Features

*   Support for mentioning users, hashtags, or any other type of entity
*   Customizable mention styles and colors
*   Support for multiple mention types
*   Easy integration with existing text fields

# Setting Up mentionable_text_field Package
=============================================

To use the `mentionable_text_field` package, you'll need to add it to your `pubspec.yaml` file:

```yml
dependencies:
  flutter:
    sdk: flutter
  mentionable_text_field: ^1.0.0
```

Then, run `flutter pub get` to install the package.

### Platform-Specific Details

This package works on both Android and iOS platforms. However, you may need to configure your app's keyboard settings to support mentionable text fields.

### Required Configurations or Optimizations

To optimize the performance of the mentionable text field, you can adjust the `debounceDuration` property to control how often the mention suggestions are updated.

```dart
import 'package:flutter/material.dart';
import 'package:mentionable_text_field/mentionable_text_field.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Mentionable Text Field Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _controller = TextEditingController();

  // List of mentionable users
  final _users = [
    {'id': 1, 'name': 'John Doe'},
    {'id': 2, 'name': 'Jane Doe'},
    {'id': 3, 'name': 'Bob Smith'},
  ];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Mentionable Text Field Demo'),
      ),
      body: Padding(
        padding: const EdgeInsets.all(16.0),
        child: MentionableTextField(
          // Controller for the text field
          controller: _controller,

          // List of mentionable users
          mentions: _users.map((user) => Mention(
            id: user['id'],
            name: user['name'],
          )).toList(),

          // Style for the mention text
          mentionStyle: TextStyle(
            color: Colors.blue,
            fontWeight: FontWeight.bold,
          ),

          // Callback for when a mention is selected
          onMentionSelected: (mention) {
            print('Mention selected: ${mention.name}');
          },
        ),
      ),
    );
  }
}
```

// When a mention is selected, the onMentionSelected callback is called.
// The mention style is customized to be blue and bold.
// The app runs on both Android and iOS platforms.
