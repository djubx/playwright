Master Detail Scaffold Flutter Package
=====================================

The `master_detail_scaffold` package is a Flutter package that provides a scaffold for creating master-detail layouts. It is useful for creating apps that require a list of items and a detailed view of each item.

When to Use This Package
------------------------

This package is ideal for apps that require a list of items and a detailed view of each item. For example:

*   A news app that lists headlines and provides a detailed view of each article
*   A music app that lists songs and provides a detailed view of each song
*   A product catalog app that lists products and provides a detailed view of each product

Features
--------

*   Master-detail layout with a list of items and a detailed view of each item
*   Customizable layout and design
*   Support for multiple platforms (Android, iOS, etc.)

Setup and Usage
---------------

To use the `master_detail_scaffold` package, follow these steps:

1.  Add the package to your `pubspec.yaml` file:

    ```yml
dependencies:
  master_detail_scaffold: ^1.0.0
```

2.  Run `flutter pub get` to get the package.

3.  Import the package in your Dart file:

    ```dart
import 'package:master_detail_scaffold/master_detail_scaffold.dart';
```

4.  Create a `MasterDetailScaffold` widget and pass in the required properties:

    ```dart
MasterDetailScaffold(
  master: MasterWidget(),
  detail: DetailWidget(),
)
```

Platform-Specific Details
-------------------------

### Android

*   To use the `master_detail_scaffold` package on Android, you need to add the following code to your `AndroidManifest.xml` file:

    ```xml
<application
  ...
  android:usesCleartextTraffic="true"
  ...
>
```

### iOS

*   To use the `master_detail_scaffold` package on iOS, you need to add the following code to your `Info.plist` file:

    ```xml
<key>NSAppTransportSecurity</key>
<dict>
  <key>NSAllowsArbitraryLoads</key>
  <true/>
</dict>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:master_detail_scaffold/master_detail_scaffold.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Master Detail Scaffold Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return MasterDetailScaffold(
      master: MasterWidget(),
      detail: DetailWidget(),
    );
  }
}

class MasterWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // Create a list of items
    List<String> items = [
      'Item 1',
      'Item 2',
      'Item 3',
      'Item 4',
      'Item 5',
    ];

    return Scaffold(
      appBar: AppBar(
        title: Text('Master'),
      ),
      body: ListView.builder(
        itemCount: items.length,
        itemBuilder: (context, index) {
          return ListTile(
            title: Text(items[index]),
            onTap: () {
              // Navigate to the detail page
              Navigator.push(
                context,
                MaterialPageRoute(builder: (context) => DetailPage()),
              );
            },
          );
        },
      ),
    );
  }
}

class DetailWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Detail'),
      ),
      body: Center(
        child: Text('Detail page'),
      ),
    );
  }
}

class DetailPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Detail Page'),
      ),
      body: Center(
        child: Text('Detail page content'),
      ),
    );
  }
}
```

// The app starts by running the `MyApp` widget.
// The `MyApp` widget builds a `MaterialApp` with a `MyHomePage` widget as its home page.
// The `MyHomePage` widget builds a `MasterDetailScaffold` with a `MasterWidget` and a `DetailWidget`.
// The `MasterWidget` builds a list of items and navigates to the `DetailPage` when an item is tapped.
// The `DetailWidget` builds a detail page with a title and content.
// The `DetailPage` builds a detail page with a title and content.

// The app flows as follows:
// 1. The user runs the app and sees the `MasterWidget` with a list of items.
// 2. The user taps an item and navigates to the `DetailPage`.
// 3. The user sees the `DetailPage` with a title and content.
