# ikchatbot Flutter Package
==========================

The ikchatbot Flutter package is a powerful tool for integrating chatbot functionality into your Flutter applications. It provides a simple and intuitive API for creating and managing chatbot conversations, making it easy to add conversational interfaces to your app.

### When to Use

The ikchatbot package is perfect for apps that require user interaction, such as:

* Customer support chatbots
* Virtual assistants
* Language learning apps
* E-commerce chatbots

### Features

* Easy-to-use API for creating and managing chatbot conversations
* Support for multiple chatbot platforms (e.g. Dialogflow, Rasa)
* Customizable chatbot UI
* Integration with popular messaging platforms (e.g. Facebook Messenger, WhatsApp)

# Setting Up ikchatbot
---------------------

To use the ikchatbot package, follow these steps:

### Step 1: Add the Package to Your Pubspec

Add the following line to your `pubspec.yaml` file:
```yml
dependencies:
  ikchatbot: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:ikchatbot/ikchatbot.dart';
```
### Step 3: Initialize the Chatbot

 Initialize the chatbot with your platform-specific credentials:
```dart
void main() {
  final chatbot = IkChatbot(
    platform: IkChatbotPlatform.dialogflow,
    credentials: {
      'projectId': 'your-project-id',
      'apiKey': 'your-api-key',
    },
  );
  runApp(MyApp(chatbot: chatbot));
}
```
### Step 4: Create a Chatbot UI

Create a chatbot UI using the `IkChatbotWidget`:
```dart
class MyApp extends StatelessWidget {
  final IkChatbot chatbot;

  MyApp({this.chatbot});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Chatbot Demo',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Chatbot Demo'),
        ),
        body: IkChatbotWidget(chatbot: chatbot),
      ),
    );
  }
}
```

```dart
import 'package:flutter/material.dart';
import 'package:ikchatbot/ikchatbot.dart';

void main() {
  final chatbot = IkChatbot(
    platform: IkChatbotPlatform.dialogflow,
    credentials: {
      'projectId': 'your-project-id',
      'apiKey': 'your-api-key',
    },
  );
  runApp(RealFlutter(chatbot: chatbot));
}

class RealFlutter extends StatelessWidget {
  final IkChatbot chatbot;

  RealFlutter({this.chatbot});

  @override
  Widget build(BuildContext context) {
    // Create a Material app with a Scaffold
    return MaterialApp(
      title: 'Chatbot Demo',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Chatbot Demo'),
        ),
        // Use the IkChatbotWidget to display the chatbot UI
        body: IkChatbotWidget(
          chatbot: chatbot,
          // Customize the chatbot UI with a custom theme
          theme: IkChatbotTheme(
            backgroundColor: Colors.blue,
            textColor: Colors.white,
          ),
        ),
      ),
    );
  }
}

// The app flow is as follows:
// 1. The user opens the app and is presented with a chatbot UI.
// 2. The user types a message and sends it to the chatbot.
// 3. The chatbot receives the message and responds with a reply.
// 4. The chatbot's response is displayed in the chatbot UI.
// 5. The user can continue to chat with the chatbot by sending more messages.
```

// In summary, this code creates a simple chatbot app using the ikchatbot package. The app displays a chatbot UI and allows the user to chat with the chatbot. The chatbot's responses are displayed in the chatbot UI, creating a conversational interface.
