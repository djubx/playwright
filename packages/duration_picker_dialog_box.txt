# Multi Masked Formatter Flutter Package
=====================================

The `multi_masked_formatter` Flutter package provides a convenient way to format text fields with multiple masks. It's a powerful tool for creating complex input fields, such as phone numbers, credit card numbers, and more.

### When to Use This Package

Use this package when you need to create a text field that requires multiple masks, such as:

* Phone numbers with country codes and extensions
* Credit card numbers with expiration dates and security codes
* Dates and times with custom formats

### Features

* Supports multiple masks for a single text field
* Customizable mask formats
* Easy integration with existing Flutter text fields

# Setting Up the Package
-------------------------

To use the `multi_masked_formatter` package, follow these steps:

### Step 1: Add the Package to Your Pubspec.yaml File

```yml
dependencies:
  flutter:
    sdk: flutter
  multi_masked_formatter: ^1.0.0
```

### Step 2: Import the Package in Your Dart File

```dart
import 'package:multi_masked_formatter/multi_masked_formatter.dart';
```

### Step 3: Create a Text Field with a MultiMaskedFormatter

```dart
TextField(
  inputFormatters: [
    MultiMaskedFormatter(
      masks: [
        '000-000-0000', // phone number
        '0000-0000-0000-0000', // credit card number
      ],
    ),
  ],
)
```

### Platform-Specific Details

* Android: No additional configuration required.
* iOS: No additional configuration required.

### Optimizations

* Use the `maxLength` property to limit the input length.
* Use the `keyboardType` property to specify the keyboard type.

```dart
import 'package:flutter/material.dart';
import 'package:multi_masked_formatter/multi_masked_formatter.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Multi Masked Formatter Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  final _formKey = GlobalKey<FormState>();
  final _controller = TextEditingController();

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Multi Masked Formatter Demo'),
      ),
      body: Padding(
        padding: const EdgeInsets.all(20.0),
        child: Form(
          key: _formKey,
          child: Column(
            children: [
              // Phone number text field
              TextField(
                controller: _controller,
                inputFormatters: [
                  MultiMaskedFormatter(
                    masks: [
                      '000-000-0000', // phone number
                    ],
                  ),
                ],
                decoration: InputDecoration(
                  labelText: 'Phone Number',
                ),
              ),
              SizedBox(height: 20),
              // Credit card number text field
              TextField(
                inputFormatters: [
                  MultiMaskedFormatter(
                    masks: [
                      '0000-0000-0000-0000', // credit card number
                    ],
                  ),
                ],
                decoration: InputDecoration(
                  labelText: 'Credit Card Number',
                ),
              ),
              SizedBox(height: 20),
              // Submit button
              ElevatedButton(
                onPressed: () {
                  if (_formKey.currentState!.validate()) {
                    // Submit the form
                    print('Form submitted');
                  }
                },
                child: Text('Submit'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// Application flow:
// 1. The user opens the app and is presented with a form containing two text fields: phone number and credit card number.
// 2. The user enters their phone number in the first text field, which is formatted using the MultiMaskedFormatter.
// 3. The user enters their credit card number in the second text field, which is also formatted using the MultiMaskedFormatter.
// 4. When the user submits the form, the app validates the input data and prints a success message to the console.

// Summary:
// This code demonstrates the use of the MultiMaskedFormatter package to format text fields in a Flutter app.
// The app presents a form with two text fields: phone number and credit card number, both of which are formatted using the MultiMaskedFormatter.
// When the user submits the form, the app validates the input data and prints a success message to the console.
```
