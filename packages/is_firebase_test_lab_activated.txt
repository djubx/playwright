# is_firebase_test_lab_activated Flutter Package
==============================================

The `is_firebase_test_lab_activated` package is a Flutter plugin that allows you to detect whether your app is running on Firebase Test Lab or not. This package is useful when you want to run certain code or tests only on Firebase Test Lab.

## Features

*   Detects whether the app is running on Firebase Test Lab or not.
*   Supports both Android and iOS platforms.

## When to Use

You can use this package in the following scenarios:

*   To run certain tests or code only on Firebase Test Lab.
*   To detect whether the app is running on a physical device or on Firebase Test Lab.
*   To optimize your app's behavior based on whether it's running on Firebase Test Lab or not.

# Setup and Usage
-----------------

To use the `is_firebase_test_lab_activated` package, follow these steps:

### Step 1: Add the Package to Your pubspec.yaml File

Add the following line to your `pubspec.yaml` file:

```yml
dependencies:
  is_firebase_test_lab_activated: ^1.0.0
```

### Step 2: Import the Package

Import the package in your Dart file:

```dart
import 'package:is_firebase_test_lab_activated/is_firebase_test_lab_activated.dart';
```

### Step 3: Use the Package

Use the `isFirebaseTestLabActivated` function to detect whether the app is running on Firebase Test Lab or not:

```dart
bool isTestLabActivated = await isFirebaseTestLabActivated();
if (isTestLabActivated) {
  // Run code or tests specific to Firebase Test Lab
} else {
  // Run code or tests specific to physical devices
}
```

### Platform-Specific Details

*   **Android:** The package uses the `android.os.Build` class to detect whether the app is running on Firebase Test Lab or not.
*   **iOS:** The package uses the `UIDevice` class to detect whether the app is running on Firebase Test Lab or not.

### Required Configurations or Optimizations

No additional configurations or optimizations are required to use this package.

```dart
import 'package:flutter/material.dart';
import 'package:is_firebase_test_lab_activated/is_firebase_test_lab_activated.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Firebase Test Lab Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  bool _isTestLabActivated = false;

  @override
  void initState() {
    super.initState();
    _checkTestLabActivation();
  }

  Future<void> _checkTestLabActivation() async {
    // Check if the app is running on Firebase Test Lab
    bool isTestLabActivated = await isFirebaseTestLabActivated();
    setState(() {
      _isTestLabActivated = isTestLabActivated;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Firebase Test Lab Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text(
              _isTestLabActivated
                  ? 'Running on Firebase Test Lab'
                  : 'Running on physical device',
              style: TextStyle(fontSize: 24),
            ),
          ],
        ),
      ),
    );
  }
}

// 1. The app starts and runs the `main` function.
// 2. The `main` function runs the `MyApp` widget.
// 3. The `MyApp` widget builds the `MyHomePage` widget.
// 4. The `MyHomePage` widget checks if the app is running on Firebase Test Lab using the `isFirebaseTestLabActivated` function.
// 5. If the app is running on Firebase Test Lab, the `_isTestLabActivated` variable is set to `true`.
// 6. The `MyHomePage` widget displays a message indicating whether the app is running on Firebase Test Lab or not.

// Summary:
// The app uses the `is_firebase_test_lab_activated` package to detect whether it's running on Firebase Test Lab or not.
// The app displays a message indicating whether it's running on Firebase Test Lab or not.
```
