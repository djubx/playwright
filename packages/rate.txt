# Rate Flutter Package
======================

The "rate" Flutter package is a simple and easy-to-use library that allows you to prompt users to rate your app. It provides a customizable dialog box that asks users to rate your app and provides a link to the app store.

### When to Use

You can use this package to:

* Prompt users to rate your app after a certain number of days or uses
* Encourage users to rate your app after a significant event or milestone
* Improve your app's visibility in the app store by increasing the number of ratings

### Features

* Customizable dialog box with title, message, and buttons
* Supports both Android and iOS platforms
* Provides a link to the app store to make it easy for users to rate your app

# Tutorial
==========

### Setup

To use the "rate" Flutter package, you need to add it to your project's `pubspec.yaml` file:

```yml
dependencies:
  rate: ^1.0.0
```

Then, run `flutter pub get` to install the package.

### Usage

To use the package, you need to import it in your Dart file:

```dart
import 'package:rate/rate.dart';
```

Then, you can use the `Rate` class to prompt the user to rate your app:

```dart
Rate(
  title: 'Rate Our App',
  message: 'Please rate our app to help us improve.',
  rateButton: 'Rate Now',
  laterButton: 'Later',
  cancelButton: 'Cancel',
  onRatePressed: () {
    // Handle rate button press
  },
  onLaterPressed: () {
    // Handle later button press
  },
  onCancelPressed: () {
    // Handle cancel button press
  },
);
```

### Platform-Specific Details

#### Android

To use the "rate" package on Android, you need to add the following code to your `AndroidManifest.xml` file:

```xml
<application>
  ...
  <activity
    android:name="com.example.RealFlutter.MainActivity"
    android:exported="true">
    <intent-filter>
      <action android:name="android.intent.action.VIEW" />
      <category android:name="android.intent.category.DEFAULT" />
      <category android:name="android.intent.category.BROWSABLE" />
      <data android:scheme="market" android:host="details" android:path="id=com.example.RealFlutter" />
    </intent-filter>
  </activity>
</application>
```

#### iOS

To use the "rate" package on iOS, you need to add the following code to your `Info.plist` file:

```xml
<key>LSApplicationQueriesSchemes</key>
<array>
  <string>itms-apps</string>
</array>
```

      appBar: AppBar(
        title: Text('Rate App Demo'),

      body: Center(
        child: ElevatedButton(
          onPressed: () {
            showDialog(
              context: context,
              builder: (context) {
                return Rate(
                  title: 'Rate Our App',
                  message: 'Please rate our app to help us improve.',
                  rateButton: 'Rate Now',
                  laterButton: 'Later',
                  cancelButton: 'Cancel',
                  onRatePressed: () {
                    // Handle rate button press
                    print('Rate button pressed');
                  },
                  onLaterPressed: () {
                    // Handle later button press
                    print('Later button pressed');
                  },
                  onCancelPressed: () {
                    // Handle cancel button press
                    print('Cancel button pressed');
                  },
                );
              },
            );
          },
          child: Text('Show Rate Dialog'),
        ),
      ),
    );
  }
}
