# App Center Bundle SDK Flutter Package
=====================================

The `app_center_bundle_sdk` Flutter package is a powerful tool for managing and deploying mobile applications. It provides a suite of features for building, testing, and distributing apps, including crash reporting, analytics, and push notifications.

When to Use This Package
------------------------

This package is ideal for developers who want to streamline their app development and deployment process. It's particularly useful for teams who need to manage multiple apps across different platforms.

Key Features
------------

* Crash reporting and analytics
* Push notifications
* App distribution and management
* Integration with App Center services

# Setting Up the App Center Bundle SDK
=====================================

To use the `app_center_bundle_sdk` package, you'll need to set up an App Center account and create a new app project.

### Step 1: Create an App Center Account

1. Go to the App Center website and sign up for an account.
2. Create a new app project and note down the app secret key.

### Step 2: Add the Package to Your Flutter Project

1. Add the `app_center_bundle_sdk` package to your `pubspec.yaml` file:
```yml
dependencies:
  app_center_bundle_sdk: ^1.0.0
```
2. Run `flutter pub get` to install the package.

### Step 3: Configure the Package

1. Import the package in your Dart file:
```dart
import 'package:app_center_bundle_sdk/app_center_bundle_sdk.dart';
```
2. Initialize the package with your app secret key:
```dart
AppCenterBundleSdk.init('YOUR_APP_SECRET_KEY');
```
### Platform-Specific Configurations

#### Android

1. Add the following code to your `AndroidManifest.xml` file:
```xml
<application>
  <meta-data
    android:name="com.microsoft.appcenter.crashes"
    android:value="YOUR_APP_SECRET_KEY" />
</application>
```
2. Add the following code to your `build.gradle` file:
```groovy
dependencies {
  implementation 'com.microsoft.appcenter:appcenter-crashes:4.3.1'
}
```
#### iOS

1. Add the following code to your `Info.plist` file:
```xml
<key>AppCenterCrashes</key>
<string>YOUR_APP_SECRET_KEY</string>
```
2. Add the following code to your `Podfile`:
```ruby
pod 'AppCenter/Crashes', '~> 4.3.1'
```

```dart
import 'package:flutter/material.dart';
import 'package:app_center_bundle_sdk/app_center_bundle_sdk.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'App Center Bundle SDK Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  void initState() {
    super.initState();
    // Initialize the App Center Bundle SDK
    AppCenterBundleSdk.init('YOUR_APP_SECRET_KEY');
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('App Center Bundle SDK Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            // Crash reporting button
            ElevatedButton(
              onPressed: () {
                // Simulate a crash
                throw Exception('Test crash');
              },
              child: Text('Crash'),
            ),
            // Analytics button
            ElevatedButton(
              onPressed: () {
                // Track an event
                AppCenterBundleSdk.trackEvent('Test event');
              },
              child: Text('Track event'),
            ),
            // Push notifications button
            ElevatedButton(
              onPressed: () {
                // Register for push notifications
                AppCenterBundleSdk.registerForPushNotifications();
              },
              child: Text('Register for push notifications'),
            ),
          ],
        ),
      ),
    );
  }
}

// Application flow:
// 1. The app initializes the App Center Bundle SDK with the app secret key.
// 2. The user can simulate a crash by pressing the "Crash" button.
// 3. The user can track an event by pressing the "Track event" button.
// 4. The user can register for push notifications by pressing the "Register for push notifications" button.
// 5. The app will receive push notifications and display them to the user.

// Summary:
// This demo app showcases the features of the App Center Bundle SDK, including crash reporting, analytics, and push notifications.
// The app initializes the SDK with the app secret key and provides buttons for simulating a crash, tracking an event, and registering for push notifications.
```
