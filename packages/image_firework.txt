# Image Firework Flutter Package
The `image_firework` Flutter package is a powerful tool for creating stunning image effects in your Flutter applications. With this package, you can easily add firework-like effects to your images, making them stand out and grab the user's attention.

## When to Use This Package
You can use this package in various scenarios, such as:

* Creating visually appealing splash screens or login screens
* Adding effects to images in a gallery or slideshow
* Enhancing the user experience in games or interactive stories

## Features
The `image_firework` package offers the following features:

* Customizable firework effects, including size, color, and duration
* Support for various image formats, including JPEG, PNG, and GIF
* Easy integration with existing Flutter widgets and layouts

# Setting Up the Package
To use the `image_firework` package, follow these steps:

1. Add the package to your `pubspec.yaml` file:
```yml
dependencies:
  image_firework: ^1.0.0
```
2. Run `flutter pub get` to install the package.
3. Import the package in your Dart file:
```dart
import 'package:image_firework/image_firework.dart';
```

## Platform-Specific Details
For Android, you need to add the following configuration to your `AndroidManifest.xml` file:
```xml
<application
  ...
  android:hardwareAccelerated="true"
  ...
>
```
For iOS, you need to add the following configuration to your `Info.plist` file:
```xml
<key>UIHardwareAcceleration</key>
<true/>
```

## Using the Package
To use the package, simply wrap your image widget with the `ImageFirework` widget:
```dart
ImageFirework(
  image: Image.asset('assets/image.jpg'),
  fireworkSize: 100,
  fireworkColor: Colors.red,
  duration: Duration(seconds: 5),
)
```

<-- START_MAIN -->
```dart
import 'package:flutter/material.dart';
import 'package:image_firework/image_firework.dart';

void main() {
  runApp(RealFlutter());
}

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Image Firework Demo',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Image Firework Demo'),
        ),
        body: Center(
          child: ImageFirework(
            // Load the image from assets
            image: Image.asset('assets/image.jpg'),
            // Set the firework size to 100
            fireworkSize: 100,
            // Set the firework color to red
            fireworkColor: Colors.red,
            // Set the duration to 5 seconds
            duration: Duration(seconds: 5),
          ),
        ),
      ),
    );
  }
}

// Application flow:
// 1. The app starts and loads the image from assets.
// 2. The image is wrapped with the ImageFirework widget.
// 3. The ImageFirework widget adds a firework effect to the image.
// 4. The firework effect is customized with a size of 100, color red, and duration of 5 seconds.
// 5. The app displays the image with the firework effect in the center of the screen.
// Summary: The app demonstrates the use of the image_firework package to add a firework effect to an image.
```
