**Startapp SDK Flutter Package: A Comprehensive Overview**
===========================================================

The Startapp SDK Flutter package is a popular mobile advertising platform that allows developers to monetize their apps with high-quality ads. With its robust features and easy integration, Startapp has become a go-to choice for many developers.

**When to Use Startapp SDK**
---------------------------

You should consider using Startapp SDK in the following scenarios:

* You want to monetize your app with ads.
* You need a reliable and high-quality ad network.
* You want to enhance user engagement with interactive ads.

**Key Features of Startapp SDK**
------------------------------

* **Interstitial Ads**: Full-screen ads that appear between app screens.
* **Rewarded Videos**: Users watch videos to earn rewards or in-app currency.
* **Banner Ads**: Display ads at the top or bottom of the screen.
* **Native Ads**: Customizable ads that blend with your app's UI.

**Setting Up Startapp SDK in Your Flutter App**
=============================================

### Step 1: Add the Startapp SDK Package

Add the following dependency to your `pubspec.yaml` file:
```yml
dependencies:
  startapp_sdk: ^1.0.0
```
Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the Startapp SDK package in your Dart file:
```dart
import 'package:startapp_sdk/startapp_sdk.dart';
```
### Step 3: Initialize the SDK

Initialize the Startapp SDK in your app's `main` function:
```dart
void main() {
  StartAppSDK.init(
    appId: 'YOUR_APP_ID',
    appSecret: 'YOUR_APP_SECRET',
  );
  runApp(MyApp());
}
```
Replace `YOUR_APP_ID` and `YOUR_APP_SECRET` with your actual Startapp credentials.

### Step 4: Display Ads

Display ads in your app using the following methods:
```dart
// Interstitial Ad
StartAppSDK.showInterstitialAd();

// Rewarded Video
StartAppSDK.showRewardedVideoAd();

// Banner Ad
StartAppSDK.showBannerAd(
  adSize: AdSize.banner,
  adPosition: AdPosition.top,
);

// Native Ad
StartAppSDK.showNativeAd(
  adSize: AdSize.native,
  adPosition: AdPosition.bottom,
);
```
### Platform-Specific Details

* **Android**: Add the following code to your `AndroidManifest.xml` file:
```xml
<application>
  <meta-data
    android:name="com.startapp.sdk.APPLICATION_ID"
    android:value="YOUR_APP_ID" />
  <meta-data
    android:name="com.startapp.sdk.APPLICATION_SECRET"
    android:value="YOUR_APP_SECRET" />
</application>
```
* **iOS**: Add the following code to your `Info.plist` file:
```xml
<key>StartAppApplicationId</key>
<string>YOUR_APP_ID</string>
<key>StartAppApplicationSecret</key>
<string>YOUR_APP_SECRET</string>
```

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:startapp_sdk/startapp_sdk.dart';

void main() {
  StartAppSDK.init(
    appId: 'YOUR_APP_ID',
    appSecret: 'YOUR_APP_SECRET',
  );
  runApp(RealFlutter());
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Real Flutter'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              ElevatedButton(
                onPressed: () {
                  // Show Interstitial Ad
                  StartAppSDK.showInterstitialAd();
                },
                child: Text('Show Interstitial Ad'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  // Show Rewarded Video Ad
                  StartAppSDK.showRewardedVideoAd();
                },
                child: Text('Show Rewarded Video Ad'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  // Show Banner Ad
                  StartAppSDK.showBannerAd(
                    adSize: AdSize.banner,
                    adPosition: AdPosition.top,
                  );
                },
                child: Text('Show Banner Ad'),
              ),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: () {
                  // Show Native Ad
                  StartAppSDK.showNativeAd(
                    adSize: AdSize.native,
                    adPosition: AdPosition.bottom,
                  );
                },
                child: Text('Show Native Ad'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

// The app starts by initializing the Startapp SDK with the app ID and secret.
// Then, it runs the RealFlutter app, which displays a scaffold with an app bar and a column of buttons.
// Each button corresponds to a different type of ad: interstitial, rewarded video, banner, and native.
// When a button is pressed, the corresponding ad is displayed.
// The ads are displayed using the StartAppSDK.showAd() methods, which take the ad size and position as parameters.
```
