# Gesture Zoom Box Package
==========================

The `gesture_zoom_box` package is a Flutter widget that allows users to zoom in and out of a container using gestures. It is useful when you want to provide an interactive way for users to view content, such as images or maps.

### Features

* Zoom in and out using pinch gestures
* Pan across the content using drag gestures
* Double tap to reset the zoom level
* Customizable minimum and maximum zoom levels
* Support for both Android and iOS platforms

### When to Use

This package is useful in scenarios where you want to provide an interactive way for users to view content, such as:

* Image galleries
* Map views
* Document viewers
* Any other scenario where zooming and panning is necessary

# Tutorial
==========

## Step 1: Add the Package to Your Project

Add the `gesture_zoom_box` package to your project by adding the following line to your `pubspec.yaml` file:
```yml
dependencies:
  gesture_zoom_box: ^1.0.0
```
Then, run `flutter pub get` to install the package.

## Step 2: Import the Package

Import the package in your Dart file:
```dart
import 'package:gesture_zoom_box/gesture_zoom_box.dart';
```
## Step 3: Create a GestureZoomBox Widget

Create a `GestureZoomBox` widget and pass in the content you want to zoom:
```dart
GestureZoomBox(
  child: Image.asset('assets/image.jpg'),
)
```
## Step 4: Customize the Zoom Levels (Optional)

You can customize the minimum and maximum zoom levels by passing in the `minScale` and `maxScale` properties:
```dart
GestureZoomBox(
  child: Image.asset('assets/image.jpg'),
  minScale: 0.5,
  maxScale: 2.0,
)
```
## Platform-Specific Details

* On Android, the `GestureZoomBox` widget uses the `ScaleGestureDetector` class to detect scale gestures.
* On iOS, the `GestureZoomBox` widget uses the `UIPinchGestureRecognizer` class to detect pinch gestures.

```dart
import 'package:flutter/material.dart';
import 'package:gesture_zoom_box/gesture_zoom_box.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Gesture Zoom Box Demo',
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Gesture Zoom Box Demo'),
      ),
      body: Center(
        child: GestureZoomBox(
          child: Image.asset('assets/image.jpg'),
          minScale: 0.5,
          maxScale: 2.0,
        ),
      ),
    );
  }
}
```

// This is the main entry point of the app. It creates a MaterialApp with a title and a home page.
// The home page is a Scaffold with an AppBar and a body.
// The body is a Center widget that contains a GestureZoomBox widget.
// The GestureZoomBox widget is passed an Image.asset widget as its child.
// The Image.asset widget displays an image from the assets folder.
// The GestureZoomBox widget allows the user to zoom in and out of the image using gestures.
// The minScale and maxScale properties are set to 0.5 and 2.0, respectively, to customize the zoom levels.

// When the user runs the app, they will see an image on the screen that they can zoom in and out of using gestures.
// The image will be displayed at its original size initially, and the user can use pinch gestures to zoom in and out.
// The user can also pan across the image using drag gestures.
// When the user double taps on the image, it will reset to its original size.
