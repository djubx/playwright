<-- START_DESCRIPTION -->

Typeset is a Flutter package that allows you to display text with advanced typography features, such as superscript, subscript, and strikethrough. It provides a simple and easy-to-use API for creating complex text layouts.

Typeset is useful when you need to display text with specific formatting, such as:

* Displaying mathematical equations or formulas
* Creating text with superscript or subscript characters
* Displaying text with strikethrough or underline
* Creating complex text layouts with multiple styles and fonts

Some of the key features of Typeset include:

* Support for superscript and subscript characters
* Support for strikethrough and underline text
* Support for multiple fonts and styles
* Easy-to-use API for creating complex text layouts

<-- END_DESCRIPTION -->

<-- START_TUTORIAL -->

To use Typeset in your Flutter app, follow these steps:

1. Add the Typeset package to your pubspec.yaml file:
```yml
dependencies:
  typeset: ^1.0.0
```
2. Import the Typeset package in your Dart file:
```dart
import 'package:typeset/typeset.dart';
```
3. Create a Typeset widget and pass in the text you want to display:
```dart
Typeset(
  text: 'Hello, world!',
  style: TextStyle(fontSize: 24),
)
```
4. Use the `superscript` and `subscript` properties to create superscript and subscript characters:
```dart
Typeset(
  text: 'E=mc^2',
  superscript: ['^2'],
  style: TextStyle(fontSize: 24),
)
```
5. Use the `strikethrough` and `underline` properties to create strikethrough and underline text:
```dart
Typeset(
  text: 'Hello, world!',
  strikethrough: true,
  style: TextStyle(fontSize: 24),
)
```
6. Use the `fontFamily` and `fontWeight` properties to change the font and font weight:
```dart
Typeset(
  text: 'Hello, world!',
  fontFamily: 'Arial',
  fontWeight: FontWeight.bold,
  style: TextStyle(fontSize: 24),
)
```
Note: On Android, you need to add the following line to your AndroidManifest.xml file to enable font embedding:
```xml
<application
  ...
  android:hardwareAccelerated="true"
  ...
>
```
On iOS, you need to add the following line to your Info.plist file to enable font embedding:
```xml
<key>UIAppFonts</key>
<array>
  <string>your_font_name.ttf</string>
</array>
```

import 'package:flutter/material.dart';

  runApp(RealFlutter());

class RealFlutter extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Typeset Example',
      home: Scaffold(
        appBar: AppBar(
          title: Text('Typeset Example'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              // Display text with superscript
              Typeset(
                text: 'E=mc^2',
                superscript: ['^2'],
                style: TextStyle(fontSize: 24),
              ),
              SizedBox(height: 20),
              // Display text with subscript
              Typeset(
                text: 'H2O',
                subscript: ['2'],
                style: TextStyle(fontSize: 24),
              ),
              SizedBox(height: 20),
              // Display text with strikethrough
              Typeset(
                text: 'Hello, world!',
                strikethrough: true,
                style: TextStyle(fontSize: 24),
              ),
              SizedBox(height: 20),
              // Display text with underline
              Typeset(
                text: 'Hello, world!',
                underline: true,
                style: TextStyle(fontSize: 24),
              ),
              SizedBox(height: 20),
              // Display text with font family and font weight
              Typeset(
                text: 'Hello, world!',
                fontFamily: 'Arial',
                fontWeight: FontWeight.bold,
                style: TextStyle(fontSize: 24),
              ),
            ],
          ),
        ),
      ),
    );
  }
}
