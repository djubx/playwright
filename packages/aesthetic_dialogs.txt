Aesthetic Dialogs for Flutter
==========================

The `aesthetic_dialogs` package provides a set of beautiful and customizable dialog boxes for Flutter applications. These dialogs can be used to display important information, warnings, or errors in a visually appealing way.

When to Use
------------

*   Displaying important information that requires user attention
*   Showing warnings or errors in a user-friendly manner
*   Creating a visually appealing and consistent dialog experience throughout the app

Features
--------

*   Customizable dialog styles and colors
*   Support for various dialog types (e.g., info, warning, error)
*   Easy-to-use API for displaying dialogs

<-- END_DESCRIPTION -->

<-- START_TUTORIAL -->

Setup and Usage
---------------

### Step 1: Add the Package to Your Project

Add the `aesthetic_dialogs` package to your `pubspec.yaml` file:

```yml
dependencies:
  flutter:
    sdk: flutter
  aesthetic_dialogs: ^1.0.0
```

Then, run `flutter pub get` to install the package.

### Step 2: Import the Package

Import the `aesthetic_dialogs` package in your Dart file:

```dart
import 'package:aesthetic_dialogs/aesthetic_dialogs.dart';
```

### Step 3: Display a Dialog

Use the `showAestheticDialog` function to display a dialog:

```dart
showAestheticDialog(
  context: context,
  dialogType: DialogType.INFO,
  title: 'Info Dialog',
  message: 'This is an info dialog.',
);
```

### Platform-Specific Details

*   Android: No additional configuration required.
*   iOS: No additional configuration required.

### Optimizations

*   You can customize the dialog style and colors by passing a `DialogStyle` object to the `showAestheticDialog` function.
*   You can also use the `DialogType` enum to specify the type of dialog to display (e.g., info, warning, error).

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:aesthetic_dialogs/aesthetic_dialogs.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Aesthetic Dialogs Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Aesthetic Dialogs Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            ElevatedButton(
              onPressed: () {
                // Display an info dialog
                showAestheticDialog(
                  context: context,
                  dialogType: DialogType.INFO,
                  title: 'Info Dialog',
                  message: 'This is an info dialog.',
                );
              },
              child: Text('Info Dialog'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                // Display a warning dialog
                showAestheticDialog(
                  context: context,
                  dialogType: DialogType.WARNING,
                  title: 'Warning Dialog',
                  message: 'This is a warning dialog.',
                );
              },
              child: Text('Warning Dialog'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                // Display an error dialog
                showAestheticDialog(
                  context: context,
                  dialogType: DialogType.ERROR,
                  title: 'Error Dialog',
                  message: 'This is an error dialog.',
                );
              },
              child: Text('Error Dialog'),
            ),
          ],
        ),
      ),
    );
  }
}

// The app starts by running the main function, which creates a new instance of the MyApp widget.
// The MyApp widget builds a MaterialApp with a title and a theme.
// The home property of the MaterialApp is set to an instance of the MyHomePage widget.
// The MyHomePage widget builds a Scaffold with an AppBar and a body.
// The body of the Scaffold contains a Column with three ElevatedButton widgets.
// Each ElevatedButton widget displays a different type of dialog when pressed.
// The dialogs are displayed using the showAestheticDialog function, which takes the context, dialog type, title, and message as arguments.
```
