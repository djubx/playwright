# Pinch to Zoom Scrollable Flutter Package
=============================================

The `pinch_to_zoom_scrollable` Flutter package provides a widget that allows users to zoom in and out of content using pinch gestures, while also enabling scrolling. This package is ideal for applications that require interactive and immersive user experiences, such as:

* Image galleries
* Document viewers
* Maps
* Interactive tutorials

Key Features:

* Pinch to zoom in and out
* Scrolling enabled while zoomed in
* Customizable zoom limits and scroll boundaries
* Support for Android and iOS platforms

# Setting Up the Package
-------------------------

To use the `pinch_to_zoom_scrollable` package, follow these steps:

1. Add the package to your `pubspec.yaml` file:
```yml
dependencies:
  flutter:
    sdk: flutter
  pinch_to_zoom_scrollable: ^1.0.0
```
2. Run `flutter pub get` to install the package.
3. Import the package in your Dart file:
```dart
import 'package:pinch_to_zoom_scrollable/pinch_to_zoom_scrollable.dart';
```
### Platform-Specific Configurations

* Android: No additional configurations required.
* iOS: Add the following lines to your `Info.plist` file:
```xml
<key>NSPhotoLibraryUsageDescription</key>
<string>This app needs access to your photo library to display images.</string>
```
### Optimizations

* To improve performance, consider using a `CachedNetworkImage` or a similar caching mechanism to load images.

<-- START_MAIN -->

```dart
import 'package:flutter/material.dart';
import 'package:pinch_to_zoom_scrollable/pinch_to_zoom_scrollable.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Pinch to Zoom Scrollable Demo',
      home: RealFlutter(),
    );
  }
}

class RealFlutter extends StatefulWidget {
  @override
  _RealFlutterState createState() => _RealFlutterState();
}

class _RealFlutterState extends State<RealFlutter> {
  // Step 1: Create a PinchToZoomScrollable widget
  PinchToZoomScrollable _pinchToZoomScrollable;

  @override
  void initState() {
    super.initState();
    _pinchToZoomScrollable = PinchToZoomScrollable(
      // Step 2: Set the content to be displayed
      child: Image.asset('assets/image.jpg'),
      // Step 3: Configure the zoom limits
      minScale: 0.5,
      maxScale: 2.0,
      // Step 4: Configure the scroll boundaries
      boundaryMargin: EdgeInsets.all(20.0),
    );
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Pinch to Zoom Scrollable Demo'),
      ),
      body: Center(
        child: _pinchToZoomScrollable,
      ),
    );
  }
}

// Application flow:
// 1. The user opens the app and is presented with an image.
// 2. The user pinches to zoom in on the image.
// 3. As the user zooms in, the image becomes larger and the scroll boundaries are adjusted.
// 4. The user can then scroll around the zoomed-in image.
// 5. The user can pinch to zoom out to return to the original image size.
// Summary: The PinchToZoomScrollable widget provides a seamless and interactive way to zoom in and out of content, while also enabling scrolling.
```
