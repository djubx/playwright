# Flutter Facebook Keyhash Package
=====================================

The `flutter_facebook_keyhash` package is a Flutter plugin that allows you to generate and retrieve the Facebook key hash for your Android and iOS applications. This key hash is required to configure Facebook login and other Facebook services for your app.

## When to Use This Package
---------------------------

You need to use this package when you want to integrate Facebook services, such as Facebook login, into your Flutter application. The key hash is a unique identifier that Facebook uses to authenticate your app and ensure that it is genuine.

## Features
------------

*   Generates the Facebook key hash for Android and iOS applications
*   Retrieves the generated key hash for use in your app
*   Supports both Android and iOS platforms

# Tutorial: Setting Up and Using the Package
=============================================

## Step 1: Add the Package to Your Pubspec File
---------------------------------------------

Add the following line to your `pubspec.yaml` file:

```yml
dependencies:
  flutter_facebook_keyhash: ^1.0.0
```

Then, run `flutter pub get` to install the package.

## Step 2: Import the Package
---------------------------

Import the package in your Dart file:

```dart
import 'package:flutter_facebook_keyhash/flutter_facebook_keyhash.dart';
```

## Step 3: Generate the Key Hash (Android)
-----------------------------------------

For Android, you need to generate the key hash using the `generateKeyHash` method:

```dart
Future<void> _generateKeyHash() async {
  final keyHash = await FlutterFacebookKeyhash.generateKeyHash();
  print('Generated key hash: $keyHash');
}
```

## Step 4: Generate the Key Hash (iOS)
--------------------------------------

For iOS, you need to generate the key hash using the `generateKeyHash` method with the `ios` platform specified:

```dart
Future<void> _generateKeyHash() async {
  final keyHash = await FlutterFacebookKeyhash.generateKeyHash(platform: Platform.ios);
  print('Generated key hash: $keyHash');
}
```

## Step 5: Retrieve the Generated Key Hash
-----------------------------------------

You can retrieve the generated key hash using the `getKeyHash` method:

```dart
Future<void> _getKeyHash() async {
  final keyHash = await FlutterFacebookKeyhash.getKeyHash();
  print('Retrieved key hash: $keyHash');
}
```

```dart
import 'package:flutter/material.dart';
import 'package:flutter_facebook_keyhash/flutter_facebook_keyhash.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Facebook Keyhash Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatefulWidget {
  @override
  _MyHomePageState createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {
  String _keyHash = '';

  Future<void> _generateKeyHash() async {
    // Generate the key hash for the current platform
    final keyHash = await FlutterFacebookKeyhash.generateKeyHash();
    setState(() {
      _keyHash = keyHash;
    });
  }

  Future<void> _getKeyHash() async {
    // Retrieve the generated key hash
    final keyHash = await FlutterFacebookKeyhash.getKeyHash();
    setState(() {
      _keyHash = keyHash;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Flutter Facebook Keyhash Demo'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text('Generated Key Hash:'),
            Text(_keyHash),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _generateKeyHash,
              child: Text('Generate Key Hash'),
            ),
            SizedBox(height: 10),
            ElevatedButton(
              onPressed: _getKeyHash,
              child: Text('Get Key Hash'),
            ),
          ],
        ),
      ),
    );
  }
}

// The application flow is as follows:
// 1. The user opens the app and is presented with a screen containing two buttons: "Generate Key Hash" and "Get Key Hash".
// 2. When the user clicks the "Generate Key Hash" button, the `_generateKeyHash` method is called, which generates the key hash for the current platform using the `FlutterFacebookKeyhash.generateKeyHash` method.
// 3. The generated key hash is then stored in the `_keyHash` variable and displayed on the screen.
// 4. When the user clicks the "Get Key Hash" button, the `_getKeyHash` method is called, which retrieves the generated key hash using the `FlutterFacebookKeyhash.getKeyHash` method.
// 5. The retrieved key hash is then stored in the `_keyHash` variable and displayed on the screen.

// In summary, this app demonstrates how to generate and retrieve the Facebook key hash using the `flutter_facebook_keyhash` package.

```
